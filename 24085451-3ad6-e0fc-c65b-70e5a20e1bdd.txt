　　他喜欢人间的东西，也好学，因此连说话的用词语法，也跟当代人差不多。
<br/>　　除了这两大爱好，他对别的事都没什么兴趣。
<br/>　　“教主，以你的身份，就算是轩辕上帝，也没法强迫你必须参战吧，”秦风一边进攻着，继续试探，“你不喜欢斗法，可以不来啊。”
<br/>　　太平教主道：“我也想来看看，一直听他们说叶少阳，我想看看这孩子到底如何。”
<br/>　　随即朝空荡荡的道观大殿看了一眼，道：“的确是个好苗子。”
<br/>　　“你们三位就陪我演一会吧，我不想伤你们，也算走过场。”
<br/>　　“哼哼，说的好像你一定能赢似的，不然你就出全力，咱们试试嘛。”小马盯着他光溜溜的脑袋，流着口水。
<br/>　　太平教主微微一笑，也不分辨。
<br/>　　秦风感觉有点不可思议，说道：“如果你那几位师兄弟遇到危难呢，你难道也不帮？”
<br/>　　“死生有命，我修炼千年，这些早就看淡了，况且，不给他们点苦头吃，他们怎知道自己做错了？有人做错有机会改正，有人要付出性命代价，都是因果循环，旁人管他作甚。”太平教主的确话多，但也句句在理。
<br/>　　秦风三人听得目瞪口呆。
<br/>　　他们不知道的是，这七大长老虽然名义上是师兄弟，但不是轩辕上帝在同一个时期收的徒弟，在轩辕山也都有自己的地盘，除了重大事情在一起商议下，平时都是各忙各的，极少来往，只是有个师兄弟的名分而已，彼此并不亲密。
<br/>　　只有星月奴和清长风，这两人入师门时间差不多，实力也相近，关系最亲。
<br/>　　太平教主把话都挑明了，三人也觉得轻松许多，于是开始了一场默契的表演，因为知道是假的，不会伤到对方，出手反而大胆了许多，一招一式地拆解着，反而看上去比其他那些都激烈。
<br/>　　“你天赋异禀，但你法术实在太槽了，杂乱无章，进攻也没有重点，你再试着打我一拳……”
<br/>　　太平教主看到小马出手，实在忍不住吐槽起来，最后竟开始指点他，小马起初不乐意，但是照他说的去做了之后，发现攻势果然流畅了许多。
<br/>　　“看到吗，是不是很容易？”
<br/>　　太平教主比他还开心，也对他出手，一边做着讲解，让他仔细看着。
<br/>　　“天赋，也分为很多种，有人适合正途修道，有人适合观想，寻求顿悟，你却适合修炼混元之法。”
<br/>　　小马立刻问他什么是混元之法。
<br/>　　太平教主解释：“混元，便是大器无形，重剑无锋，只求气势，不求招式……”
<br/>　　小马品味了一番，困惑道：“可你之前又说我法术太糙……”
<br/>　　“混元，是一种境界，是让别人看不到你的锋芒，自然无懈可击，这说白了是一种战术，在你自己，自然是有招式手段的，而且要更精细，才会产生混元之效果，我来教你……我们走远一些吧，免得他们看出问题，终归师兄弟一场，面子上过不去。”
<br/>　　于是四人且战且退，到了更远处的山谷里，索性也不装了，一板一眼地教起小马，不过强迫秦风夫妻在一边看着不许走，因为他们几个是对手，他不能因为自己不想打架，而让他们去对付自己的同门，给他们增加压力。
<br/>　　这奇怪的逻辑……
<br/>　　秦风夫妻也就在一边围观，听了一会，这太平教主不愧是个当老师的，讲解极为精准，而且通俗易懂，不光小马感到像打开了新世界的大门，秦风夫妻也受益匪浅……
<br/>　　与他们这边和谐而怪诞的画面对比强烈的是，在大殿附近，另外的几场战斗，已经进行到了白热化地步，不少人身受重伤，但还在苦苦支撑着，对方也在支撑着。
<br/>　　这些人中，大部分都不认识对方，谈不上仇恨，只因为身属两个不同的阵营，就得拼死搏杀。
<br/>　　捉鬼联盟这边人多，在战斗持续一段时间后，遇到人少打不过的对手，就互相串联，几个打一个，打不过也能周旋，因此也没有伤亡。
<br/>　　打到这个地步，双方其实都是在等，一边等叶少阳，一边等星月奴，大家都清楚，在缺乏这二人的情况下，这种战争是不可能有什么结果的。
<br/>　　捉鬼联盟这边还好，轩辕山那边，打到最后，几个人都有点烦了，来的时候，每一个来的时候都是盛气凌人，他对人间法术界不了解，也一直有蔑视心理，抱着好奇心和虐菜的心理，这才跟星月奴一起来，他们哪里想到这帮人这么能打，而且显然提前做足了准备，只围着打消耗，不强攻。
<br/>　　等他们被惹恼之后，试图强攻杀死一两个人，这人立刻退走，其余人又围攻过来。捉鬼联盟这些成员，单打都不是他们的对手，但平均实力都不差，至少不会被秒杀，又一起战斗多次，配合极为默契，让这些看不起人间法师和邪物的高高在上的神祇吃尽了苦头，有气也没处发，攻击也更加癫狂，如此一来，破绽也就出现了。
<br/>　　大伙等的就是这个，继续不动声色地跟他们消耗。
<br/>　　第一个被抓到破绽的是毕方，他身为火鸟，性格也是最暴躁的，在岳恒和孙映月的围攻之下消耗了半天，后来四宝和吴嘉伟也加入战斗，选择毕方，是因为在持续的消耗之中，他们发现毕方最好对付一点。
<br/>　　不是他实力差，而是他脾气暴躁，更容易被激怒，从而现出破绽。
<br/>　　“吼……”
<br/>　　在四人围攻之下，毕方终于忍耐不住，化成一只巨大的火鸟，一飞冲天，扇动着翅膀，用力俯冲下来，身体宛如一颗巨大的火球，边上那些飞舞的羽毛，形成了一个半圆形的结界，将他们几人罩在中间，结界并不是很坚固，但火鸟飞落的速度极快，想赶在它落下之前打开结界逃出去是不可能了，只能接下这一招了。
<br/>　　谁都看得出，这是毕方实在不想被消耗下去，因此全力一搏，使出了最强一击。
