　　第19章
<br/>　　“大王饶命，俺们都是穷人，实在没钱……您要杀就杀我，千万别杀我儿子。”李狗剩吓得亡魂皆冒。
<br/>　　“谁说要杀你，站起来说话。”
<br/>　　李狗剩四个怯怯的站起来，李狗剩道：“您不杀我，要做什么，我们可什么都没有？”
<br/>　　“刚才我听见你说要买马，对不对？”
<br/>　　你这种一脸兴奋的表情是几个意思，李狗剩满头雾水。他怯怯道：“俺哪里买得起马，只是我这大儿子娶不起亲，家里如果能买头小毛驴耕地，也好找人说亲。”
<br/>　　“哈哈，你的造化到了，我正好是卖马的，激不激动，幸不幸福？”
<br/>　　李狗剩现在死的心都有，自己肯定是出门没看黄历，才撞到这两个山贼，当然是很激动，激动今天能不能活下去。
<br/>　　“可俺没有那么多钱啊？”
<br/>　　“嗯，毕竟还是要拿钱买的，你有多少？”
<br/>　　“两个……铜板。”李狗剩小心翼翼从衣服中掏出两个快要磨光的铜板，也不知道他揣了多久。
<br/>　　“这个……”程大雷一咧嘴：“你身上还有别的么？”
<br/>　　“我这儿还有四个野菜饼，俺娘给我们烙的，让我们路上充饥。”
<br/>　　李大打开包裹，里面有四个烤糊的菜饼，程大雷掰下一块，又递给林少羽。
<br/>　　“味道还行？”
<br/>　　“嗯，还不错。”
<br/>　　程大雷和林少羽对视一眼，扭头走到一边，凑在一起窃窃私语。
<br/>　　“大当家，两个铜板，四张菜饼这价有点低？”
<br/>　　“我也知道有点低，但不卖怎么办，难道真的杀了吃肉？”
<br/>　　“要不卖给他们，看他们都是穷人家，当做好事了。”
<br/>　　“那你的意思说卖了？”
<br/>　　“卖了。”
<br/>　　李狗剩父子看着二人背影，不知道二人嘀咕什么，心脏砰砰打鼓。落到山贼手中，哪有什么好下场，一刀被砍掉脑袋，这情况也是有的。
<br/>　　“哈哈。”
<br/>　　忽然一阵大笑，吓得四人又差点跪在地上。
<br/>　　“好啦，我头一次开张，算你们运气，上山吧。”
<br/>　　“上山！”李狗剩眼睛瞪直，这是要把自己父子掳到山上当山贼啊。他赶紧哀求：“大王爷爷，你要抓就抓我，起码放了我儿子吧，还得给我们李家传香火。”
<br/>　　“怎么！你们不想上山！”程大雷举起大斧，跃跃欲试要刮胡子，见到这一幕，李狗剩那还敢说话。
<br/>　　“走着，上山。林少侠，照顾好我们的客户。”
<br/>　　林少羽走在前头，程大雷扛着大斧走在最后，中间是李家父子四人。
<br/>　　四人魂都要下没了，感觉命在蛛丝上系着，随时都会掉下来。
<br/>　　“爹……他们会杀我们么？”李三。
<br/>　　“孩子，挺住，不能怂啊。”
<br/>　　“爹，你腿先别逗再说这话。”
<br/>　　“等他们要杀我们，我就和他们玩命，小三，你死活得跑出去，你是咱们家的香火啊。”
<br/>　　四人一步一摇，脚像踏在云里，他们走得哪里是上山的路，明明是通向鬼门关。远远看清寨门，四人已经面如死灰，踏入这山寨，再想出来就难了。
<br/>　　“四位，请了！”林少羽引着四人前往马圈。
<br/>　　“我要和你们拼……”李狗剩大吼一声，等看清马圈情形时，突然顿住。
<br/>　　“拼什么？”林少羽满目困惑。
<br/>　　“拼哆哆么？”程大雷道：“快点，选一匹？”
<br/>　　李狗剩父子四人都处于极度的震撼中，只见马圈中，挤满各色马匹，黑得，白的，红的，灰的……数量多到都要挤在一起。一匹匹高昂头颅，全套马鞍，这可并不是一般的马，而是健壮的战马。体力，块头，身条都不是一般马匹可比。
<br/>　　李家三兄弟，口水都快流到敌人，既然是男儿身，谁要不想跃马扬鞭，驰骋天下。
<br/>　　李狗剩如坠云里雾里，怯生生道：“大王，您真要卖给我们？”
<br/>　　“嗯呐，你要几匹？”
<br/>　　“一匹。”
<br/>　　“一匹，我费了这么大劲儿，你才要一匹！”程大雷眉头拧起。
<br/>　　“要不两匹？”李狗剩忐忑不安的举起两根手指，生怕自己太贪心触怒这位山大王。
<br/>　　“才两匹。”程大雷眉头皱的更紧：“你有三个儿子，难道不该一人一匹，你自己再要一匹，你说四匹怎么样？”
<br/>　　李家三兄弟满面兴奋的红光，示意自己爹爹赶紧答应。李狗剩狠狠等他们一眼，无奈道：“养不起。”
<br/>　　“那就两匹吧。”程大雷忽然想起一事：“你们会割麦子么？”
<br/>　　“俺们都是种田的，割麦子有什么不会的。”
<br/>　　“好，天黑之前只要你们一人能割一亩麦子，我就让你们牵走两匹马。”
<br/>　　程大雷冲林少羽点点头，林少羽佩服的竖起大拇指，大当家果然机智得一逼，二人都因逃过苦差长松一口气。
<br/>　　带着，或者说押着四人前往麦田，一人发他们一把镰刀割麦子。最后程大雷也没有逃过一劫，被徐神机抢塞给一把镰刀。而最让程大雷不忿的是，林少羽竟然借口再往山下卖马，趁机溜号。
<br/>　　镰刀刷刷扫过，在夏天的日头下放倒一片片麦子，程大雷站直身子擦掉额头汗珠。
<br/>　　“大当家，你又偷懒！”徐神机。
<br/>　　“没有啦，你难道没看我也很勤奋。”
<br/>　　“快点，赶不上吃晚饭啦。”徐神机催促道。
<br/>　　看着太阳下大片大片的金黄麦田，程大雷恍然失神，觉得蛤蟆寨就像一个小小的家一样。秦蛮是憨厚的大哥，林少羽是倔强的弟弟，徐灵儿是调皮水灵的……没过门的媳妇，对，对就是媳妇，那徐神机自然就是势利的老丈人……他觉得，自己有些喜欢这里了。
<br/>　　李狗剩四人一个都不敢吭声，生怕触怒这山大王。
<br/>　　“爹，你说他真会给我们马么？”
<br/>　　“别做梦了，快点干活。”
<br/>　　“我估计他说的是天黑之前割不完麦子，就送我们见阎王。”
<br/>　　四人唰唰的割麦，都是田里人，不怵这样活计，在死亡的威胁下，他们都爆发了潜力。大片的麦田被割到，又扎成麦捆，拉到山寨的校场。
<br/>　　“大王，用不用我们帮忙碾成麦粒？”
<br/>　　“不必啦，去马圈选两匹马吧。”
<br/>　　咦，真给啊！
