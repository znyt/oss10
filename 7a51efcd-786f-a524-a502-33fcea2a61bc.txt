　　“老师！”
<br/>　　在巫族离开后，一旁的女娲来到鸿钧道祖身边，躬身行礼道。
<br/>　　鸿钧道祖淡淡的看了她一眼，便将目光投向洪荒之中，传音天下：
<br/>　　“从今以后，妖族掌天，巫族掌地，万年之内，不得再战！”
<br/>　　至于万年之后都的事情，他却是不会去管了，巫妖之间注定有一战，之后，两族衰败，人族才有崛起的机会。
<br/>　　随着鸿钧道祖的身影消失，妖帝帝俊与东皇太一、妖师鲲鹏等人不由得大出几口气，不管是巫族的十二都天神煞大阵演化出来的盘古真身，还是鸿钧道祖，他们的气场都实在是太大了，众人在他们眼前都不敢大口出气。
<br/>　　此处大战，他们身受重伤，若不是道祖鸿钧出手，只怕妖族此番真的会就此覆灭了。
<br/>　　待众人回到凌霄宝殿，还没来得及缓口气，随即便有一个小妖神情慌张的跑过来，大声喊道：
<br/>　　“妖帝陛下，不好了，天庭宝库失窃，里面的宝物都被人盗走了！”
<br/>　　“什么？！”
<br/>　　“噗——”
<br/>　　众人闻言，不由得为之大惊，妖帝帝俊与东皇太一更是为之身形一震，张口便是一口鲜血狂喷而出……
<br/>　　过了不知多久，众人才缓过来，只是脸色依旧很难看。
<br/>　　“大哥，巫族竟能结出盘古真身，这该怎么办？”太一脸色苍白道。
<br/>　　“那都天神煞大阵如此厉害，也必有致命的缺点，我们只要谋划一番，不是不可战胜。”帝俊脸色同样苍白，眼中闪过狠厉之色。
<br/>　　“有何缺点？”
<br/>　　“都天神煞大阵必须有十二祖巫同时出手，方可结成，若是缺少一人，它便不攻自破了。”帝俊低沉道。
<br/>　　“对啊，十二祖巫是不好对付，但是一个祖巫，我们还是可以对付的。”太一眼前一亮道。
<br/>　　“嗯。”
<br/>　　……
<br/>　　此番巫妖大战，持续时间极短，并不像第一次大战，持续数百上千年。但是，巫族大军杀入天庭，妖族军队倾尽全力抵挡，一番大战，死伤无数。
<br/>　　虽然，最后有鸿钧道祖出手，结束了这场战争，但是，巫妖二族，损失依旧不可估量。巫族大军死伤千万，几乎损失一半，妖族更是惨烈，被巫族打得个措手不及，又遭受到盘古真身的屠杀，死伤更是无数。
<br/>　　此后，两族都安静下来了，各自回去休养生息。道祖威压不可冒犯，至少万年之内，两族不敢大战。虽说大战没有，但是小战依旧不断，不过，那些小鱼小虾之事，也引不起多大风浪。
<br/>　　洪荒世界经此番大战，除了无尽生灵遭受涂炭，环境倒是受损不大，毕竟巫妖两族争的就是地盘，若是将洪荒破坏了，还争了干什么？是以，两族之间都有意控制自己的手段，没有使出太过毁天灭地的大神通。
<br/>　　巫妖大战刚刚结束，这一段时间内，洪荒倒是安静了许多，洪荒生灵得到了一个暂时相对稳定的发展环境。
<br/>　　见识了此次巫妖大战，那等惨烈的场面，吓住了不少洪荒普通修士。于是，有不少修士都行走于洪荒名川大山之间，想访仙拜师求道，希望能够拜入一方势力，一是为了得其庇护，保证以后再次遇到此等大战，不至于只能等死，可以安心参悟大道。
<br/>　　二是为了得到更好的修炼资源，提升自己的修为。洪荒此刻的修行，大多还只是本能的修行，朝臀暮吐，采集天地灵气，洪荒灵力充盈，倒是法力增长也快，只是那道行却不见长进，若是能够得到更好的指点，修行便更快了。
<br/>　　一时间，洪荒中掀起了一股拜师热潮。
<br/>　　这场战争影响深远，许多大神通者见识到了盘古真身的强大威力，纷纷为之惊骇，那种无力之感，他们确实不想再经历了。
<br/>　　应对此等危机，诸人各有谋算。
<br/>　　洪荒修士中，也有人来到昆仑山拜师。正所谓大树底下好乘凉，在这洪荒之中，圣人就是最大，若是能够拜入道圣人门下，还有什么人能欺负自己？
<br/>　　圣人之中，有三清与女娲，女娲为妖族，道场在混沌之中，没人能够进入混沌去拜师，所以，大多数人将目光投向昆仑三清，至于西方灵山，大家都知道西方贫瘠不堪，却是没有多少人去拜师。
<br/>　　这段时间，昆仑山附近汇聚了好多修士，他们纷纷想要拜师。元始无所谓，没有想要收徒的意思，老子无为，更是没有收徒的意思，倒是通天，对收徒一事，颇感兴趣，他也想自己的道统，有几个传承之人。
<br/>　　于是，他设下考验，通过者，便可被他收为徒弟。
<br/>　　通天如今还没有立教，所以他也只是想要收几个传承道统之人，尽管他本人不在乎什么出生与资质，但他此次还是想收几个资质高的徒弟。
<br/>　　通过筛选，最后通天收了传说中的多宝道人，金灵圣母，无当圣母三位亲传弟子。其中，多宝道人为多宝塔化形而出，资质极高，颇得通天喜爱。
<br/>　　金灵圣母为葵水之精所化，资质也是极高的，为日后的斗姆元君，执掌金阙，坐镇斗府，居周天烈宿之首，为北极紫薇之尊。
<br/>　　无当圣母，出生黎山，为日后的黎山老母，资质自然也是极高的，她生而为人形。日后更是创出罗教，为无生老母，或称无极老母。
<br/>　　通天喜得三徒，心情极好，赐下修行功法与灵宝，让他们自己修炼。
<br/>　　元始本来是不想收徒的，但是元神抖动，好像冥冥之中，有人与自己有缘一般。他掐指一算，顿时明了，还真是有两个与自己有缘的徒弟。
<br/>　　他想了想，收个徒弟也没什么，反正日后也要立教，收徒弟是迟早的事情，早收与晚收，又有多大区别？更何况，早收徒弟，也好让他们早早修炼。
<br/>　　这两人一个正是日后阐教中的十二金仙之二，一为广成子，日后的阐教大师兄，二为玉鼎真人。
<br/>　　两人资质都极为不俗，广成子为十二金仙之一，化形极为金仙，日后能成为阐教大师兄，自然不用多说。
<br/>　　玉鼎真人乃玉泉山一先天玉石所化，化形就为金仙，资质逆天，要不然元始也不会看上他，他日后更是教出杨戬那样的徒弟。
<br/>　　既然有缘，元始自然是收了。他一样赐下两人修炼功法《玉清元始经》，让他们自行修行，若有不会之处，便向自己提问即可。
<br/>　　至于昆仑山外的其他人，便被元始遣散了，他也不可能将其他人全部收了。在收徒一事上，他其实与原著中的原始极为相像，不是资质逆天者，或者他极为顺眼的，他绝对不会收。
<br/>　　收完徒弟，元始继续闭关，他感应到了突破混元中期的契机，是应该突破了。
<br/>　　……
