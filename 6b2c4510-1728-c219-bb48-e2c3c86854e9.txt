
    东晋的文学家陶渊明曾经写过一篇诗集桃花源记，其中有一段写的是‘忽逢桃花林夹岸数百步，中无树芳草鲜美……林尽水源便得一山，山有小口仿佛若有光，复行数十步豁然开朗’。<br/>　　<br/>　　    说的就是一处从未有人抵达过的世外桃源，那里仿佛跟外面的世界呈现出两个不同的状态，妥妥的与世隔绝着。<br/>　　<br/>　　    多年前，好莱坞也曾经拍过一部电影，叫做地心引力，说的大概也是这么回事，在那处同样与世隔绝的世界里，蚊子和麻雀一样大，大象却小得跟个蚂蚱似的，同样是一片天空却是既然不同的两种环境，属实让人大吃一惊。<br/>　　<br/>　　    但这毕竟是小说和电影里的故事，而在现实生活中有没有这种跟桃花源和地心引力差不多一样的地方呢，以前可能无人所知或者鲜有人知，但现在这个谜底似乎要大白于天下了。<br/>　　<br/>　　    三井家族往东南，王惊蛰等人向东，差不多相同的时间启程，一方正好比另一方快了十几分钟左右，三井家族是靠村民领路，王惊蛰则是凭借着山中脉络的走向过去的。<br/>　　<br/>　　    时间一晃而过，几个小时后，那领路的藏民在前面忽然之间就不走了，反倒是一脸虔诚的突然就跪在了地上，朝着东南方向叩拜起来。<br/>　　<br/>　　    三井小叶问道：“怎么回事？”<br/>　　<br/>　　    小林昙一询问了一番后，说道：“他说，前面就是他们的圣地了，那里有神庙和神明，他不能再往前走了”<br/>　　<br/>　　    三井美奈等人仰头张望，看了片刻后一无所获，从他们这里往前看，仍旧是山叠着山，一眼都望不到头，入眼的景色枯燥的很，哪里来的什么神明。<br/>　　<br/>　　    三井良河从身上掏出枪扳开了撞针，咬牙说道：“你跟他讲，要是再不带路，干脆给他杀了算了，实在不行我们会到那个村子里再找人过来，我就不信了，他们是怕死还是怕什么狗屁的神明”<br/>　　<br/>　　    小林昙一淡淡的说道：“那你真说错了，对于这些藏民来说死并不可怕，他们有人甚至会崇尚死亡，觉得尸体不过是一幅普通的皮囊而已，死后灵魂可以轮回，而神明则是他们的信仰，不容亵渎的”<br/>　　<br/>　　    三井良河顿时一愣，三井美奈皱眉说道：“你在问下，甚至我们可以在多拿出一些补给也行，这里一眼望不到头都是山，没有人领路我们根本不知道怎么走”<br/>　　<br/>　　    当下，小林昙一继续和对方沟通，良久之后在他又拿出一部分药品后藏民才答应继续领路，不过这次的速度却慢了很多。<br/>　　<br/>　　    藏民每走三步就会双膝跪地举起双手叩拜，行五体投地大礼，然后起身再走几步，接着再拜。<br/>　　<br/>　　    小林昙一解释说他这是他答应领路的唯一要求了，为了不打扰神明的安息和避免受到责罚，他只能用这种方式来领路了。<br/>　　<br/>　　    如此一来，时间无疑是慢了不少，足足过去两个多小时，忽然之间从他们前方一处冰天雪地的雪山后面飞出一片鸟群，盘旋在山头上，飞了两圈后又扑扇着翅膀落入了山头下。<br/>　　<br/>　　    三井美奈仰头看着，惊呼道：“这不可能，斑斓蓝鸦，只生长于亚热带的一种鸟类，不可能会生活在这种高海拔地区的”<br/>　　<br/>　　    小林昙一笑道：“不可能就对了，这说明我们来对了地方……”<br/>　　<br/>　　    三井美奈吐了口气，点头说道：“你说的有道理，不正常，才是最正常的，我们找到了马丁鲍曼来到的地方”<br/>　　<br/>　　    翻过一座山头，天色将黑之际，一幕令人震惊，不解，只能用一句鬼斧神工来形容的画面出现在了众人眼前。<br/>　　<br/>　　    群山之巅下，出现一处仿佛巨大天坑一样的地方，那天坑的口子至少有几个足球场加在一起那么大，周围全都是一眼也望不到底的悬崖峭壁，在天坑的底部是一片郁郁葱葱的丛林，从丛林到山顶的峭壁至少得有百米以上的高度了。<br/>　　<br/>　　    如此鬼斧神工的画面，让人望去时不免心生感叹，大自然的奇妙你永远都无法想象。<br/>　　<br/>　　    也许鲜有人知，在喜马拉雅山脉腹地中，层层群山环绕下，会有着这样一片世界，当周围全都是高耸的雪山，高海拔的地势时，却在山下忽然冒出一片春暖花开的地方，这里似乎被某种神秘的力量给隔绝开一样，外界白雪皑皑，此处却生意盎然。<br/>　　<br/>　　    那带路来的藏民跪在地上不住的高举着双手叩拜着，脸上虔诚的表情，很轻易的就能感觉到他的激动和敬仰，当看见这处巨大天坑的时候，他死活都不肯在往前走一步了，一直跪拜个不停。<br/>　　<br/>　　    小林昙一和三井美奈心中的惊奇久久都未能平静下来，直到一群群飞鸟从天坑底部飞上来环绕一圈后又飞了回去，他们才慢慢的回过神来。<br/>　　<br/>　　    “真是令人意想不到，当年马丁鲍曼是如何知道藏区喜马拉雅山脉深处，会有这样一个地方，是他之前就知道，还是运送珍宝过来时偶然碰到的，完全令人想不通啊”三井美奈很遗憾，那个笔记本中关于马丁鲍曼经历的后半段的记载被人撕去了，他们可能永远都无法得知几十年前二战要结束时，马丁鲍曼是如何跋涉千里来到藏区的了。<br/>　　<br/>　　    在天坑周围呆了片刻，夕阳已经开始逐渐朝着山头的那一边落去了，三井良河等人在天坑周围寻觅了一阵，想要找到下去的通道，从峭壁顶上到底部，他们只能循序渐进的走下去，根本不可能采用直降的方式下到底部的。<br/>　　<br/>　　    于此同时，在天坑的另外一头，半山腰处有个狭窄的山洞，洞内地面杂乱的印着几个脚印，洞内由于温度过低气温偏差太大已经结起了冰层。<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
