　　“夜家的人，够狠！”
<br/>　　楚风眠听完邓灵副院长说完的这一切，心中也是冰冷无比。
<br/>　　看来这夜家，是无论如何也想要将楚风眠置于死地，才会想起这种计谋来。
<br/>　　要是一旦是楚风眠被打入奸细，那就算是这一次，楚风眠的师尊玄北圣者出手，都无法救下楚风眠，甚至是可能会被夜家的人，一起对付。
<br/>　　夜家的人，在这北邙学院中，的确足以算得上是一手遮天的存在，不过倒是也有着像是楚风眠的师尊，玄北圣者，活着这邓灵副院长这样的人物在。
<br/>　　倒是也让楚风眠，安心不少，起码说整个北邙学院，还没有全部落入到夜家的手中去。
<br/>　　“夜家的人，定然不知道我现在有着如此实力吧！想要对付我，很好，看来这一次就是撕破脸面的时候了，人善被人欺，不展现出足够的实力来，夜家随时随时都会欺压于我！”
<br/>　　楚风眠眼神冰冷到了极点。
<br/>　　夜家，尤其是那夜牧，是楚风眠的必杀之人。
<br/>　　之前在武斗大会上，此人便是不顾规矩，要强行轰杀楚风眠，救走夜天君，现在反倒是倒打一耙，来追究楚风眠的责任。
<br/>　　这样的人，不杀，也难解楚风眠的心头只恨。
<br/>　　“就算是这些人不利于我，我也可以强行回到云羽天宫之中去！就算是在北邙学院的北邙宫中，我也可以随时回到云羽天宫，倒时候有着云老的存在，这些人不可能杀的了我！”
<br/>　　楚风眠心中已经是算计好了自己的退路。
<br/>　　这一次若是学院高层，还有夜家的人，一定是要杀了楚风眠的话，那楚风眠便是就要来一个鱼死网破也不足惜。
<br/>　　至于向夜家俯首称臣？以楚风眠的性格，他就算是宁死也不会如此。
<br/>　　“我都已经知道了，走吧。”
<br/>　　楚风眠看着邓灵副院长，平静的开口道。
<br/>　　“哎，小心一点吧。”
<br/>　　邓灵副院长看着楚风眠的脸色，也自然清楚，像是这样的一代天骄，那一个愿意去低下头颅来。
<br/>　　不过他也只能劝解楚风眠，毕竟夜家要对付的人，还是楚风眠，他也不想要惹祸上身。
<br/>　　邓灵副院长凝聚灵力，突然打出了一道独特的符文，这独特的符文，只属于北邙学院，是北邙学院的一种特殊标记。
<br/>　　随着这符文，打入到空中，面前突然出现了一道通道，在这通道的背后，一片混沌。
<br/>　　“这是前往北邙宫的法决，你现在的实力，已经是不亚于寻常的碎命之境，自然也有着进入北邙宫的资格了，好好看着，日后也可以随来北邙宫。”
<br/>　　邓灵副院长看着楚风眠，故意放慢了速度，让楚风眠记住这符文的模样。
<br/>　　随后通道打开，他便是率先一步，步入其中。
<br/>　　楚风眠则是紧跟其后，一样的步入了其中。
<br/>　　通过隧道。
<br/>　　楚风眠跟随邓灵副院长，便是来到了一处极为庞大的宫殿之中，这宫殿中，处处充满着古老混沌的气息。
<br/>　　“混沌之气！”
<br/>　　在这宫殿之中所充满的，正是最为古老的混沌之气，比起楚风眠曾经在玄北圣者洞府之中，所见到过的恒古气息，还要古老。
<br/>　　像是生死九重，想要突破，便是就要参悟本源，混沌，正是一切的本源。
<br/>　　在这北邙宫中修炼，便是比起外面，要快上千倍万倍。
<br/>　　“果然，北邙学院的那些老古董，平日里不出现在外面，都是躲藏在了这北邙宫中修炼。”
<br/>　　楚风眠心中暗自想到。
<br/>　　像是北邙学院，明面上超越生死的老古董，也仅仅只有四五位，对北邙学院这样称霸北域的势力来说，显然不可能就是这般弱小。
<br/>　　如今来到这北邙宫中，楚风眠一眼望去便是看上了上百位身影，这些人，每一位都是超越生死的人物。
<br/>　　一共上百位超越生死的武者，这数量，倒是跟楚风眠想象之中的差不多。
<br/>　　北邙学院万年积累，自然也足以培养出了一些属于他们的强者，这便是北邙学院的底蕴。
<br/>　　若是这些人全部陨落的话，北邙学院也可以就此灭亡了，再也无法从九域立足。
<br/>　　“楚风眠，还不速速跪下，当着无数老祖的面，将你的罪孽，一个个都给说出来！”
<br/>　　楚风眠刚刚踏入到北邙宫的一刻。
<br/>　　在周围的王座之上，一道尖锐的声音，突然喊出，正是那夜牧，他看着楚风眠，眼神充斥着杀意，恨不得现在便是就要将楚风眠彻底轰杀一样，疯狂的如同一尊魔头一般。
<br/>　　“跪下？罪孽？我楚风眠，何罪之有？”
<br/>　　楚风眠听到那夜牧的话，不咸不淡的开口道。
<br/>　　“夜牧，之前在武斗大会上，你曾经不顾规矩，出手轰杀我，真正有着罪孽的人，也是你！还让我跪下？我看要跪下的人，是你！”
<br/>　　楚风眠指着那夜牧，字字如雷。
<br/>　　楚风眠的这一个举动，直接将在他的身边的邓灵副院长，吓了一跳。
<br/>　　本来他带楚风眠前来北邙宫，也是让楚风眠稍微低头，好让夜家的人平息怒火，如此一来，夜家的人想要在对付楚风眠，也找不到借口。
<br/>　　可现在楚风眠的开口，居然是丝毫没有将夜家的人，放在眼中。
<br/>　　“好你一个楚风眠，你自己犯了什么罪孽？莫非还不知道？你为我北邙学院，惹来了多少麻烦！现在七大宗门的人，都满是怒火！你现在速速将一切奇遇都交出来，然后在主动前往七大宗门，自裁谢罪，这才是你现在应该做的！”
<br/>　　又是一道声音，突然响起，乃是那陆家的老祖，陆千刃。
<br/>　　楚风眠之前轰杀陆画的时候，便是他突然出手，救走了陆画，当时还想要轰杀楚风眠。
<br/>　　也幸好是被玄北圣者所救，当时楚风眠才可以转危为安。
<br/>　　看来这一次，楚风眠的仇人，倒是都聚集到了一起来了。
<br/>　　“来到这北邙宫，你居然都不知悔改，你都不将诸位老祖，放在眼中了，很好，玄北不来管教你，就由老祖我来！”
<br/>　　陆千刃猛然怒喝一声，他的身躯从王座上，突然飞了出来，凝聚灵力，一掌便是冲着楚风眠轰击过来。
<br/>　　他便是依仗身份，来轰击，他身为老祖，自然有身份教训弟子，这突然出手，周围的其他老祖，都没有任何反应。
<br/>　　楚风眠看着陆千刃，嘴角一动，一字一字的开口道。
<br/>　　“陆千刃，你找死？”
