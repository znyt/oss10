　　猎妖殿，闭关室内。
<br/>　　萧逸盘膝坐着，并没有做特别的修炼。
<br/>　　而是在吸收武道力量。
<br/>　　武道力量，吸收，并能储存于小世界中多少，与修为境界有关。
<br/>　　他现在仅是天元七重巅峰修为。
<br/>　　霸道剑道，以及寒冰剑道，两种武道的天地武道力量，早在上次在极寒裂缝的洞穴修炼时，就已经吸收到上限了。
<br/>　　哪怕他现在再怎么修炼吸收，也不会有任何作用。
<br/>　　只有将修为提上去，达到天元八重、九重时，才能吸收更多。
<br/>　　当然了，达到天元九重巅峰时，就可以考虑将其中一种武道力量吸收至‘一成’，晋升极境了。
<br/>　　故，他现在在闭关室中闭关，吸收的，是其它武道的天地力量。
<br/>　　正是紫晶灵炎、十界灭生火、地脉金火、星辰之火，四种世间强悍火焰的武道力量。
<br/>　　不错，这四种火焰，在天地间，也存在着武道力量。
<br/>　　属于火属性武道。
<br/>　　像上古时期的‘天火圣者’，他修的，便是星辰火道。
<br/>　　之所以说武道万千，拥有无数种。
<br/>　　正是因为武道力量便是这个世界的规则力量。
<br/>　　这世间所存在的一切，都因规则而化。
<br/>　　自然的，世间万物，都在天地间拥有着武道力量。
<br/>　　“吸。”萧逸心头低喝一声。
<br/>　　天地间，属于这四种火焰的武道力量，顷刻间涌来。
<br/>　　霎那间，四种肉眼难视的，飘渺无形的力量，充斥了整个闭关室。
<br/>　　紫色的、青色的、金色的、青白相间的。
<br/>　　缤纷而绚烂的色彩，在这闭关室中如精灵般舞动着。
<br/>　　它们看起来是那么的玄妙、完美，又充斥着深奥的天地至理。
<br/>　　这就是武道力量。
<br/>　　萧逸一个念头下，四种武道力量，悉数涌来。
<br/>　　最后储存于小世界中。
<br/>　　加起来，他现在小世界中，共有六种武道力量。
<br/>　　六种武道力量的数量，皆是一样，强弱也在伯仲之间。
<br/>　　若想增加数量，只有提升修为，继续吸收，这一途径。
<br/>　　说起来，正常武者，若无特殊因素，一般不会修多重武道。
<br/>　　因为，参悟一种武道，再到吸收武道力量，是非常难之事。
<br/>　　每多修一种武道，会生生将自己的修炼速度拉慢，修炼时间，也凭空增加一倍以上。
<br/>　　越多，速度越慢，耗时越久。
<br/>　　可对萧逸而言，他没有这个阻碍。
<br/>　　唯一阻碍他的，只是填满庞大的小世界。
<br/>　　小世界，反正是要填满的，武道力量吸收方面，又毫无阻碍。
<br/>　　自然的，武道种数，多多益善。
<br/>　　历史上，出现过的火之一道强大武者，多不胜数。
<br/>　　其中，掌控强大火焰的，也不在少数。
<br/>　　紫晶灵炎、地脉金火、星辰之火等等世间强悍火焰的掌控者，也都出现过。
<br/>　　他们仅仅掌握一种，在修炼到高深地步后，已有滔天之能，成就不世威名。
<br/>　　萧逸四种齐修，日后真正武道有成，其战力，必定强得发指。
<br/>　　时间，渐渐过去。
<br/>　　三天后，萧逸停下了修炼，彻底出关。
<br/>　　离开猎妖殿，化出紫炎火翼，振翅离去。
<br/>　　目的地，极寒之地。
<br/>　　……
<br/>　　以萧逸现今实力，就是横跨整个炎武王国，也要不了几天时间。
<br/>　　一路赶到冰雪郡，不过耗时一天多些。
<br/>　　出了冰雪郡，越过冰雪圣山，便是极寒之地的范围了。
<br/>　　这边的六座猎妖殿，均衡分布于极寒之地的数十座城池中。
<br/>　　平均七、八城范围，就会有一座猎妖殿。
<br/>　　而这六座猎妖殿的主殿，正好位于不夜城、雪刀城等大城的中央。
<br/>　　当然了，极寒之地这边，太过辽阔了。
<br/>　　真正长期有人类武者足迹的地方，不足百分之一。
<br/>　　对在这里历练的武者而言，真正安全的地方，只有那数十座大城，以及猎妖殿。
<br/>　　若是某些武者离这些地方太远，便只能自求多福了。
<br/>　　这边，终究是妖兽横行。
<br/>　　此刻，萧逸一路飞行，往不夜城而去。
<br/>　　不过一天多些，已穿越了不夜城。
<br/>　　萧逸微微停下了飞行，俯瞰着这座伫立于冰天雪地中的庞然大物。
<br/>　　上次来时，他还为此而赞叹，为此而心惊。
<br/>　　可此时再看，这座大城，在他眼中，却不过如此。
<br/>　　他若看不顺眼了，挥手可灭之。
<br/>　　咔咔，萧逸握了握拳头。
<br/>　　这就是实力所带来的变化。
<br/>　　实力为尊，是这个世界奉行的最高法则。
<br/>　　跨过不夜城，萧逸飞行了数个时辰，已来到这边的猎妖殿主殿附近。
<br/>　　同时，再过不远，便是雪刀城以及冰武城与冰炎城的势力范围了。
<br/>　　一宫二门，皆在寒冰主殿附近。
<br/>　　萧逸远远地，便已能看到那座气派而庄严的猎妖殿。
<br/>　　“嗯？”萧逸忽然眉头一皱。
<br/>　　远方，猎妖殿之外，竟有几个熟悉的面孔。
<br/>　　“是他们？”萧逸心头一惊。
<br/>　　……
<br/>　　寒冰主殿之外，此刻竟是聚集了极寒之地这边过半以上的大人物。
<br/>　　寒冰殿主、不夜宫主、雪刀门主、冰武教主等强者，皆在此地。
<br/>　　而同时，竟有数人，与他们对峙着。
<br/>　　何人如此大胆，竟敢在极寒之地与他们过不去？
<br/>　　看真切些，竟是秦飞扬、林劲与柳嫣然三人。
<br/>　　“三只小老鼠，这次跑不掉了吧？”寒冰殿主冷冷地看着秦飞扬三人。
<br/>　　“这近一月时间，你们大肆杀戮我寒冰猎妖殿的执法队和猎妖师。”
<br/>　　“犯下如此滔天大祸，就是雪翼雕一族也保不住你们。”
<br/>　　“不错。”不夜宫主等人，冷声道，“一月时间，我们一宫二门死在你们三人手上的弟子，不下千人。”
<br/>　　“雪刀门少门主风花月，惨死于你们手上。”
<br/>　　“冰炎教教子韩苍，被你们伏击，至今仍重伤昏迷。”
<br/>　　“如此大仇，今日必将你们三人挫骨扬灰。”
<br/>　　“还挫骨扬灰？”秦飞扬不屑一笑。
<br/>　　“你们是不是忘了，这片极寒之地，谁才是霸主。”
<br/>　　“想捉我们？做梦吧。”
<br/>　　说罢，秦飞扬猛地化出武魂。
<br/>　　一只紫色大雕，凭空而现。
<br/>　　林劲、柳嫣然二人，连忙跳上。
<br/>　　三人御空飞起，立刻逃离。
<br/>　　“哈哈哈哈，记住了，你们害死我萧逸兄弟，此事没那么容易了结。”
<br/>　　“一宫二门，寒冰主殿，终有一日，我要将你们屠个干净，血债血偿。”
<br/>　　秦飞扬杀意凛然地大喝着。
<br/>　　孰料，一张雪白的巨网，却猛地出现。
<br/>　　并瞬间将他们三人束缚。
<br/>　　三人，竟在半空中，瞬间跌落回地面。
<br/>　　“哈哈哈哈，血债血偿？”寒冰殿主冷笑一声。
<br/>　　“我等足足准备了一月，今日岂会让你们离去。”
<br/>　　“紫翼雕在极寒罡风中飞行速度虽快，我等极境强者也难以追上。”
<br/>　　“可被我的寒丝天罗网罩住，你插翼也难飞。”
<br/>　　“该死。”秦飞扬脸色大变。
<br/>　　“秦飞扬是吧。”寒冰殿主狰狞一笑，“听说你原本只是炎武王国北山郡那等小地方的一个废物。”
<br/>　　“却不曾想，竟是武魂变异，乃是紫翼雕武魂，位列淡紫色品阶。”
<br/>　　“而且还是唯一一只紫翼雕，深受雪翼雕一族族长看重。”
<br/>　　“但，仅仅如此，你便忘了自己的出身何等低微了吗？”
<br/>　　“谁给你的胆子挑衅我们？”
<br/>　　“没记错的话，北山郡那个小地方，出过最厉害的，是一个姓易的疯子。”
<br/>　　“可他也死了。”
<br/>　　“记住了，小地方便是小地方，卑微便永远是卑微。”
<br/>　　寒冰殿主的声音，阴寒无比。
<br/>　　那狰狞的脸色，一如当日推萧逸落极寒裂缝那般。
<br/>　　“不知尊卑，挑衅我等的下场，便是死。”寒冰殿主，眼中杀意凛然，一掌轰出。
<br/>　　“不好。”秦飞扬等人，脸色大变。
<br/>　　恰在此时，一道火焰流光，从远方天际疾速袭来。
<br/>　　无尽冰冷罡风，竟是未能挡他分毫。
<br/>　　人未至，漫天紫炎，已悉数打来，拦下了寒冰殿主。
<br/>　　一秒后，人影落下，一掌轰出。
<br/>　　寒冰殿主，连退数步。
<br/>　　冰冷的声音，从人影口中吐出，“我北山郡武者如何，还轮不到你肆意评论。”
<br/>　　来人，正是萧逸。
