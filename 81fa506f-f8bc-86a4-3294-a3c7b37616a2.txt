　　鞠文泰病了，病的很重，在宫城之上看完大唐军队的全部阵容之后，老鞠连城头都没下来，问了一句话之后就晕了过去，不知道是吓的还是什么其它原因。
<br/>　　慌乱无措间，高昌丞相马万杰力排众议，请出了老鞠的儿子——鞠智盛，由他来主持大局。
<br/>　　而与此同时，马万杰又在私下里召集了高昌的几位统兵将军，地点依旧是当初与王玄策见面的那间酒店之中，只不过这一次酒店被彻底包了下来，每一个房间都安排人手防御，整个酒店一个外人都没有，包括老板都被赶了出去。
<br/>　　“丞相大人，如此郑重到底所为何事？”几位将军被马万杰如此严密的布置吓了一跳，还以为他想要杀人灭口啥的，个个面色凝重，连大气都不敢喘。
<br/>　　“几位将军，事关重大，还请不要见怪。”马万杰对门口的侍卫打了一个眼色，示意把门关上，然后正色说道：“国主现在身体堪忧，已经一连四日昏迷不醒，城外有大唐数万军队，城内有其他诸国排挤，内忧外患之际，将军们可有何良策？”
<br/>　　“良策？”一个满脸胡子的家伙冷哼一声：“有什么良策？不说其他诸国，单单就城外那几万唐军，谁有办法安抚他们？谁有能挡住他们？”
<br/>　　“是啊，唐人远来必有所图，可偏偏他们还是国主给请来的，唉！”一个长的比较帅气的家伙叹气说道。
<br/>　　这家伙姓田，叫田长平，他老子读过一些中原书籍，特别崇拜武安君白起，所以给儿子起名叫田长平，用以纪念。
<br/>　　而田长平也没有辜负老爹的期望，长大之后颇有些军事天赋，现在是高昌兵马大元帅（当然，高昌国内肯定不是这么叫的，反正大家领会精神就好），统领全国两万左右的军队，平日里有些威望，属于智将一类。
<br/>　　“不如联合其他诸国，共同出兵将唐人赶出西域如何？”说话的是高昌王室的同族，鞠文泰的同辈兄弟鞠树凡，主要掌管城防。
<br/>　　“放屁，不等你联合完，估计大唐的军队就能打进城来。还真当别的国家都跟咱是一条心呢？告诉你，别指望了，联合他们说不定咱这边没散会，城外的唐军就已经得到消息了。”大胡子撇撇嘴说道。
<br/>　　“那怎么办？总不能就这么等着吧，谁知道外面唐军得到消息之后会不会有什么其他的企图，一旦他们要图某高昌，谁能挡得住？”鞠树凡作为王亲，被大胡子顶了一句有些气不顺，瞪着大胡子说道，意在指责其‘窝里横’。
<br/>　　“好了，都先静一静，丞相既然召集我们在此相聚，想必一定有了妥善的办法，不如我们听听丞相的计划如何？”田长平伸手压住要跳起来的和鞠树凡拼命的大胡子，故作坦然的看着马万杰，淡然说道。
<br/>　　“不错，老马，有啥话你就说吧，别吞吞吐吐像个娘们儿，没的给你家祖上丢脸。”大胡子被田长平的话吸引了注意力，暂时放过了鞠树凡，却把矛头指向了马万杰。
<br/>　　“老韩，怎么说我们两家也是世交，你说话就不能嘴下留点德？”马万杰一脸苦笑，遥遥用手指点点大胡子，随后换上一副正经的面孔说道：“几位，你们认为大唐是否会在西域长期驻扎下去？”
<br/>　　“长期？那是多久？一年，十年，还是永远？你能不能说的清楚一点。”姓韩的大胡子翻了个白眼，鄙夷的说道。
<br/>　　韩家和马家的确是世交，而且可以往上数不知道多少代，这事儿自打他们两个出生开始就被父辈灌输进脑子里面，时间长了也就不追问往上多少代了，反正他们的父辈也是这么过来的，不知道就不知道吧。
<br/>　　“你别管多久，我问的是你们认为大唐的目的是什么？猜一下，随便猜，哪怕是猜他们明天就会走也可以。”马万杰摆摆手，示意大胡子不要打断他：“说说你们的看法，不要想着大唐是否会打进高昌之类的事情。”
<br/>　　“我觉得大唐一定是另有目的，但却不知道他们的目的是什么，但想来应该是离不开城外的那条商路，所以我觉得他们应该会长期驻扎。”田长平面色平静，一派儒雅，看着去着实有些名将的架式。
<br/>　　“我也这样认为。”鞠树凡赞同的点点头。
<br/>　　“我不知道，你们都这么想，那我也这么想好了。”大胡子性子直，没什么主见，见两人都这么说，他也跟着附和道。
<br/>　　“那你们觉得我们是与大唐合作好？还是对抗好呢？”马万杰目光扫过三人，身子微微前倾沉声问道。
<br/>　　“你什么意思？”田长平眉头一皱，感觉隐隐猜到了些什么。
<br/>　　“什么意思？”马万杰淡淡一笑，将身体重新靠回椅子：“我的意思是：把唐军请进城好呢？还是让唐军打进城好呢？就是这么简单。”
<br/>　　“你想要投降大唐？”大胡子这下总算是听明白了马万杰的话，一时间激动的差点从椅子上跳起来。
<br/>　　“坐下，急什么，听我把话说完。”马万杰比大胡子年长几岁，自然有瞪眼的权利。
<br/>　　田长平同样以目示意大胡子坐下，然后看着马万杰说道：“丞相请说，在下愿闻高见。”
<br/>　　“田元帅，两权相害取其轻啊。”马万杰叹了口气，舔舔嘴唇沉默片刻，似乎在组织语言，半晌之后才说道：“对于西域，大唐到底是一个外来者，如果想管理好西域，他们一定需要一个代言人，那么这个代言人，你们认为谁来做比较好呢？”
<br/>　　“这……”田长平犹豫了一下，陷入良久的沉默之中。
<br/>　　而旁边的鞠树凡也从刚开始的激奋中回过神来，开始思考起马万杰的问题，三位军方人士，只有大胡子没事儿人一样坐在边上东张西望。
<br/>　　他没那个脑子去想这些东西，对于他来说，只要马万杰和田长平认为是对的东西，那就一定是对的，其他人的说法都是扯淡，没有任何意义。
