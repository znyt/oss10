　　收徒大会，如今结束。
<br/>　　这没有被长老挑选看中的武者，便是失去了一步登天的机会，这让不少武者都有着几分沮丧。
<br/>　　“你们这些人没有被各位长老看中，也不用气馁，只要在学院中，安心修炼，将来也是有着晋升内门弟子的机会的，甚至是晋升种子弟子，也有可能。”
<br/>　　苏凌长老看着剩下的人，笑着开口道。
<br/>　　“如今种子弟子之中的一位，欧阳晨便是就算从外门弟子，一步步晋升，成为种子弟子的，甚至是冲入到了天榜之中，你们也不是没有机会。”
<br/>　　苏凌长老的一句话，让下方的不少武者，眼神中又露出了几分斗志。
<br/>　　“不错，我也听闻过欧阳师兄的故事，从外门弟子，一步步晋升，甚至是击败了许多长老的弟子，踏入种子弟子的。“
<br/>　　“对，我们也可以做得到，也可以跟那些长老弟子一起竞争。”
<br/>　　几尊武者，纷纷开口道，一个个重拾自信。
<br/>　　不过这个故事，在楚风眠听来，只能是不由的摇了摇头。
<br/>　　欧阳晨的故事，在北邙学院中，的确是真的不错，但是他的崛起，十万人都不一定可以有一人可以做到。
<br/>　　毕竟这些成为长老弟子，本身便是就是资质极高之人，在加上更多的资源，只怕是察觉会越拉越大。
<br/>　　苏凌长老的这一番话，也只是在众人重拾自信罢了。
<br/>　　“好了，现在你们先去外门住下，已经给你们准备好了洞府，等到过几日之后，门派任务便是就会发放下来，倒时候你们就要真正为门派做事了。”
<br/>　　苏凌长老开口，便是离开。
<br/>　　剩下的几尊北邙学院弟子，便是带着剩下的人，来到了一处山脉之中。
<br/>　　这附近的几座山脉，便是属于外门弟子的区域，期中林立这一座座的洞府。
<br/>　　北邙学院的外门弟子，还是占据了大多数，但是所占有的区域，却是不多。
<br/>　　像是内门弟子，一个人占据的区域，就要比外门弟子要大上百倍，而一个种子弟子占有的区域，更是一尊外门弟子的万倍。
<br/>　　在北邙学院中，一切地位，资源，都是因实力而定。
<br/>　　占据的区域越大，便是也就可以汲取更多的灵气，修炼起来更加快速。
<br/>　　“这些，都是些无主洞府，你们自行分配，一个人只允许占据一个洞府，你们先休息几日，熟读好北邙学院的规矩之后，十日之后，便是前往广场，来等待分配任务。”
<br/>　　北邙学院的外门弟子，不能像是跟内门弟子一样，每日只需要修炼，若是还需要负担任务，每年都有着固定的任务下来。
<br/>　　必须要完成这些任务之后，才可以得到时间修炼，虽然完成任务也可以得到一定好处，但是远不如一直苦修。
<br/>　　不入内门，就连安心修炼都做不到。
<br/>　　那北邙学院弟子，说完便是直接离开了，便是留下楚风眠一群人，站在山脉之中，看着周围的无数洞府。
<br/>　　“哼，我们千辛万苦加入到北邙学院中，居然还需要听从吩咐，去完成什么任务，简直便是如同奴隶一样。”
<br/>　　看到那弟子离开，不少人都是抱怨起来。
<br/>　　他们都是大家族的弟子，如今来到北邙学院中，反倒是被人管束，一个个极为不满。
<br/>　　“别那么多废话了，我们还是潜心修炼，早日晋升内门弟子，才可以甩开这繁重的任务。”
<br/>　　“这些洞府，大小不一，还是快去挑选一个上好洞府才行。”
<br/>　　一尊武者说完，便是直接离开，去选择洞府。
<br/>　　这些洞府，虽然都在一处山脉之中，但是其中的每一个都是大小不一，高低不一。
<br/>　　一些洞府，占据山峰，地势极高，灵气便是浓郁许多。
<br/>　　一些洞府，则是就在一些低洼的角落中，灵气稀薄，每个洞府之间，都有着不小的差距。
<br/>　　“这片山脉中最好的洞府，应该就是哪里。”
<br/>　　楚风眠灵识扫视了一圈，这周围的洞府便是全部被他尽收眼底，楚风眠身形一动，直接便是来到了一处半山腰处的一处洞府。
<br/>　　这一处洞府，虽然不是占据最高点，但是地势也高，并且巨大无比，比起寻常洞府大上四五倍，是这处山脉最好的洞府。
<br/>　　在这处洞府中修炼，便是比起寻常洞府，要快上四五倍。
<br/>　　“刚刚入门，倒是只能这样了。”
<br/>　　感受着周围的灵气，楚风眠自言自语一句。
<br/>　　这样的洞府，的确远远满足不了楚风眠，楚风眠知晓，这处山脉已经是距离那灵泉比较远的区域了。
<br/>　　靠近灵泉的几座山脉，都是种子弟子，甚至是核心弟子的区域。
<br/>　　还远不是楚风眠可以触及的到的。
<br/>　　现在他只能在这处山脉活动，只能是寻找这山脉最好的一座洞府。
<br/>　　楚风眠说完，便是直接开始布置起洞府中的种种来，必须要从中修炼，自然需要布置一些阵法，禁制等才行。
<br/>　　“哦？居然有人占据了这处洞府吗？”
<br/>　　就在楚风眠布置的时候，洞府外却是突然传出来了声音。
<br/>　　听起来是两个人，结伴前来，目的也是如今楚风眠这个洞府。
<br/>　　楚风眠的这个洞府，可是整个山脉最好的一个，自然也有不少人，也想要打这个洞府的注意。
<br/>　　“居然只是一个御风境的小子。”
<br/>　　其中一名中年男子，看着楚风眠，不屑的冷哼一声。
<br/>　　“这样的废物，也敢占据最好的洞府，真是不知死活。”
<br/>　　“小子，快滚出来！”
<br/>　　他突然怒喝一声，直接一拳轰出，将楚风眠刚刚布置到一半的禁制，直接打破。
<br/>　　“恩？”
<br/>　　听到这动静，楚风眠回头，便是看见两尊中年男子。
<br/>　　这二人，楚风眠倒是有一点印象，刚刚开口那人，名为牛峰，算是这一届新人中的强者了，实力接近神力境九重的地步，只是资质不高，没有被长老看重。
<br/>　　而另外一人，则是马允，实力跟牛峰极为接近。
<br/>　　这二人的实力，本来都是位列与这一届新人的巅峰，如今居然联起手来。
<br/>　　只怕是这一届新人中，无人能敌。
