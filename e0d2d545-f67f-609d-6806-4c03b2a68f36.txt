 陈兴原本还想问沈光明为什么会被流放到这里，但话到嘴边，又咽了回去。因为这种个人私隐只有团队首领才会问。沈光明是聪明人，问得太多反而容易引起警惕。<br/>    <br/>    由于沈光明的存在，原本熟悉的雷光团蒙上了一层神秘的面纱，让他看不透、摸不清。<br/>    <br/>    没想到底层出身的铁诺，沦落到这般绝死之地，却还能得到白塔学者的支持，实在令人唏嘘感慨。<br/>    <br/>    回到营帐后，铁诺邀请陈兴和阿乔木共进晚餐。陈兴却之不恭，一起吃了顿行军餐。席间，铁诺对阿乔木引魔灵进营地的事情只字不提。这就是为什么铁诺能做老大，火咀只能做老二的原因。<br/>    <br/>    火咀性格豪爽、讲义气、爱憎分明，却不会转弯，容易捡了芝麻丢西瓜。有时候为了大局，是要懂得取舍的。真正的团队领袖，就是在保持团队精神的前提下，带领团队获得最大利益。<br/>    <br/>    不过话又说回来，陈兴虽然敬重铁诺，却不想成为铁诺那样的人。因为那样的人，太累了……<br/>    <br/>    吃完晚餐，铁诺安排了一个较大的帐篷给陈兴和阿乔木休息，并派了一个勤务兵服务他们。这个勤务兵就是之前要求火咀加薪的那个小年轻，性格有些腼腆，又有些许跳脱。<br/>    <br/>    按照约定，每十二个小时四个战团的团长会进行一次例会，下一个例会在八小时后，到时候陈兴也会去参加。<br/>    <br/>    由于阿乔木刚睡醒，陈兴也没什么睡意，于是聊起了天。阿乔木性格内向，不怎么爱说话，所以陈兴的主要聊天对象是那个勤务兵。<br/>    <br/>    “你是哪里人？”<br/>    <br/>    “龙涎河红龙公国的。”勤务兵双手拧在一起，显得有些紧张。<br/>    <br/>    “哎哟，还是老乡啊，我也是红龙公国的，龙石镇那边的。”陈兴说道。<br/>    <br/>    “我是大山岭那边的，木兮镇，听说过吗？”勤务兵眼睛一亮，带着几分兴奋问道。<br/>    <br/>    “去过，‘山有木兮木有枝，心悦君兮君不知’，你们镇长是个祖籍东大陆的云游诗人。因为诗写好，得到了公国王后的赏识，被破格提拔为镇长。”陈兴根据上一世的记忆，说出了木兮镇名字的由来和镇长的过往。<br/>    <br/>    “哎哟，真是老乡啊！”勤务兵用力一跺脚，显得激动无比，“大哥，去过桃木林吗？”<br/>    <br/>    “两棵桃树而已，也敢说是树林。”陈兴不屑地说道。木兮镇的镇长有着非常浓烈的文人酸腐，在镇长府的后花园里种了两棵桃树，就对下面的人说，“双木为林，以后这里就叫桃木林。”<br/>    <br/>    自古以来，文人和武夫就互相看不起，所以木兮镇的趣闻一直是佣兵们茶余饭后的消遣。每次说起，都有一帮兵痞笑得前仰后翻。<br/>    <br/>    “那腐儒就是这样的，有时候还跑到喷水池边上弹琴，说要提高大伙的文化素养。但大伙都听不懂，还不如送把白菜来得实在点儿。”勤务兵乐呵呵地说道。<br/>    <br/>    “怎么样，家里的人都还好吧？”“都还好，能吃能睡。”“有没有讨老婆啊？”“还没呢，不过有女朋友，她叫小莹，比我小一岁……”<br/>    <br/>    “班花啊，可以啊！”陈兴作出一副吃惊的样子，旁边的阿乔木不易觉察地翻了下白眼。<br/>    <br/>    “我还带着她的照片呢。”说到自己的女朋友，勤务兵的目光变得骄傲无比，手忙脚乱地拿出项链，打开圆形的吊坠给陈兴看。<br/>    <br/>    照片中的女孩小巧精致，脸上带着甜美的笑容，有个七八分的模样。光看脸蛋的话，虽然比不上苏雪菲，但比起旁边的阿乔木还是有些优势的。这种清纯可爱的类型，对于情窦初开的小男生特别有杀伤力，因为这个阶段的男生还懵懵懂懂，没有体会过韵味和身材的重要性，以为女人就是拿来看的。<br/>    <br/>    但如果放在陈兴这里，估计这种小女生他连看都懒得多看一眼。<br/>    <br/>    “嗯，长得不错，很漂亮。”陈兴说道。<br/>    <br/>    “小莹是我们班里最漂亮的，连邻班的男生都跑来追她，但她只喜欢我……”勤务兵得意地说道。<br/>    <br/>    不一会儿，陈兴就拉近了他和勤务兵之间的关系，然后开始旁敲侧击地打听沈光明的事情。<br/>    <br/>    “我加入的时间不长，以前的事情都不知道，但我听火咀老大说过，沈参谋和铁诺团长是一起过来的，他们应该很早就认识了。”勤务兵说道。<br/>    <br/>    “听说沈参谋是大学士，怎么会沦落到这里来？”陈兴又问道。白城学者的地位非常高，仅次于快速治疗师，就算犯了重罪，也很少被流放。大部分都关押在监狱里，以便于利用。<br/>    <br/>    “不知道呢……”勤务兵摇了摇头，“沈参谋很少和我们接触，大部分时候都在大本营里，听说身体不是很好。”<br/>    <br/>    “这样啊……”陈兴沉吟了一句，然后扯开话题，拉回关于对方女朋友的正轨上。又聊了一会儿，陈兴就把勤务兵打发走了。<br/>    <br/>    勤务兵走后，阿乔木起身搭了下陈兴的肩膀，说道，“我怎么觉得，你也有些好奇。”<br/>    <br/>    “这个沈光明不简单，如果我们想要在接下来的行动中获得利益，就要多点儿了解……”陈兴压低声音，意有所指地说道，“我们潜在的对手。”<br/>    <br/>    “嗯哼~”阿乔木发出一声轻佻的鼻音，“怪不得老沙民们总说，狐狸最不喜欢的就是狐狸。”<br/>    <br/>    <br/>
