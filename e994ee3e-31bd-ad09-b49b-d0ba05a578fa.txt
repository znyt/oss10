<br/>				两人用了不过一柱香左右的时间，便将那十七头妖兽一一砍杀，看着那一地的血腥，两人越了过去，走向那个小结界所在的地方。<br/>　　<br/>　　走近一看，两人才看见先前被杂草挡着没看到下方，这时才看到，这个小孩原来并不是在地面上，而是在一个像小碗一样的法器里面。<br/>　　<br/>　　“这是什么法器？看不出材质。”凤九看着那结界里小孩身下的东西看着。<br/>　　<br/>　　“先把结界破了，拿出来瞧瞧不就知道了。”老头说着，示意她动手。<br/>　　<br/>　　见此，凤九却没动手，而是盯着那小孩看着，问：“老头，你说这小孩应该没问题了吧？应该不是妖兽幻化的吧？”<br/>　　<br/>　　听到这话，混元子忍不住的哈哈一笑，瞅了凤九一眼，瞧着她一脸的纠结，笑眯眯的道：“怎么？一朝被蛇咬，十年怕草绳了不成？”<br/>　　<br/>　　“这倒不是，只是，要是再抱起来又变成了妖兽，那感觉实在是不太美妙。”凤九拧了拧眉头说着，却还是伸手结出印记，灵力一涌，将结界破开。<br/>　　<br/>　　也许是哭得太久，婴儿啼哭的声音听起来略显沙哑。凤九看着那个舞动手脚的孩子，见他身上只穿着一件小肚兜，包在外面的一件毛毯已经被踢开垫在身下，看着这个粉嫩嫩胖嘟嘟的孩子，她弯腰抱了起来。<br/>　　<br/>　　入手软绵绵沉甸甸的，她一手托着，掀开了肚兜的一角瞥了一眼，笑道：“是个男孩子。”<br/>　　<br/>　　说着，她看向那正拿起那件法器在看着的混元子，略有些无奈的问：“我怎么感觉我跟这些小奶娃和小奶兽都比较有缘？一出门就捡一个。”<br/>　　<br/>　　“反正你也还没捡过小奶娃，这个跟你遇上也是有缘，别说养那些契约兽类了，就是再养几个小奶娃你也养得起，这孩子眼下这样，估计着，以后还真得你养着了。”<br/>　　<br/>　　混元子说着，看了看那到了凤九怀里后就没再哭的婴儿，打趣的道：“你看，这孩子到你手上就不哭了，好像知道你是他娘亲似的。”<br/>　　<br/>　　闻言，凤九额头划过几道黑线：“我还没成亲呢！”<br/>　　<br/>　　“没成亲又怎样？没成亲就不能收养孩子了？看得顺眼就留在身边养着，又不是多大的问题。”<br/>　　<br/>　　老头不以为然的说着，将那毛毯和那件东西递给凤九，道：“照我看，这孩子的来历估计不普通，那先前我们见到的尸体，估计应该不是这孩子的父母。”<br/>　　<br/>　　“嗯？怎么说？”她挑了挑眉，有些微讶。<br/>　　<br/>　　“这件东西是仙器，可不是寻常的法器，这上面有图纹，只是，就是老头我也看不出这是出自哪里的仙器，但是，这件仙器上面却是无主之物，只是用来这样给这个孩子当睡觉的小床，可见，这孩子的父母必定不凡。”<br/>　　<br/>　　混元子的声音一顿，抚着胡子道：“据老头我的推断和猜测，那些死去的人应该只是照顾这婴儿的下人，下人带着小主子又到了这样的地方，我想，肯定是为了躲避仇家。”<br/>
