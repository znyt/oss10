这一次进入禁地并没有找到宝箱，算是失败了，但也不算是无功而返，因为夏夕绾看到了狼还有两个幻象，她需要翻阅相关的典籍来寻找线索。
<br/>
<br/>        夏夕绾将禁地里发生的事情跟夏邦还有蓝烟说了，夏邦惊道，“绾绾，你是说你们遇到了狼？”
<br/>
<br/>        “是啊。”夏夕绾点头。
<br/>
<br/>        “不应该啊，为什么科学院电台的后院里会养着两头狼，狼是十分凶猛可怕的兽类，除了训狼秘术，没有人可以降服它们。”
<br/>
<br/>        夏夕绾心里一动，“夏爸，什么叫训狼秘术？”
<br/>
<br/>        夏夕绾第一次听说“训狼秘术”这个词。
<br/>
<br/>        夏邦看着夏夕绾，“我也是听说的，听说曾经有一个少女身怀训狼秘术，她骑在狼背上奔腾，所有的狼都听她号令，以她为王。”
<br/>
<br/>        夏夕绾想起了昨晚那两头狼曾经单腿跪在了她的面前，虔诚的向她拜服过，很像是将她当成了主人。
<br/>
<br/>        可是，她并不会训狼秘术啊。
<br/>
<br/>        夏夕绾想了一下，“夏爸，蓝妈，其实有一个问题我一直很好奇，你们知道我妈咪来自于哪里吗？我一直以为我妈咪来自于帝都城，但是来到这里我才发现帝都城的人都不知道我妈咪来自于哪里。”
<br/>
<br/>        蓝烟摇头，“绾绾，我们跟你一样，都不知道小姐来自于哪里，她是突然降临帝都城的，她也从来没有跟我们说过她来自于哪里，不过我总觉得小姐是来自于某个神秘的国度。”
<br/>
<br/>        这样啊…
<br/>
<br/>        夏夕绾拧起了秀眉，“现在谜团越来越大，但是我总觉得…我已经知道了些什么，只要我能抓住一个点，我就可以亲手揭开这层层迷雾，可是，我现在需要线索，你们知道这个帝都城有什么地方卖消息吗，类似于包打听的那种。”
<br/>
<br/>        现在每个城市都有一个类似于包打听的地方，卖各种各样的信息，而帝都城藏龙卧虎，应该也有的。
<br/>
<br/>        果然夏邦点了点头，“有，帝都城有一个七公子。”
<br/>
<br/>        “七公子？”夏夕绾双眼一亮，“这是什么人？”
<br/>
<br/>        “我们也不知道他是什么人，这个人从来没有露面过，但是据说他手上掌握了整个帝都所有的秘密，每个人身上都有明有暗，那些暗色的地方被我们小心翼翼的藏了什么，并不想让外人知道，这个七公子却掌握了所有人的秘密，据说他手下的眼线遍布在整个帝都的每个角落里，一个贩夫走卒，一个豪门太太，都有可能是他的人。”
<br/>
<br/>        这么厉害？
<br/>
<br/>        夏夕绾突然想起了范恬，范恬喜欢夜老这算是深藏在她心底的秘密了吧，这世上应该没有人知道的，但是苏希知道。
<br/>
<br/>        苏希？
<br/>
<br/>        苏希！
<br/>
<br/>        夏夕绾颤了一下纤长的羽捷，上一次她就知道苏希最擅长玩弄人心，是一个神秘而强大的存在，那他和这个七公子有没有什么关系？
<br/>
<br/>        夏夕绾突然有一个很大胆的想法，苏希是不是…就是七公子？
<br/>
<br/>        妈咪来到帝都城之后，收了几个徒弟，这几个徒弟都是医学有天赋的人，苏希是妈咪的关门小弟子，他是不会医术的，那他会什么，妈咪交给了他什么？
