<br/>    金色的龙爪落下，将那个异人瞬间穿透，异人倒下，到死眼中还是欣喜的神情，五人看向这诡异莫测的龙君，分不清他到底是敌是友。
<br/>    
<br/>        刘玄刹传音道:“这龙君恐怕是君主级，我们要想办法立刻逃走，君主级的力量，根本不是我们可以对抗的。”
<br/>    
<br/>        陈明心说，不用恐怕了，就是君主级！
<br/>    
<br/>        陈明也是传音道:“我放下一座传送阵，通往燕山，只要战斗一开始，我们就直接传送走！”
<br/>    
<br/>        几人看向陈明，不是，你还有这种骚操作？
<br/>    
<br/>        随时随地开阵法传送走？
<br/>    
<br/>        不愧是修了假仙的人！
<br/>    
<br/>        龙君的面庞清秀，身材也并不魁梧，但是五人都不敢小视龙君，龙君睁着金色的瞳子审视着五人，五人只感觉有凶兽在窥视着他们，“你们几个是来杀祖龙的？”
<br/>    
<br/>        陈明瞬间开口道:“不是！我们是来拜访祖龙的！听说他老人家过寿，我们特地前来送礼！”
<br/>    
<br/>        开玩笑，这龙君这么厉害，我会说实话？
<br/>    
<br/>        龙君淡淡道:“你们之前和龙伥的对话，我听到了的！”
<br/>    
<br/>        陈明无言以对，“你听到了还这么问，是有病吗？”
<br/>    
<br/>        龙君无所谓的摆摆手，然后道:“我只是再确认一下。如果你们是来杀祖龙的，可能需要一番准备，这个老家伙虽然已经快要腐朽了，但是还是有一定的实力的。以你们几个的实力，倒是有那么一丝机会可以杀掉他。”
<br/>    
<br/>        嗯，这个龙君到底是怎么回事？
<br/>    
<br/>        刘玄刹收起了手中的黑色镰刀，问道:“你不是龙伥吗？为什么想要我们去杀了道龙？”
<br/>    
<br/>        龙君道:“你们听到的那个故事，是道龙编造出来的，我这个故事，才是真正的历史。”
<br/>    
<br/>        陈明眉头一跳，这是要走剧情了吗？
<br/>    
<br/>        龙君找了一块青石坐下，也不管几人的表情诧异，缓缓道来，“在八百年前，我进入龙藏之地，愿意奉献己身给道龙，成功的成为了道龙麾下的龙伥，学习了道龙的功法和术法，我不知道的是，道龙给予我的功法中，有一种诡异的效果，那就是会让我为他所控，凡是他说的话，我都觉得是对的，直到三百年前，我才突然莫名的挣脱了这种束缚，而就是在那时起，我才了解了真正的历史。
<br/>    
<br/>        凡是成仙，都需要一颗星辰作为自己的命星，供养自己的修行，而道龙，便是看上了这一片大陆，想要将这一片大陆作为他的命星，所以他来到了这一块大陆之上，杀尽了这一片大陆之上所有的修行者，然后宣扬自己的道，企图让这一片大陆化作他的仙国，可惜的是，在他就要成功的蛊惑所有人，让所有人化身龙伥的时候，蟠龙大仙成仙了。
<br/>    
<br/>        蟠龙大仙搏杀道龙，重伤了道龙，不过蟠龙大仙也是身陨。而后修行者一次次组织起义军，进攻道龙，终于是降服了道龙。
<br/>    
<br/>        道龙身死，化作一条骨龙，他的修为也跌落至大君的境界，已经无力再征服整个大陆，恰好此时大陆之上的那个时代的人已经死绝，所以它便编造了你们听到的那个故事，让它自己化作正义的化身。
<br/>    
<br/>        在数年之前，我曾经听到骨龙说过，他已经沟通了一位好友，名为尸九泉，企图让尸九泉将整片大陆化作鬼界，然后他以骨龙之身重新回到仙人的境界。
<br/>    
<br/>        不过据他所说，尸九泉似乎是险胜了，但是最后却是被众多古仙门合力封印了，而现在，他正在召集众多的龙伥，企图反攻你们古仙门。
<br/>    
<br/>        曾经，我以为邪恶便是滥杀无辜，直到今日我才直到，滥杀无辜并算不上邪恶，真正邪恶的是操控别人的意识，连意识都没有，那么剩下的，唯有一具行尸走肉。”
<br/>    
<br/>        刘玄刹道:“既然你也知道道龙是邪恶的，以你如今的修为，一定是可以战胜道龙的，那你为什么不直接去斩杀了道龙？”
<br/>    
<br/>        龙君道:“我并不是不想杀了道龙，我说过了，道龙能够操控龙伥的思维，越是靠近道龙，这种控制力也就越强，如果我靠近道龙，那你们只有死路一条，因为那个时候的我，已经不再是我，而是龙伥！”
<br/>    
<br/>        陈明倒是懒得废话，直接问道:“那道龙有什么弱点，你既然说我们有机会杀了他，他一定是有什么弱点的吧？”
<br/>    
<br/>        龙君点了点头，道:“不错，在他的胸口处，有着蟠龙大仙留下的一处道伤，你们只要能够抓住它的这个弱点，还是有希望斩杀掉他的。”
<br/>    
<br/>        刘玄刹拱手道:“那就多谢了，既然如此，我们便出发吧！”
<br/>    
<br/>        待得几人准备继续深入龙藏之地，却是听到龙君淡淡道:“对了，遇到其他的龙伥，不必有任何留手，他们已经没有在清醒过来的机会了，他们，都已经堕落成邪恶的化身了。”
<br/>    
<br/>        龙藏之地中，一位大君看向陈明他们的方向，“就连龙君也败了吗？看来这一次的敌人，不可小觑啊！”
<br/>    
<br/>        在他的背后，一条百丈长的白骨长龙双眸之中跳动着蓝色的火焰，“杀了他们！”

<br/>        大君拱手道:“是，祖龙大人，我这就去杀了他们，保护龙藏之地的安宁！”

<br/>    
<br/>        听到陈明的话，几人也是瞬间停住了脚步，刘玄刹深思片刻，“你说的，好像，有那么一些道理啊！”
<br/>    
<br/>        雪寻梅也是道:“既然这里并没有传说中的那么多的经文，那么我们为什么还要去进攻龙藏之地呢？”

<br/>        在龙藏之地的入口，陈明看了一眼躺在青石上喝酒的龙君，拱手道:“既然道龙的龙鳞上并没有经文，我们和道龙无冤无仇，还是算了，告辞！”
