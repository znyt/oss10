　　阴魂丹的效果，便是在一段时间内祛除阳气，能够让慕清霄的体温变得冰冷无比，与寻常鬼族一样。
<br/>　　当然，阴魂丹也只能瞒过普通鬼修。
<br/>　　至于鬼王这种级别的，只要两者一接触，便能察觉出你的来历，更别说轮回路中的老怪物了。
<br/>　　不过，天鬼皇的目标是阴寒域的双煞鬼王兄弟，阴寒域属于鬼界最边缘，碰到那种存在的几率，微乎其微。
<br/>　　将阴魂丹吞入腹中，慕清霄诧异的瞥了眼天鬼皇。
<br/>　　从这种细微的小事情上可以看出，天鬼皇骨子里并不蠢，否则也不可能在这种弱肉强食的世界里活至今日。
<br/>　　在感觉到一股凉意袭来后，慕清霄的气息变得阴冷无比，配上他淡漠的眼神，整个人都显得极为冷酷。
<br/>　　似乎，他就是一名鬼修。
<br/>　　见阴魂丹起到效果，天鬼皇也是松了口气，同时，目光若有若无的瞥向遥远的天际外。
<br/>　　就在下一刻，天际外陡然传来一阵闷雷声。
<br/>　　慕清霄抬头望去，只见原本昏沉的天空，瞬间变得漆黑如墨，只见一片漆黑积云，携着阴风铺天盖地的狂涌而来。
<br/>　　“那是……”
<br/>　　慕清霄剑眉一皱，微微眯着星眸，注视着袭来的黑云，道：“十五名练虚，五名合道巅峰的鬼修，手笔倒是不小。”
<br/>　　慕清霄的灵魂力量何其庞大，只是惊鸿一瞥，就察觉出黑云中的身影。
<br/>　　在黑云中总共存在着二十名鬼修，其中有五名是合道巅峰，另外十五名是练虚巅峰，这种豪华阵容，恐怕鬼王都不可能轻易拿下。
<br/>　　“他们是鬼差，一旦空间门开启就会惊动他们，他们负责检查进入鬼界的人员，这也是本座为你准备阴魂丹的原因。”
<br/>　　话语刚落，慕清霄若有所思，看来六界也不像自己想象中那样平静，暗地里恐怕摩擦不断吧。
<br/>　　“轰轰……”
<br/>　　漆黑如墨的乌云携着漫天阴风而来，最后在头顶的天际上徐徐停下，旋即，一道沙哑且阴冷的声音自黑云中传出。
<br/>　　“阴差黑鳞，见过天鬼皇殿下。”
<br/>　　伴随着沙哑阴冷的声音响起，云层被渐渐拨开，只见一道身穿黑袍的身影掠了下来。
<br/>　　阴差是一名身影消瘦的青年，着装有点像古时候宫中的太监，除了惨白的脸色，阴冷的气息与声音，其他都与正常人一样。
<br/>　　鬼族不像妖族，本身姿态就是千奇百怪的妖兽。
<br/>　　微微行礼过后，阴差黑鳞的目光就在天魑皇，魉妹与慕清霄身上打转。
<br/>　　最后，目光在慕清霄身上定格，他总觉的后者给他的感觉有点不同，于是开口问道：“天鬼皇殿下，不知这位大人是？”
<br/>　　闻言，天鬼皇面色一沉，道：“这位是本座的手下，怎么？”
<br/>　　“不不，小人只是一时好奇。”
<br/>　　说罢，黑鳞眼珠子一转，目光看向天魑皇和魉妹，道：“这两位想必就是天魑皇大人与魉妹大人吧。”
<br/>　　黑燐语气中颇有讨好的意思，他虽是鬼差，但有的人他也得罪不起，比如面前的天鬼皇。
<br/>　　他虽是轮回路中高层指派的鬼差，但权利并不大，哪怕死了，也有别的鬼差来顶替他。
<br/>　　而且，他也是个聪明人，知道什么事情该问，什么事情不该问。
<br/>　　天鬼皇屈指一弹，黑燐抓住袭来之物，见手中是数颗七品鬼丹后，眼底闪过一抹激动之色。
<br/>　　“本座与你大哥白磷也是老相识了，本座在鬼界有点事情需要处理，不知黑燐兄可否通融一下？”
<br/>　　闻言，黑燐悄无声息的将鬼丹收起，献媚一笑，道：“天鬼皇殿下与家兄乃是旧识，不过，希望殿下不要让小人难做。”
<br/>　　“放心，本座只是来拿点东西，往后少不了你的好处。”
<br/>　　见黑燐收下鬼丹，天鬼皇眼中闪过一抹喜意，鬼差的实力虽没有他强，可却是一种及其难缠的存在。
<br/>　　若是将其击杀，绝对会引起鬼界最高层的注意，得不偿失，区区几颗鬼丹，便能避免检查，这一笔交易非常的划算。
<br/>　　说罢，天鬼皇就化作一团黑雾，笼罩着慕清霄三人，向西北方向掠去。
<br/>　　待天鬼皇离去后，黑燐的眼珠子转了转，低喃道：“西北，似乎是双煞鬼王的地盘……不过，跟小人有什么关系呢？”
<br/>　　想到数颗鬼丹，黑燐心里喜滋滋的，化作一道黑影掠向云层中，最终消失不见。
<br/>　　……
<br/>　　……
<br/>　　与此同时，在百里外的上空，黑雾徐徐停下，其中露出四道身影。
<br/>　　避开鬼差的检查后，无论是天鬼皇，或者是天魑皇与魉妹，都是微不可查的松了口气，唯有慕清霄面色如常。
<br/>　　天鬼一族发展至今，已经在岁月的长河中衰落，若是毁在他们手中，哪怕永堕轮回，都不足以抵消他们的罪行。
<br/>　　好在鬼差能够贿赂，若是检查出，慕清霄是人族，那么他们天鬼一族，可真的会面临灭顶之灾。
<br/>　　见状，慕清霄依旧是一脸平静，似笑非笑，道：“鬼差很强吗？需要你们如此谨慎？”
<br/>　　闻言，天鬼皇无奈的摇了摇头，道：“这就是弱者的悲哀，本座虽是鬼王，但轮回路中的强者，本座惹不起。”
<br/>　　天鬼皇无奈的叹息一声，心中也有点悔意，但是已经到了这一步，开弓没有回头箭。
<br/>　　慕清霄立于空中，沉默不语。
<br/>　　在他看来，根本没有惧怕的必要，整个鬼界中，除了鬼仙能让他忌惮外，其他鬼修都无所畏惧，哪怕鬼圣。
<br/>　　鬼圣也不过是相当于大乘期修士而已。
<br/>　　他现在是大乘期二重，施展黑凤化身，对鬼物有本能的压制效果，哪怕不敌鬼仙，但也有自保的能力。
<br/>　　“在过百里就是阴寒域了，那里是双煞鬼王的地盘，我们将气息收敛好，找好时机就动手。”
<br/>　　慕清霄道：“将双煞鬼王斩杀后，不管任何理由，即可带我去火鬼界。”
<br/>　　“阁下放心，本座一言九鼎。”
<br/>　　说罢，天鬼皇化作一团黑雾向西北方向掠去，就算慕清霄不提醒，他也会这样做。
<br/>　　慕清霄在鬼界中待的时间越久，那么他们天鬼一族暴露的风险就越大。
<br/>　　……
<br/>　　……
