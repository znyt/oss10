
.LA，最快更新天命赊刀人最新章节！<br/>　　<br/>　　宝瑞福的一二楼上客率差不多有三成左右，在这个基本其他茶馆都打烊了的时间里，还来宝瑞福的人明显就不是奔着喝茶来的了。<br/>　　<br/>　　王惊蛰一走一路过间，就看见几张桌子旁坐的人，胳膊都伸到了桌子底下，面对面的两人表情都没什么变化，最多只有点头或者摇头，这叫算账，是一种很有年代感的叫价方式。<br/>　　<br/>　　买东西的人手在桌子底下叫价，卖东西的人如果不满意或者觉得价格差多了，同样也在下面还价，如果成交了后两人就欣然一笑，你收钱他收东西，这种算账的方式已经很古老了，以前在做古董买卖出价或者要价的时候怕旁边人看见，所以手就都伸到了桌子底下比划，这种方式现在已经不多了，也就潘家园或者其他一些专门做老货生意的地方才保持着。<br/>　　<br/>　　“我这有上好的明前茶，楼上房间安静，咱们先喝点茶叶？”贺三刀一边走一边招呼伙计吩咐要上茶。<br/>　　<br/>　　王惊蛰则摇头说道：“茶就先不喝了，掌柜的我赶时间，咱们还是直奔主题吧”<br/>　　<br/>　　“呵呵，也行”贺三刀也没坚持，领着王惊蛰就上了楼。<br/>　　<br/>　　宝瑞福的三楼是不开放的，这一层是贺三刀专门用来放东西的地方，整整一层摆放的都是他和以前家里人搜罗来的老物件，东西虽然不是特别多，但样样都是精品，随便拿出去一件都能让搞收藏的人眼红不已。<br/>　　<br/>　　贺三刀来到防盗门前也不避讳，当着王惊蛰的面也没太掩饰就输入了防盗密码，门“咔嚓”一下开了后，王惊蛰就说道：“你也不怕我是江湖骗子？从见到我到现在，你可还没确定过我的身份呢，万一我是个假拐子呢？”<br/>　　<br/>　　假拐子是行话，那意思是我要是冒充的江湖骗子呢？<br/>　　<br/>　　贺三刀回头指着王惊蛰长袍下面隆庆祥的标志说道：“巧了不是，你这件衣服我还真有点印象，两年前的时候我去找隆庆祥的大掌柜要做身衣服，闲着没事跟他聊起过，就问最近一段时间往外出过几身褂子啊，大掌柜的说除了上供的几件外，流出去的真不多，一只手都数得过来，然后他就跟我聊了一下让我有机会看见了的话，就认识认识，其中你这件藏青色的长袍大掌柜的特意提了一嘴，说是差不多一年前他们隆庆祥的关系户来求做的，我就问他对方是什么人呢，他说是赊刀人”<br/>　　<br/>　　王惊蛰的这身长袍，是他爹临近云游之前送给他的，也就是三年多前的时候吧，王爹说别看他们赊刀一脉表面日子过的挺穷苦，但也有穷讲究的时候，内联升的千层底和隆庆祥的长袍肯定不能少，这是给碰到识货的人看的，恰巧了今天碰见的贺三刀就是这一类识货的。<br/>　　<br/>　　防盗门开了之后，贺三刀伸手在墙壁上按了下开关棚顶上的灯就亮了起来，屋子内摆放着几排架子和玻璃罩子，大概放了能有二三十件东西，一看就是挺有历史感的玩意儿。<br/>　　<br/>　　贺三刀颇有些得意的说道：“你看那一件是明成化的鸡缸······”<br/>　　<br/>　　王惊蛰笑着摆了下手，说道：“贺老板你这些藏品说实话，我家要是想收也有，但我们真不太感兴趣，老实和您讲吧，我是想要一件东西当聘礼，这才和您做一次生意的，不然······呵呵，你就是请，我也未必愿意和你换”<br/>　　<br/>　　贺三刀愣了愣，也有点皱眉，王惊蛰这话说的挺不客气的了，换个人的话他可能直接就拂袖赶人走了，不过稍微一想王惊蛰背后的身份他也就释然了。<br/>　　<br/>　　最近几十年，赊刀人都不露面了想碰到他们太难了，一般人都不知道赊刀人的底细，也就他这种在民间颇有些见识的人才知晓。<br/>　　<br/>　　贺三刀就知道关于赊刀人的传说有一种不算是夸大其词的说法，那就是赊刀一句话都等于你的万贯家财了，也许他们指点的一句就能让你免得家破人亡，或者捡回一条命，也可能会让你飞黄腾达，曾经有多少人想求赊刀人铁口一算来定乾坤，但可惜都不得其门而入。<br/>　　<br/>　　“好，那你随便看看，怎么换咱们稍后在说······”<br/>　　<br/>　　王惊蛰嗯了一声，背着手就溜达在了贺三刀的藏品室里，这老头确实挺有底子的，就那件成化的鸡缸杯恐怕都能拿到三环内置换栋房子了。<br/>　　<br/>　　不过，王惊蛰对于这种玩意儿都不太感兴趣，其实他也说不清自己想要什么，就觉得这聘礼拿得出手不说，最好还是得能把人给镇住。<br/>　　<br/>　　王惊蛰走了两圈后，眼神始终没有聚焦在哪件东西上太长的时间，看的等在旁边的贺三刀有点抓心挠肝的直担忧，他这些藏品是挺值钱的，但对他来说就是个喜好，真不如能求来赊刀人的一句定乾坤的话来得重要了。<br/>　　<br/>　　“小朋友，没有中意的东西？”贺三刀忍不住的问道。<br/>　　<br/>　　王惊蛰摇了摇头，叹了口气说道：“和我想象中的差了一点，嗯，查了点感觉吧”<br/>　　<br/>　　贺三刀皱眉说道：“你总得说说你想要的是什么啊，形容一下？”<br/>　　<br/>　　“额·····”王惊蛰憋了半天，才说道：“送的人家可能也不缺钱，所以价值什么的无所谓了，关键是能把人给镇住，就像你去送世界首富一座金矿银矿什么的人家会有多大的兴趣？”<br/>　　<br/>　　“那你这可就难了！”贺三刀叹了口气说道。<br/>　　<br/>　　“呵呵，实在不行我就随便······”<br/>　　<br/>　　王惊蛰话刚说一半，贺三刀就拍了拍脑袋，然后说了声等会就出去了，过了片刻他又回到楼上手里多了一串钥匙，带着王惊蛰来到一处角落里后，指着地上放着的一个箱子说道：“这里面的东西你要是还挑不上，那我就只能说咱俩的生意，还是缘分没到”<br/>　　<br/>　　“那行，你打开我看看吧”角落里放的是个鲁班箱，这东西可是个稀缺的物品，拿到现在来说等同于一个保险柜，这是传自大工匠鲁班的手艺，现代人已经没办法仿制出来了。
