　　萧若寒你输了！
<br/>　　当萧逸说出这句话时，没有任何人反驳，更没有任何人表露出不满。
<br/>　　因为，现在的萧逸已经表现出足够的实力去说这句话。
<br/>　　虽然萧逸是废武魂，萧若寒是强大的黄阶武魂。
<br/>　　武魂是天赋最主要的指标，天才还是废物，全看武魂强弱。
<br/>　　但，这个世界最看重的还是实力，实力才是一切权利的来源。
<br/>　　毕竟，天赋再好，没有成长起来，卵用都没有。而实力，却是实打实存在。
<br/>　　长老席上，五长老的脸色阴沉如水，心中暗恨，“该死，萧逸竟然真的打败了萧子木。这样一来，我的计划完全被打乱了。”
<br/>　　萧若寒也是难看，求助的目光看向五长老。
<br/>　　五长老点了点头，示意他放心。
<br/>　　随即，五长老站了起来，高声道，“萧逸，家族比武已经结束，你可以下去了。”
<br/>　　说罢，他看向身旁的其他长老说道，“众位长老，该是开启紫云洞的时候了，我们走吧。”
<br/>　　他打算快刀斩乱麻，不给萧逸说话的机会。
<br/>　　萧逸自有应对之法，同样高声道，“怎么，五长老你想耍赖不成？”
<br/>　　他这样高声地说五长老耍赖，五长老就不能再含糊其辞地推脱过去了，必须认真地与自己对话。
<br/>　　五长老皱眉道，“萧逸，你这话什么意思？别以为你是少家主，就能随意污蔑长老，小心我定你一个不尊长辈之罪。”
<br/>　　这个老狐狸，当即反扣萧逸一个不尊长辈的罪名。
<br/>　　萧逸怡然不惧，道，“五长老，半月前我与萧若寒的打赌，人人皆知。待会儿开启紫云洞，还请您看好你儿子，可别让他浑水摸鱼溜了进去。”
<br/>　　“哈哈哈。”族人们听着萧逸打趣的话，顿时哄堂大笑。
<br/>　　“你...”五长老脸色一黑，喝道，“萧逸，你们这些小辈间的打赌不过是小打小闹，当不得真。”
<br/>　　五长老加重了语气，试图施压，吓退萧逸。
<br/>　　萧逸自然不惧，反问道，“当不得真？这个赌约，可是半月前在议事厅，当着长老和执事们的面定下来的。你若觉得这都当不得真，岂不是说我们萧家的议事厅只是儿戏之地？”
<br/>　　“哼。”五长老冷哼了一声，心中似乎想到了应对之策，高声道，“萧逸，你觉得能当真。那好，现在当着所有族人的面，我倒要问问，谁会当这种胡闹一般的打赌是真？”
<br/>　　五长老说着，霸道的目光扫视着所有族人。
<br/>　　同一时间，七、八、九长老全部站到他身边。
<br/>　　四大长老一同施压，族人们全部噤声，沉默着。
<br/>　　五长老露出了得意的笑容，他早就料到，没有人会因为这种事不关己的打赌，而公然和自己以及另外三位长老叫板。
<br/>　　忽然，一道威严的声音传来，“我当真！”
<br/>　　声音响彻全场，传入所有人耳朵。
<br/>　　“嗯？”五长老顿时脸色一冷，打算看看是什么人敢在自己面前放肆。
<br/>　　“你算个什么东西....”五长老顿时出言呵斥，以他长老的身份，足以呵斥萧家所有人。
<br/>　　然而，当他看到来人后，立马收住了话语。
<br/>　　来人是个老者，须发皆白，但精神抖擞，眉宇间闪过道道精光，显然非寻常之辈。
<br/>　　“参见二长老！”所有族人忽然全都恭敬地行了个礼。
<br/>　　“二长老。”各个长老和执事们也微微弯腰。
<br/>　　不错，来人乃是一直没有出现过的二长老。
<br/>　　萧家九大长老中，一般掌控家族权力，商议家族要事的，都是后面七位长老。
<br/>　　至于大长老和二长老，他们都是家族里的老人，辈分最高，比之其他长老要高上一辈，年事已高，所以极少出现在族人们面前，也不管家族里的闲事。
<br/>　　上次的长老会议，还有今天的家族大比，这两人都没有出现。
<br/>　　大长老是家族里唯一的三品炼药师，本身就非常忙，自然没有闲工夫管其他事。
<br/>　　而这二长老，是个武痴，醉心修炼，极少现身。
<br/>　　同时，他还是如今家族里武力最高之人，先天境九重武者，距离洞玄境武者，只差一步。
<br/>　　所以，哪怕二长老平时不掌权，但他的辈分还有实力摆在这，威望比之其他长老高上一大截，哪怕是七大长老共同决定的事，他都能一力推翻。
<br/>　　二长老先是面露赞赏之色地看了萧逸一眼，随后再次恢复了威严的目光看向萧若寒和五长老。
<br/>　　“我说此事当真，你可有意见？”二长老威严地问道。
<br/>　　五长老连忙恭敬地说道，“不敢，既然是二长老您决定的事，我自然没有意见。”
<br/>　　二长老曾经还是各大长老以及萧逸父亲的武道启蒙老师，他反问一句，五长老屁都不敢放一个。
<br/>　　“哼，没意见就好。”二长老哼了一声，道，“打赌之事，我也略有耳闻。虽然这是小辈间的打赌，但他们两人都是武者，作为一个武者，如果连承担失败后的后果都不敢，那还有什么资格当武者。”
<br/>　　“一个武者，若没有一颗坚定不屈的心，这点儿小事都不敢面对，日后如何在艰难的武道之路上劈波斩机，又如何能有所成就。”
<br/>　　“就算不谈武者，单单言而无信这一条，就不配当我萧家子孙。”
<br/>　　二长老愤愤地教育了一番。
<br/>　　“是是是。”五长老恭敬地点头，而后为难地说道，“可是二长老，您也知道，若寒他是家族里天赋最高的子弟，凡境七重的修为冠绝全族，而且又是黄阶武魂，若不让他进紫云洞修炼，会误了他一生的。”
<br/>　　“这...”二长老陷入了沉默，他自然不可能误了家族里的子弟。
<br/>　　这时，萧逸高声道，“五长老，要让萧若寒进紫云洞修炼也不是不可以。半月前在议事厅我就说过了，和萧若寒打赌，我自身存在着风险，所以萧若寒也必须承担相应的代价。”
<br/>　　“那么，只要你赋予等价的代价，倒是可以给萧若寒换回这个名额。”萧逸笑着说道。
<br/>　　“等价的代价？”二长老看向萧逸，问道，“你这是什么意思？”
<br/>　　萧逸回答道，“很简单，一万两，用一万两买回这个名额。”
<br/>　　五长老当即大怒，道，“萧逸，你不要太过分了。老夫在家族一个月的俸银不过数百两，如何有一万两给你。”
<br/>　　萧逸淡淡道，“那是你的事，钱不够，那就拿东西抵。我想，进紫云洞这么珍贵的机会，是值一万两的吧。”
<br/>　　“值，当然值。”二长老点点头，道，“此事就这么定了。”
<br/>　　五长老即刻哭丧着脸，但二长老决定的事，他又确实无法反对。
<br/>　　“萧逸，五千两，外加三颗先天丹，十颗后天丹，可行？”五长老只能低声下气地问道。
<br/>　　进紫云洞修炼，对于萧若寒来说乃是重中之重，五长老根本没有选择的余地。
<br/>　　萧逸皱了皱眉头，道，“五千两，外加五颗先天丹，二十颗后天丹。”
<br/>　　“五颗先天丹，二十颗后天丹？”五长老瞪大了眼睛，而后怒吼道，“萧逸，你不如去抢好了。”
<br/>　　二长老也咳嗽了两声，道，“萧逸，不要得理不饶人。这样吧，五千两，外加三颗先天丹，十五颗后天丹。”
<br/>　　萧逸一笑，道，“既然是二长老决定的事，我没有意见。”
<br/>　　“谢二长老。”五长老对二长老满脸感激之色，但看向萧逸的目光，仍旧充满了恶毒。
<br/>　　“好了。”二长老威严地说道，“既然事情已经解决，那么便去开启紫云洞吧，此乃家族中的大事，马虎不得。”
<br/>　　“是。”众长老齐声应答。
