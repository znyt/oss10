<br/>				一个小时之后，一行人已经出了云州市。<br/>　　<br/>　　又过去两个小时，一连跨越两个州市之后，众人便已经进入天海市的地界。<br/>　　<br/>　　只不过，车子并没有进入天海市市区，而是绕过天海市，直奔北面而去。<br/>　　<br/>　　经过曹坤的一番解释，叶尘才知道，因为每次武林大会，高手云集，死伤自然在所难免，如果在市内举行的话，影响不好，而且也放不开手脚。<br/>　　<br/>　　因而，历届武林大会的地点，都选在天海市北面，靠近长江的一个小镇子上，甚至偶尔有时，以免动静太大，引起社会关注，众大佬还会登上游轮，沿江而下，直接开到公海之上。<br/>　　<br/>　　又过去半个多小时，终于抵达此行的目的地，这是一个废弃港口。<br/>　　<br/>　　在废弃港口附近，还有一些村落、小镇，因为位置比较偏僻，看上去相当落后。<br/>　　<br/>　　曹坤指着那些破旧村落，道：<br/>　　<br/>　　“叶先生，你别看这里破，但是这里可是有名的古武之乡，几乎人人都会几手庄稼把式，而且还出过不少武道高手呢！”<br/>　　<br/>　　说到这里，曹坤微微一顿，接着又道：<br/>　　<br/>　　“前面大江沿岸，有一个小镇，名为太平镇，十大州市的大佬们，一般都会先在那里汇合，至于要不要出海，得由武家说的算！”<br/>　　<br/>　　武家，天南省公认的第一大家族，实力还在天海莫家之上，而且还是传承了数百年的武道世家！<br/>　　<br/>　　而整个天南省的地下世界，名义上的话事人，正是武家的那位老祖宗，武长空！<br/>　　<br/>　　据说这武长空早在三十多年前，就已经是化劲宗师境界，论资历，比莫家老祖莫寻欢还要老，如今的修为，肯定更加深不可测！<br/>　　<br/>　　因而这数十年以来，几乎可以毫不夸张的说，整个天南省，一直都是武家说的算。<br/>　　<br/>　　十大州市的巨头，哪怕斗的再凶，却也没有谁胆敢挑衅武家的威严。<br/>　　<br/>　　听到这话，叶尘不由得眉头微微一挑，<br/>　　<br/>　　“曹四爷，我记得你之前说过，每一省的话事人，都是以武力取得，如果我我帮你打败武长空，让你来做天南省的话事人，你意下如何？”<br/>　　<br/>　　听到这话，曹坤顿时脸色大变，连忙摆手道：<br/>　　<br/>　　“不行，不行！此事万万不可！先不说那武长空的修为深不可测，我不能让叶先生冒险，即便叶先生真的可以击败武长空，但是那武家的势力，在天南省根深蒂固，我也没有资格当话事人呀！”<br/>　　<br/>　　叶尘原本就是随口一说，见曹坤如此惊慌失措，于是点了点头，不再多说什么。<br/>　　<br/>　　……<br/>　　<br/>　　很快，来到江边小镇，车子在一处四层小楼前停下，应该是一处酒店，看上去规模倒是不小，只不过远远无法和城市中的那些大酒店相提并论。<br/>　　<br/>　　众人才刚一下车，一名身材肥胖的中年男子，一路小跑着迎了上来，冲着曹坤一脸谄笑道：<br/>　　<br/>　　“哎呀，曹四爷，好久不见！”<br/>　　<br/>　　原来这中年男子，是这酒店的老板，名叫宋宝德，因为每次武林大会，都在这里举行，而且他又擅长交际，所以和十大州市的大佬们都十分熟悉。<br/>　　<br/>　　两人随意寒暄了两句，宋宝德笑道：<br/>　　<br/>　　“房间已经为诸位准备好了，曹四爷，里面请吧！”<br/>　　<br/>　　曹坤点了点头，不过却并没有迈步，而是冲着一旁的叶尘恭恭敬敬道：<br/>　　<br/>　　“叶先生，您先请！”<br/>　　<br/>　　宋宝德看到这一幕，顿时暗暗心惊，等到叶尘从他身旁走过之后，旋即想到了什么，忍不住低声喃喃道：<br/>　　<br/>　　“莫非这位，就是最近名震天南省的云州叶先生么？没想到竟如此年轻！”<br/>　　<br/>　　……<br/>　　<br/>　　酒店外面看着破烂不堪，没想到里面居然装修的十分豪华，丝毫不亚于那些大酒店的总统套房。<br/>　　<br/>　　一行人安顿下来之后，曹坤和曹文叔侄，再次来到叶尘的房间，<br/>　　<br/>　　“叶先生，我要先去和其他大佬碰个头，顺便打探一下这次大会的情况，要不让阿文陪您四处走走吧？”<br/>　　<br/>　　叶尘摆了摆手，<br/>　　<br/>　　“不必！你们忙你们的事情去吧，我若想出去的话，就自己随便逛逛。”<br/>　　<br/>　　……<br/>　　<br/>　　曹坤叔侄离开之后，叶尘在酒店中休息了一会，也顿时觉得有些无趣，于是便离开房间，走出了酒店。<br/>　　<br/>　　太平镇虽然不大，但是街道上却异常热闹，叶尘寻思着，多半是因为这场武林大会的缘故。<br/>　　<br/>　　毕竟十州大佬齐聚，而且还有各路武道高手比试，对于那些武术爱好者来说，绝对是一场不容错过的盛宴。<br/>　　<br/>　　甚至，在小镇外面的空旷之地，已经有一些好事之人，搭起了擂台，只不过周围用红布遮挡，只能听到里面的打斗声，以及观众们的叫好声，想要观看的话，需要购买门票才行。<br/>　　<br/>　　叶尘左右闲着也无事，于是便掏钱买了一张门票，准备进去看看。<br/>　　<br/>　　然而就在叶尘，一只脚已经迈入场内的时候，不经意间一回头，顿时看到了一张无比熟悉、但又有些遥远的美丽面庞！<br/>　　<br/>　　“蔓姐……”<br/>　　<br/>　　原来这人，竟然是苏岚的女儿，叶尘的姐姐，苏蔓！<br/>　　<br/>　　饶是以叶尘如今的心境，时隔八百年，再次看到这道无比亲切的身影，内心也不由得激动起来，险些有些难以自持。<br/>　　<br/>　　记得前世，他先是被叶无伤和莫长云陷害，在好兄弟吴磊的帮助下，逃到了国外避难，后来叶无伤为了将叶尘逼出，竟然抓了苏岚，想要拿她的性命来威胁叶尘，苏岚为了保护叶尘，决然选择了自尽……<br/>　　<br/>　　后来，似乎是叶尘的父亲叶天歌知晓了此事，叶无伤便没有再继续追杀叶尘，苏蔓也幸免于难。<br/>　　<br/>　　只不过，苏蔓先是面临母亲惨死，弟弟逃亡国外，男朋友也畏于叶家的权势，也将她狠狠抛弃，整个人的精神一度面临崩溃！<br/>　　<br/>　　叶尘至今仍然清楚的记得，自己成为血影天罚的杀手，悄悄返回国内见到苏蔓时，她那疯疯癫癫的模样……<br/>　　<br/>　　因而，从某种角度来讲，叶尘前世亏欠最多的人，不是苏岚，不是唐清雅，也不是吴磊，而是自己的这位的姐姐！<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
