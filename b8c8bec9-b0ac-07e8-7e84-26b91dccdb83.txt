　　今日楚风眠本来就花费了两百亿九鼎丹，购买了大批圣者精血。
<br/>　　如今击败王傲，又是足足燃烧了两百亿九鼎丹，然后更是再一次的拿出了两百亿九鼎丹来。
<br/>　　楚风眠如今单单展现出来的财富，就是足足有着六百亿九鼎丹，而且看起来，楚风眠所拥有的财富，还远远不止这些。
<br/>　　虚堇试探性的开口道。
<br/>　　“楚公子如今是需要大批的圣者精血？”
<br/>　　楚风眠点了点头道。
<br/>　　“不错，我如今为了提升实力，早日踏入神力境成就圣者，好加入到北邙学院中去，正是需要大量圣者精血来炼化。”
<br/>　　北邙学院中，想要成为内门弟子，至少需要神力境的境界才行。
<br/>　　不然的话，只能是成为外门弟子。
<br/>　　外门弟子，实际上都必须要承受沉重任务才行，不能全身心修炼。
<br/>　　只有内门弟子，才算得上是真正北邙学院的弟子，可以安心修炼。
<br/>　　楚风眠的境界，便是御风境巅峰，这是楚风眠真正的境界，无论如何探察，都是这个结果。
<br/>　　如今楚风眠的话，却是天衣无缝，令虚堇相信。
<br/>　　“我们虚家中，其实还有着一批圣者精血，不知道楚公子敢不敢兴趣？不过这批圣者精血的数量巨大……”
<br/>　　虚堇沉吟了一下，开口道。
<br/>　　“楚公子若是需要的话，我可以代表虚家卖给你一部分。”
<br/>　　“一部分？虚堇姑娘放心，无论你们虚家有着多少的圣者精血，我都可以买下来，五百亿？六百亿？”
<br/>　　楚风眠大笑连连道。
<br/>　　楚风眠的举动看起来倒是真的跟那种钱财无数的暴发户一样。
<br/>　　楚风眠挥手之间，又是悄然的拿出了无数九鼎丹来，这一次足足是有着接近千亿之多。
<br/>　　看到楚风眠展现出来的千亿九鼎丹，虚堇眼神中都要贪婪的发狂，恨不得现在就要将楚风眠的财富全部抢过来一样。
<br/>　　虚堇都没有见过居然有人可以一口气拿出来千亿九鼎丹，就算是像是虚家这样庞大的家族，想要一口气拿出千亿九鼎丹来，也需要伤筋动骨。
<br/>　　虚堇很快便是冷静了下来，这个时候，她还不想要表现出任何的异常出来。
<br/>　　反正其中无数圣者精血中，都蕴含着魔血，楚风眠炼化的越多，也就越是受到他们的控制。
<br/>　　倒时候无形之间，他们就可以控制了楚风眠，楚风眠身上的一切，也都全归他们了。
<br/>　　想到这里，虚堇的眼神中不由的露出几分冷笑。
<br/>　　财富众多又如何，楚风眠身上的财富，迟早是属于他们虚家的。
<br/>　　虚堇笑了起来道。
<br/>　　“我们虚家，一共是有着价值一千五百亿的圣者精血，不知道楚公子需要多少？”
<br/>　　“才不过一千五百亿罢了，虚堇姑娘放心，这点财富，楚某还是拿的出来的。”
<br/>　　楚风眠不以为然的开口道，似乎这一千五百亿的庞大财富，在楚风眠看来，也不算什么一样。
<br/>　　他大手一挥，便是一道深邃到极点的灵药长河出现，其中密密麻麻的九鼎丹，足足有着一千五百亿整。
<br/>　　“这批圣者精血，楚某要了。”
<br/>　　说着，楚风眠便是手一挥，这一千五百亿九鼎丹，便是向着虚堇飞了过去。
<br/>　　楚风眠话音一落，这虚堇的脸色都是微微僵住了，似乎也想不到，楚风眠竟然是如此痛快。
<br/>　　拿出一千五百亿九鼎丹，简直就像是喝水一样简单。
<br/>　　这可是一千五百亿！这等财富！让一些生死境强者，都拿不出来吧。
<br/>　　他们虚家想要调动一千五百亿九鼎丹，都需要伤筋动骨，而且很有可能根本拿不出如此之多的丹药来，毕竟这可是一笔巨大财富了。
<br/>　　楚风眠现在展现出来的财富，简直是足以跟一个庞大无比的家族媲美了。
<br/>　　虚堇本来猜测楚风眠只是有些小奇遇，但是现在看来，楚风眠得到的奇遇简直要惊人，不然怎么可能会有如此财富。
<br/>　　虚堇被楚风眠镇住，许久之后才反应过来，急忙的接住了那一千五百亿九鼎丹，小心翼翼的收入到空戒之中去。
<br/>　　毕竟就算是她，也是第一次见到如此之多的财富，极为紧张。
<br/>　　“这批圣者精血，我会命人，送到楚公子的住处去。”
<br/>　　虚堇开口道。
<br/>　　“不急，反正虚堇小姐的诚意，楚某还是信得过的。”
<br/>　　楚风眠笑了笑，突然话风一转开口道。
<br/>　　“对了虚堇小姐，不知道关于我们三人北邙学院的考核资格名额的事？如何了？”
<br/>　　听到楚风眠的话，虚堇眼神中冷笑连连，这楚风眠的命都要丢了，还在想着北邙学院考核的事。
<br/>　　不过她脸上的表情却是极为平静，面露出几分难色说道。
<br/>　　“这北邙学院的考核资格名额，其实已经是定下来了，许多虚家的弟子，不愿意将名额交出来，我也是极为难办。”
<br/>　　“若是给他们一些补偿的话，倒是可能会轻松一点。”
<br/>　　虚堇的表情，好像是极为为难，一副无力的样子。
<br/>　　“补偿？好说，这北邙学院考试资格的名额价值珍贵，这一点我自然清楚，我愿意出三百亿九鼎丹来，来换取其中的三个名额来。”
<br/>　　楚风眠笑着说道，语气中极为轻松，好像是关于财富的事，在楚风眠看来，都不算事一样。
<br/>　　他大手一挥，又是足足三百亿九鼎丹出现，向着虚堇飞了过去。
<br/>　　这才是真正视金钱如粪土。
<br/>　　旁边的许多众人，看着楚风眠的眼神，简直就像是看着一个肥羊，这样巨大的财富，人人都想要得到。
<br/>　　不过现在这只肥羊，却是被虚家宰了，他们也只能看看，不由的叹息一声自己怎么就遇不到楚风眠这样的肥羊。
<br/>　　“好，有了这些九鼎丹，应该会有不少的虚家弟子愿意放弃这一次的名额来。”
<br/>　　虚堇接过这三百亿九鼎丹，脸色瞬间变得轻松了许多，惊喜道。
<br/>　　“还请楚公子多等几日，几日之后，这名额一定会送过去的。”
<br/>　　“好。”
<br/>　　楚风眠点了点头。
<br/>　　接下里的晚宴，倒是极为平静，散场后，楚风眠一行人便是又回到了宫殿之中去。
