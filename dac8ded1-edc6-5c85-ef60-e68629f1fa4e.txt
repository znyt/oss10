　　所长想起之前说的这句话，尴尬地想要撞墙。
<br/>　　美华跟在叶少阳身后，扑哧一笑。
<br/>　　“叶先生，这边请！”县长十分热情，邀请叶少阳三人出门。
<br/>　　所长跟在后面一再赔不是。县长看也没看他一眼。
<br/>　　警察所的大院里，挺着一辆绿色的小汽车，这是叶少阳在这个时代看见的第一辆汽车，跟民国的影视剧里演的差不多，圆嘟嘟的跟蛤蟆似的小汽车。
<br/>　　司机下来给他们打开车门。
<br/>　　“叶先生，您先请！还有这两位先生，不必客气。”县长十分热情。
<br/>　　三人钻进车里之后，道渊真人和毛小方一起朝叶少阳看过来，眼神在问他是什么回事。
<br/>　　“别问我，我什么都不知道，我比你们还懵。”
<br/>　　县长坐上来之后，冲三人笑了笑，招呼司机开车。
<br/>　　警察所长看着汽车扬尘而去，都消失许久了，还在门口傻傻地站着，对身边一个跟他同样发呆的下属说道：“你说，这个叶少阳到底是什么人，值得县长这么对待他？”
<br/>　　“而且县长看上也是帮人跑腿的。”
<br/>　　“帮人跑腿……”那得是什么样的大人物啊。
<br/>　　所长吸气都哆嗦起来，喃喃道：“我说，我怎么得罪了这么一个贵人？明明是拍马屁的好机会啊，生生让我给搞砸了……”
<br/>　　“是啊，别的其实还好，就是您那句……说人家长的看着就不像好人，还说了两遍，生怕人记不住。”
<br/>　　看走眼了……所长现在回想叶少阳的长相，哪里是什么贼人的样子，分明就是眉清目秀，一看就是深藏不露的高人……可惜人家走了，想说也没机会了。
<br/>　　县长的汽车，载着叶少阳一行人穿过县城，来到最宽阔的一条街道上，最后停在了一座深宅大院的门前。门口有两个士兵在站岗，叶少阳看这就能猜到，这栋宅子绝对不是寻常的地方，但怎么看也不像是衙门，倒像是民宅的模样。
<br/>　　叶少阳正在纳闷，县长已经在请他们下车了。
<br/>　　三人刚下车，从宅子里走出来一个胖胖的五十来岁的男子，穿着清式的长袍马褂，看上去就是个有地位的人。
<br/>　　“徐管家。”县长拱手。
<br/>　　“劳驾，劳驾，还烦劳县长大人亲自走一趟，真是劳驾了。”徐管家也表现得非常亲热。
<br/>　　县长也跟着客气，引他见叶少阳。
<br/>　　徐管家冲叶少阳点了点头，像是很熟悉似的，也没有多说，对县长道：“这件事，我代大小姐夫妇谢过县长大人了。”
<br/>　　“这可使不得，既然这位叶先生是大小姐的贵客，便是属下的分内之事，理当如此！徐管家，今日夜深，徐某人不方便拜会大小姐，这便告辞，烦请徐管家跟大小姐通报一声，徐某人明日再来拜会！”
<br/>　　说完又上来跟叶少阳握手，说了几句客气话，这个过程叶少阳是一脸懵逼的……
<br/>　　“你牛逼啊，少阳子，什么时候有这么尊贵的一个朋友？”毛小方低声说道。
<br/>　　叶少阳无言以对，现在的他，内心里一片迷茫。大小姐？自己什么时候认识过一个民国的大小姐？
<br/>　　“叶先生，适才多有怠慢，还请先生原谅则个。”徐管家这才上前见礼，“大小姐已经在客厅等着了，还请叶先生和两位朋友随我进去。”
<br/>　　进门之后，还要穿过一道长长的过道，里面的装修古色古香，但也有一些新式的家具，这家人……绝对是个大户人家。
<br/>　　这个念头刚一浮现，叶少阳就觉得自己是想多了，就凭之前那位县长鞍前马后的表现，也可以断定，这家的主人一定有着不凡的地位。
<br/>　　叶少阳三人被带进一间宽敞的客厅。客厅的摆设是中西结合，铺着地毯，有沙发和玻璃茶几，同时也有一个中式的屏风和一些小物件，整体搭配得十分协调，叶少阳感觉，就这样的装修风格，就算放在自己那个时代，也绝对是时髦的复古风。
<br/>　　“我说，这位大叔，你家大小姐……会不会认错人了？我完全不认识她啊。”叶少阳心里有点发虚，拉住徐管家问道。
<br/>　　徐管家一怔，道：“你是不是茅山天师叶少阳？”
<br/>　　叶少阳比他更吃惊，能说出“茅山天师”四个字，那肯定就没错了，傻傻地点了点头。
<br/>　　徐管家笑起来，“那就是没错了，三位略等啊，我去知会大小姐一声，顺便让人看茶。”
<br/>　　管家走后，毛小方和道渊真人一起追问叶少阳怎么回事，之前还怀疑是误会，听到徐管家说起“茅山天师”四个字，都知道不可能错了。
<br/>　　“我也不知道……反正快来了，一会看看吧。”
<br/>　　美华很喜欢这个地方，不顾叶少阳的提醒，到处转悠去了，包子也跟着跑了，反正也没人能看得见他们。
<br/>　　“这电话，好精美啊。”毛小方伸手注意了矮柜上的电话机，伸手轻轻摸起来。
<br/>　　这时候脚步声响起，三人立刻转头看去，一对青年男女从内室走出来。
<br/>　　男的三十来岁，穿西装，戴着眼镜，看着还是挺有文化的，女的……当叶少阳把目光移到女子身上的时候，整个人立刻就呆住了，从沙发上缓缓站起来，面目呆滞地望着这个妹子。
<br/>　　与身边的男子不同，这姑娘一身古装，亭亭玉立，她的脸……叶少阳简直太熟悉不过了。
<br/>　　姑娘冲身边的男子笑了笑，转向叶少阳，歪着头，笑了笑，道：“怎么呢？”
<br/>　　“你……”叶少阳深深吸了一口气，道出她的名字：“凤兮？”
<br/>　　面前这个一袭古装的美女，不是别人，正是失散已久的凤兮！
<br/>　　是真的凤兮，还是凤兮的前世或者别的什么？
<br/>　　叶少阳一瞬间有点犹豫起来。
<br/>　　如果是一切都没发生之前，叶少阳看到凤兮，一定会立刻上前相认，但是……在见过了长得像芮冷玉的梧桐、长得像苏沫的碧清之后，此时见到凤兮，一时之间，叶少阳真的有点不敢随便相认了。
<br/>　　“过了一百年，你就不认识我了吗？”凤兮望着叶少阳，轻轻笑着。
