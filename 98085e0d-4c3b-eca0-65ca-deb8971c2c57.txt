　　这才是罗昊应该有的教育自己儿子的方式。
<br/>　　什么“退一步海阔天空，忍一时风平浪静”，不存在的。
<br/>　　蓝若依听完罗战的话，眉头顿时就皱了起来，眼神不善的看向诸磊，云悠可不像罗战那样，是被当成罗昊接班人，未来“龙刺”来培养的，云悠擅长的不是近身搏击和格斗，而是远超常人的空间思维能力。
<br/>　　而诸磊这个经过专业格斗的学员，居然向云悠挑战，实在太过分了！
<br/>　　沈琛看了眼躲在诸钟伟身边的诸磊，眼神中透露出一丝不满，微笑着对坐在自己身边的罗战说道：“所以，你就把他的鼻子打断了。”
<br/>　　“我也不是这么不讲道理。”
<br/>　　罗战打了一个嗝，说道：“小悠儿虽然没我这么厉害，但不管怎么说，我们两个都是一个爸妈生的，优秀的基因在那儿摆着，对付他这种人渣也没什么大问题……”
<br/>　　“你说谁是人渣！？”诸磊这个时候探出头来，对着罗战质问道。
<br/>　　“就说的你！”罗战从沙发上滑下来，瞪着诸磊说道：“你自己说，你跟小悠儿比试的时候，用了什么见不得人的招数？”
<br/>　　“小悠儿，他对你做什么了？”
<br/>　　蓝若依看着手中拿着可乐，咬着嘴唇，精致的小脸上满是委屈的云悠，意识到情况的不对劲。看到云悠不愿意说，蓝若依看向了罗战，问道：“小战，他对你姐姐做了什么？”
<br/>　　“这人渣被小悠儿抢攻了两招，觉得丢了面子，居然对小悠儿用下三滥的招数。”罗战愤怒的瞪着诸磊，说道：“小悠儿还说，上次她去上厕所的时候，还看到他跟另外几个人在后面偷偷跟着她。”
<br/>　　“诸磊！！”
<br/>　　蓝若依对着诸磊怒喝一声，向沈琛说道：“学校明确规定，男学员不得进入女学员宿舍区，不得在女生厕所前逗留！诸磊这么做，已经严重违反了红旗学校的校规校纪，按照校规，他应该被开除出红旗学校！”
<br/>　　“我没有！”
<br/>　　诸磊极力否认罗战对自己的控诉，他也是红旗学校的老学员了，他很清楚被红旗学校开除意味着什么，这种开除的影响会有多么的严重，这种影响将会是一辈子的，会在他的个人档案留下一个永远也擦不去的污点！
<br/>　　只要有这个污点在，诸磊原本从红旗学校毕业后可以拥有的光明前途，会被彻底摧毁；甚至他的家人也会以此为耻，不再让他回家。
<br/>　　诸钟伟的脸色也变得有些难看，罗战所说的，诸磊尾随云悠的事情他根本就不知情！他虽然是诸磊的叔叔，但也不意味着他会无视红旗学校的校规校纪包庇诸磊。
<br/>　　“你再说一遍没有！？”罗战抬手就把手中的可乐罐子扔向诸磊。
<br/>　　当啷！
<br/>　　可乐罐子落在诸磊的脚边，反弹而起，弹在他身上；诸磊的衣服上也被溅上了好几处可乐的痕迹。
<br/>　　当啷！当啷！
<br/>　　骨碌碌……
<br/>　　可乐罐子在地上又弹了几下后，滚落到一旁，可乐罐里剩下那些满是气泡的可乐，还在“咕噜咕噜”的往外流淌……
<br/>　　看到罗战举起拳头又要朝着自己冲来，诸磊吓得又躲了回去。尽管诸磊年纪比罗战大，可是他真的打不过罗战啊！
<br/>　　“校长，你也看到罗战的态度了，我认为他刚才说得话是对诸磊的诬陷！”诸钟伟对沈琛说道：“这件事情必须要严肃处理，我还是认为罗战和云悠不适合待在红旗学校！”
<br/>　　最后，诸钟伟还是选择相信诸磊。
<br/>　　沈琛的确是红旗学校的校长，也有绝对的权利决定任何一个学院的去留；但是罗战和云悠的去留，却不是沈琛一个人能决定的。罗战和云悠是老校长夏洋带进红旗学校的，所以就算是他们离开，也要询问过老校长的意见。
<br/>　　“我也认为这件事情必须要严肃处理。”
<br/>　　这时，一道声音从校长办公室外出来，“我的儿子和女儿可不能就这么被人给欺负了！”
<br/>　　听到从办公室外传来的声音，诸钟伟心头“咯噔”一跳，紧接着眉头锁紧，看向罗战和云悠的眼神变得有些复杂，这姐弟俩是他的孩子！？
<br/>　　对于罗昊，诸钟伟心中可以说是有阴影的，他比罗昊要高两届，可是在罗昊进入红旗学校的第一天，就被罗昊给揍了！他经过系统的训练，掌握了各种格斗技巧，而且人数也占优势，可罗昊比他敢玩命啊！打起架来，完全就是一副拼命的架势，自己这边气势瞬间就弱了下去，愣是被罗昊和齐凡两个人拿着板砖追着打。
<br/>　　这一架打出了罗昊和齐凡的名气，也让诸钟伟丢尽了面子，后面诸钟伟几次想要找回场子；可那个时候，诸钟伟哪里还能是罗昊和齐凡两个人的对手，这两个“混世大小魔王”在红旗学校里成就了一段“传奇”，你不来惹我那没问题，你敢来惹我，打得你连你爹妈都不认识！一度成了那些低年级学员崇拜的偶像！
<br/>　　沈琛听到罗昊的声音，也是在心中感叹了一声，这“混世大魔王”还是来了！沈琛也知道，罗昊不可能不会出面；但是沈琛没有料到，这次不仅罗昊来了，就连齐凡也来了！
<br/>　　看到走进校长办公室的罗昊、齐凡和云熙，沈琛的眉头是“突突”直跳，来一个“混世大魔王”也就算了，怎么连齐凡这个“混世小魔王”也来了？
<br/>　　诸钟伟看到罗昊和齐凡，脸色也是变得非常难看，当年被罗昊和齐凡拿着板砖追着打的画面又是浮现在了他的眼前。
<br/>　　看到罗昊，再看罗战，诸钟伟瞬间就明白了，果然是父子啊。当年，罗昊以插班生的身份进入红旗学校，吊打他们这几个学长，一架打出了名气；现在他儿子又来了，而且几乎是把他老子做过的事情重演了一遍！
<br/>　　“老爹！”
<br/>　　看到罗昊，罗战立刻就朝着罗昊跑了过去，炫耀着跟罗昊说道：“我没给你丢脸，我有好好保护小悠儿哦！”
