　　第582章
<br/>　　白沙岛上，燕不归跳下悬崖坠海。大海波涛万千，想要肉身横渡，必定九死一生。不过程大雷隐隐有种感觉，燕不归并没有死。
<br/>　　这也在清理之中，古往今来，又有那个侠客或魔头是坠崖死的。悬崖，一般意味着奇遇、宝藏、世外高人，就是不会死。
<br/>　　燕不归如果没死在大海中，必然会返回戎族草原。程大雷虽然不知道燕不归的具体来历，可他这样的高手，在戎族定然也是有数的，只要细细查访总是能找到的。
<br/>　　走火入魔的燕不归要将自己的痛苦给予程大雷，他当然算错了一件事，李莫愁对程大雷来说并不重要，程大雷和她连话都没有说过几句。可世上毕竟没有杀了人还能轻飘飘离开的道理。
<br/>　　所以，程大雷才打算去一趟草原，也查一查戎族千里盗经的目的。
<br/>　　因为这件事和戎族过往的行事风格有些不同。帝国将戎族视为蛮夷，倒也不是完全因为偏见。帝国和戎族之间的战争进行过许多年，连长安城都被打下过一次。但戎族并没有占领帝国城池的意愿，就算签订合约，也是赔款居多，割地很少。
<br/>　　夫子死后，琅琊城一片大乱，程大雷自然成了诸多学子咬牙切齿的对象。这件事很快就会传出去，程大雷将会臭名远扬，估计火烧藏书楼的事情也得按到他身上。
<br/>　　这自然不是什么好事，不过程大雷也是虱子多了不咬，他的名声也不差这一点半点。
<br/>　　琅琊城大乱，倒是给了程大雷三人离开的机会，等他们三个出了琅琊城后，书院就算反应过来也晚了。
<br/>　　三人在城外分离，程大雷骑上黑牛，单人独骑前往草原。刘发财则和华正则一起赶往蛤蟆城，毕竟，那里还有个不生不死的关鱼在等着。
<br/>　　却说书院万卷山，在夫子死后很是乱了一阵。不过，夫子年岁已大，就算不死在程大雷手中，其实也活不了多久。所以，他很早就在安排继承人的问题，早已全权放手书院事务给下面的弟子。
<br/>　　所谓的七贤十二秀，不过是书院年轻弟子中的佼佼者。书院真正的中坚力量，还是上了些年岁的师长。
<br/>　　书院的混乱并未持续多久，大概三天后，书院便稳住了眼前混乱的形势。目前书院的大小事务，都是由张明理负责。并未出现争权夺势的事情发生，大家同仇敌忾，想法都是一致的：找程大雷复仇。
<br/>　　这一日，书院大厅内一片肃穆，张明理并未坐在首座，而是坐在次席。那张空着的椅子，是给夫子留的。大家看到这张椅子，便想起夫子的音容相貌，不得不说，夫子生前还是个很和蔼的老人。
<br/>　　于是，就是对程大雷咬牙切齿的狠。
<br/>　　张明理的脸色不是很好看，原因是他刚刚得到消息，三天前，有一人骑着黑牛出城，根据相貌的描述，当是程大雷无疑。
<br/>　　可当时，书院还处于悲痛之中，疏忽了对城门的封闭，也没想到，程大雷竟敢大着胆子回到琅琊城，于是，也就看着他在眼皮子底下溜之大吉。
<br/>　　这，当然不是一件令人愉悦的事情。
<br/>　　在这几天内，人人心如刀绞，书院已经出了好几篇『讨贼赋』『怀夫子赋』之类的文章，辞藻朴实，真情动人，都算是当世一等一的文章。
<br/>　　大家复仇的心情是坚决的，本该立刻派出高手，去追杀程大雷。但眼下，还有一件极其尴尬的事情，令人不知该如何是好。
<br/>　　这个问题是：程大雷的确很了得。书院摆下天罗地网，七十二剑阵都摆开，最后也没能困住程大雷。如今再想重设白沙岛上的大局已经不可能，书院纵有高手，怕就算找上程大雷，也是去送死。
<br/>　　大厅内一片沉默，棘手的便是这件事。
<br/>　　“师兄，我倒是有一计，必能取程大雷性命。”
<br/>　　李亚子托着手出现在大厅，他自从被程大雷废掉一只手后，性情更加阴鹜。当然，他原先的性格也很歹毒，所以在书院并没有什么朋友。
<br/>　　“师弟有什么主意？”张明理问。
<br/>　　李亚子阴沉着一张脸，道：“书院弟子，遍布帝国，只要我们把要找程大雷复仇的消息散出去，程大雷必然走不脱。”
<br/>　　张明理眼睛一亮，明白了李亚子的意思。书院弟子桃李满天下，许多达官贵人也乐意将家中的子弟送到书院求学，日后，他们说不得就成为某个诸侯的谋士，某个大世家的家主，军中的某位将军……也不乏一些江湖高手。
<br/>　　书院经营了几十年，早已织成一张铺天盖地的大网，所以，夫子如果真的想要复国，赶上这个乱世，也未必做不到。不过，他已经老了，复国的心早就淡了，不过是认认真真为帝国培养一些人才罢了。
<br/>　　而这次程大雷很显然是犯了天下读书人的众怒，再加上火烧藏书楼的事情估计也和他少不了关系。
<br/>　　张明理深吸一口气，站起来道：“好，立刻将消息传出去，谁若能杀了程大雷为夫子报仇，书院三千弟子，愿唯他马首是瞻。”
<br/>　　口耳相传的力量是大的，极短的时间内，消息便散了出去，就连落凤城的李行哉也听闻了。
<br/>　　“啧……”他看着手下传来的消息：“程大雷这次犯了众怒，怕天下读书人都要杀他了。”
<br/>　　李婉儿担忧道：“那他这次会很危险吧？”
<br/>　　“危险自然是危险的。”李行哉挥挥手：“不过吉人自有天相，应该也没多大事。要我说这程大雷也是，你杀人就杀人，没事烧什么藏书楼，这件事，真真的太缺德了。”
<br/>　　“阿嚏”
<br/>　　李行哉奇怪的看着楚云生：“你怎么了，风寒？”
<br/>　　“那个没有，我没什么事，哈哈。”
<br/>　　楚云生表情有些尴尬，他误打误撞投到李行哉门下。倒也不是他有什么惊才绝艳的能力，的确是李行哉手头缺人，颇有种来者不拒的架势。
<br/>　　但楚云生并没有想到，李行哉竟和程大雷有些瓜葛，而楚云生也并没有透漏自己和程大雷认识的事。
