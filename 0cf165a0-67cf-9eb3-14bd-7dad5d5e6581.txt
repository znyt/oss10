夏夕绾根本就没有抢到书，连书的一角都没有碰到，简直是奇耻大辱，她战斗之心熊熊燃起时突然听到他出声，动作僵住了，她迅速看向他。
<br/>
<br/>        这下才发现两个人的姿势有多暧昧，她不知何时已经爬到了他的身上，刚才抢书的时候她隔着薄薄的衣料在他身上乱蹭。
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，后知后觉的才发现他在逗她，他是故意抢她医书来逗她的，她迅速捏着拳头锤了他两下，“陆先生，你怎么这么坏！”
<br/>
<br/>        陆寒霆将手里的医书直接丢了出去，丢在了松软的羊毛地毯上。
<br/>
<br/>        “哎，我的书！”夏夕绾想下床去捡。
<br/>
<br/>        陆寒霆箍住她盈盈一握的腰身将她抱了回来，“陆太太，你怎么一点为人妻的自觉性都没有，今晚就在看医书里度过了，你当我是个摆设么？”
<br/>
<br/>        “那…那你想怎么样，刚才不是才说好让我再长长的么？”
<br/>
<br/>        “我说了让你再长长，但是我已经长好了啊，你整天研究医书，怎么不研究研究你老公，你老公最近有点身体不适？”
<br/>
<br/>        夏夕绾心里咯噔一跳，他不会是又要发病了吧？
<br/>
<br/>        “你哪里不舒服，给我看看，是脑袋么？”夏夕绾将小手放在了他的额头上，也不烫啊。
<br/>
<br/>        陆寒霆将她的小手拿了下来，“我身体有一个地方最近经常不适，看不见你的时候会好一点，看见你的时候就不好了，还会慢慢…变疼…”
<br/>
<br/>        夏夕绾迅速拧眉，巴掌大的绝色小脸已经十分凝重了，“陆先生，先别慌，我给你把个脉看看。”
<br/>
<br/>        陆寒霆看她认真的样子迅速拽住了她的小手，然后缓缓将她的小手放在了自己精硕紧窄的腰身上，“陆太太，你是要好好的看一看。”
<br/>
<br/>        夏夕绾的小脑袋“当”了一声，迅速明白了他的意思，她柔软的指腹碰到了他精健灼热的肌肤好像被电了一样，瞬间想抽回，“陆先生，我要离你远一点，你色火攻心的样子实在太可怕了。”
<br/>
<br/>        陆寒霆一把抱住了她，将她锁在自己的怀里，他覆在她雪白的耳垂边嗓音低哑的诱哄道，“陆太太~我的亲亲好太太~”
<br/>
<br/>        夏夕绾小脸爆红的迅速闭上了眼，大概不会有人知道陆寒霆这位权贵，翻云覆雨的一代商界大佬，他竟然也有撒泼，求欢的时候，她骨子都软了。
<br/>
<br/>        ……
<br/>
<br/>        夏家。
<br/>
<br/>        夏妍妍回了家，刚走进客厅就听到了夏振国和李玉兰的争吵声。
<br/>
<br/>        夏振国脸色阴沉的骂道，“李玉兰，你真是一个丧门星，当初如果你没有让绾绾替嫁，而是让妍妍嫁进幽兰苑，那现在妍妍就是陆太太了！”
<br/>
<br/>        李玉兰气不过，迅速回嘴道，“夏振国，当初替嫁也是你点头的，谁知道幽兰苑的主人是陆寒霆，现在你将全部责任都推到我的头上，你自己就没有错吗？”
