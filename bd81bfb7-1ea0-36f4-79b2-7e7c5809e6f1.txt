　　收回目光，慕清霄并不打算在此位面多做停留，旋即化作一道剑光，向聂小倩所在的方向掠去。
<br/>　　以他现在的境界，瞬息百里丝毫不为过。
<br/>　　朝阳东升，几十个呼吸的时间，慕清霄就穿过荒山野岭，抵达万里外的一座城市上空。
<br/>　　望着城门上的‘京城’两字，慕清霄视线落在繁华的城市中，蜿蜒曲折的大街小巷，视线的尽头则是金碧辉煌的宫殿。
<br/>　　身影一闪，慕清霄就来到地面，隐匿气息，缓步向城内走去。
<br/>　　城门口有士兵把守，检查进出的行人，但慕清霄隐匿气息，周围的士兵或行人根本不可能注意到他。
<br/>　　来到繁华的京城，慕清霄走在繁荣的街道上。
<br/>　　如果记得没错，聂小倩的身份，似乎原本就是京城的富家小姐，没想到脱离千年树妖的掌控后，重新回到了京城。
<br/>　　不多时，顺着蜿蜒的街道，慕清霄来到城北，最后在一座酒楼下顿住了脚步。
<br/>　　“似水年华，倒是有些意思。”
<br/>　　酒楼不错，进出客人非常的多，但是酒楼中阴气旺盛，慕清霄站在门外就看见其中待客女子部分都是魂体。
<br/>　　看来，聂小倩也不算笨，看着酒楼中的景象，慕清霄就能猜出一二。
<br/>　　曾经，慕清霄与聂小倩有过肌肤之亲，或许是因为心法的缘故，两者之间也有那种微妙的感应。
<br/>　　比如，他现在虽站在酒楼门口，但非常确信，聂小倩就在这座酒楼中。
<br/>　　不再隐匿身形，慕清霄一袭白衣，面如冠玉，腰戴龙玉佩饰，看上去就像是世家翩翩公子，风姿卓越。
<br/>　　刚踏进似水年华，门口便有一位魂体女子迎了上来。
<br/>　　女子身着薄衫，姿色算是上等，杏眼中闪烁着困惑，粉/唇轻启，道：“欢迎公子来似水年华，不知公子是要找那位姑娘？”
<br/>　　女子正巧站在被酒楼遮挡的阴暗处，阳光照射不到的地方。
<br/>　　慕清霄知道后者身份，只是微微一笑，道：“我找小倩姑娘，不知小倩姑娘可有空一叙？”
<br/>　　话语刚落，女子嫣然一笑，似乎看出了他的来意，只是这一抹笑容落在慕清霄眼中，却是显得有些牵强。
<br/>　　“公子可能是第一次来似水年华，不知我们酒楼的规矩，如果公子想听小倩姑娘弹琴，请晚上在来，若是有其他要求，大可找其他姐妹们，相信一定能满足公子的需求。”
<br/>　　慕清霄怔了怔，后者言下之意他怎会不知道。
<br/>　　意思就是说，聂小倩只是卖艺不卖身，在这种时代，似水年华说好听点是酒楼，说难听点那就是青楼。
<br/>　　青楼就是专门给有钱人提供乐趣与服务的地方，招待客人的都是些风尘女子。
<br/>　　当然，这只是一部分女子，许多名门闺秀都有专门的教育，除非沦落到需要卖身的地步，否则根本不可能接客。
<br/>　　而聂小倩曾经就是名门闺秀，虽现在已经是鬼魂，但第一次却是给了慕清霄，而且自身修为不弱，自然不可能出来接客。
<br/>　　在京城开酒楼也是生活，她们虽是鬼魂，但也希望像正常人一样生活。
<br/>　　见后者热情的模样，慕清霄道：“姑娘误会了，我与小倩姑娘乃是旧识，希望姑娘前去通报一声。”
<br/>　　“……原来公子不是来找乐子的，倒是我误会了，请您在大厅稍等片刻。”
<br/>　　诧异的瞥了眼慕清霄，女子脸色有些古怪，转身向酒楼内部走去。
<br/>　　慕清霄紧随其后，进入酒店后，其中的光线就暗淡下来，甚至可以说光线根本照射不进来，难怪魂体敢在白日出来。
<br/>　　甚至，招待男性客人。
<br/>　　酒楼中所有的窗户，只要能透光的地方都被红色帘幕遮挡了起来，墙上则点着橙黄的长明灯，光线敞亮。
<br/>　　来到客厅后，慕清霄就察觉到许多异样的目光，甚至有的魂体脸色惨白，露出惊惧之色。
<br/>　　因为，其中有的魂体居然是当初在兰若寺中的女鬼。
<br/>　　慕清霄稍稍思考了下就猜出了前因后果。
<br/>　　必然是聂小倩在离开兰若寺后，将这群女鬼都聚集了起来，最后在京城落脚谋生。
<br/>　　显然，这群女鬼都认出了他来。
<br/>　　毕竟，慕清霄的容貌实在是太扎眼了，脸颊俊逸完美，甚至透着一丝小邪魅，简直像上天的杰作。
<br/>　　而且，由于修为与血脉的缘故，浑身都透着一股出尘高贵的气质。
<br/>　　不多时，二楼就有两道身影徐徐走了下来。
<br/>　　慕清霄抬头望去，只见女子身后跟着一位身材娇小的女子，身穿翠绿薄衫，长发披肩，说不出的可爱。
<br/>　　此时，女子也注意到了慕清霄，杏眼一瞪，捂嘴惊呼出声道：“是你，真的是你啊……”
<br/>　　女子正是聂小倩的小跟班，小蝶。
<br/>　　只见小蝶将女子打发后，快速来到慕清霄身旁，左右打量了他几眼，道：“你怎么知道我和小倩姐姐在这里？”
<br/>　　慕清霄微笑道：“想知道，自然就知道了，而且我不叫‘你’。”
<br/>　　小蝶顿时云里雾里，反应过来，俏脸一红，道：“慕……慕公子，你消失了两年，没想到真的来找我和小倩姐姐了呢。”
<br/>　　“带我去见她吧。”
<br/>　　“喔，你跟我来吧。”
<br/>　　小蝶撅着小嘴，似乎有些失望，随后带着慕清霄向二楼走去。
<br/>　　跟在小蝶身后，慕清霄发现，小蝶居然已经是筑基修为，记得当初在兰若寺的时候，似乎只是个练气三重的小女鬼。
<br/>　　看来，聂小倩是真的将她当成了自己人，并且将《玄阴素女经》都传给了她。
<br/>　　不过，慕清霄也没在意，当初他就说过，心法很重要，任凭她自己决定。
<br/>　　而且，后者似乎只传给了小蝶。
<br/>　　跟在小蝶身后，最后脚步停在了三楼拐角后的最后一间房间门口。
<br/>　　三楼似乎不招待客人，专门给自己人居住的，这里就是聂小倩的房间。
<br/>　　“姐姐就在里面，你自己进去吧。”
<br/>　　小蝶似乎有些闷闷不乐，瞪了眼慕清霄后，便转身离开。
<br/>　　微微摇头，慕清霄轻轻敲响房门，随后其中就传来一道悦耳清冷的询问声。
<br/>　　“是谁？”
