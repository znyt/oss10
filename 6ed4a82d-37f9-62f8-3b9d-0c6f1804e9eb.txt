　　昆仑山，玉清宫大殿。
<br/>　　道印天尊元始一袭白玉道袍，高坐云床之上，目中宝光一闪，法相庄严，慧耳垂光，手持三宝玉如意，轻轻摇动，谈论玄道，指点造化之理，讲说无极之妙。
<br/>　　在他的对面，道德天尊老子一声素袍，清淡无为，神色沉默，一脸专注，灵宝天尊通天一身青色道袍，剑眉挺翘，神色微动，似有所悟。
<br/>　　只听元始天尊口吐大道之言，化为五轮紫金光明圆象，龙腾虎跃，碧落緾虚，天霞流曼，神风鼓激，万音相合。沉浮清浊，一样婉转，错而成歌，其声响彻，交唱诸天。
<br/>　　“……欲免生死，不入流浪，先当无心。尔若无心，即无三业。众生迷妄，指识为心，分别有无，计度生死。心本妄名，以妄取名，心则虚幻。觉了能知，本是道元，何关心事。人心着境，即往生死，若无妄心，不往生死，即用短轮转……”
<br/>　　天尊说法，自是直指大道本质，说得那是突破混元大罗金仙的关键，为两人的突破做准备。
<br/>　　“混元，即为浑源、浑圆，道法不圆，何为证混元？混元者，元气为分，混沌如一，元气之始也，大道之源也。”
<br/>　　“元气生于混沌之中，于明之内，暗之外。因明暗之间生空洞，空洞之内生太无，太无变而三气分明，玄气、元气、始气，三气混沌，生太虚而立洞，因洞立无，因无生有，因有生空，观空无之变化，便能虚生自然。”
<br/>　　“上气曰始，中气曰元，下气曰玄，玄气所生在于空，元气所生在于洞，始气所生在于无，故能一生二，二生三，三生万物，万物负阴而抱阳，太极也。”
<br/>　　“一生二，二生三，三者化生，以至九玄，从九返一，乃道真也。阳清成天，滓凝成地，中合成人，三者分判，万物禀生，日月列照，五属唤名，乃圣人也。三者生于三气之元，处于元气之始，极乎无极，混元无极也。”
<br/>　　“混元无所不包，无所不有，阴阳也好，剑道也罢，无为亦然，截天也罢，都为混元，混元一气，道即混元。智取于身心，让心与身合，空而不空，与道同体，与道合一，即证混元。”
<br/>　　元始天尊将成就混元之道讲解与老子和通天，两人听得如痴如醉，再结合自己的大道，一一印证，他们明悟许多，对证道一事更有把握了。
<br/>　　只见那元始天尊不停，继续说道：
<br/>　　“证道，证道，便是要想天下证明你的大道。证法有三，一者，以力证道成圣，以自身强大的法力破开天道束缚，炼化大道之基鸿蒙紫气，介入于天道之中索取道果。
<br/>　　其二，三尸证道，斩断善念之尸，恶念之尸，执念之尸。专断自身七情六欲，以淡漠之心纵观时间千姿百态，万事万物。在明确自我、本我之后，在重新将三念之尸融合。炼化大道之基鸿蒙紫气，寄托元神于天道之中，可证得混元。以此法成圣，体悟凝结天道之心，调用大道至理甚多。”
<br/>　　“其三者，功德证道，以无量功德感动天道，大道皆为其所感动，至仁成圣。以偌大功德、气运、原力，引动天道宏念，破开自身枷锁，炼化大道之基鸿蒙紫气，寄托元神与天道之中，可证得混元。”
<br/>　　“你们，可否想好要以何法证道成圣？”在为两人分析了三种成圣方法之后，元始向他们问道。
<br/>　　“那……大兄又是以何种方法证道？”通天问道。
<br/>　　元始天尊淡淡地看了一眼通天，道：
<br/>　　“本座走的是以力证道之路，将自身大道修到混元一体的地步，再以自身强大法力，破开天道空间，炼化大道之基，凝结圣人道果。”
<br/>　　“这……大兄，三种方法各有何种特点？”老子犹豫了一下，看向元始，问道。
<br/>　　元始深深的看了一眼老子，看得他不明所以，才淡言道：
<br/>　　“三种方法皆可证道，证得那历万劫而不灭、染因果而不沾，与天道同在，与大道同存的混元圣人。不过，虽说如此，但三种方法，也各有好坏。其中，以力证道，证的是混元大道，靠的完全是自己，最为艰难，证道之后，身心自由，天地不管，其战力也完全靠自己。而三尸成圣，中等，战斗之时，可调用天道之力作战，不过，就是有时会受天道的影响以及控制。”
<br/>　　“功德成圣，地位最贵，功德圣人与众生意志加持其身，但因为不是靠自身证道，而是靠无量功德愿力而成道的，其战力最弱，而且受天道影响亦最大，有时身心不得自主，走出洪荒宇宙后，没有天道之力的加持，便不为圣人。”
<br/>　　“什么？受天道控制？这怎么可能？”
<br/>　　老子与通天大惊，这么说来，自己若是以此成道，便一生不能走出洪荒了？
<br/>　　元始不管他们的心情，继续说道：
<br/>　　“以力证道，不管是对道的领悟，还是要破开天道空间的大法力，你们都还差得远。三尸成圣，最先需要的就是斩去三尸，你们三尸已斩，而且因为之前积累深厚的缘故，三尸都已圆满，对道的领悟也足够了，倒是可以考虑。至于，第三种方法，功德成圣——”
<br/>　　“我等本是盘古元神所化，生而具有开天大功德，若是在谋划一番，倒也足够你们成圣了，但，我想你们也不想以此成道吧。所以——”
<br/>　　“所以，你们还是用第二种方法，三尸证道吧。若是成圣后，再谋划一番功德，说不定还可以再次以功德成圣，两次成圣，你们就要比其他圣人强许多。”
<br/>　　“啊？还可以这样？”
<br/>　　两人顿时觉得有些不可思议，还可以两次成圣，怎么我之前没有想到？
<br/>　　“嗯。”
<br/>　　两人见此，觉得元始说得在理，他们默默思索了一番，最终还是决定以三尸之法成圣，虽然三尸之法成圣不是最强大的圣人，但它也不是最弱的圣人啊，三尸成圣后，不高不低，居于中间。若是以此成圣后，究竟谁强谁弱还不一定呢？毕竟，他们对道的领悟也会有所不同，还有各自手中的灵宝强弱，一样会影响自身的战力。
<br/>　　比拼法宝，三清会怕过谁吗？笑话，他们身上的先天至宝，就不只有一件两件，会怕谁？
<br/>　　至于成圣之后的事，之后在考虑，毕竟不在其位，不谋其事嘛。
<br/>　　最终，他们决定，三年之后，三尸融合，证道成圣！
