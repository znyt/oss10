
<br/>　　<br/>　　    从这老人嘴里诉说出来的是一个挺凄惨的故事。<br/>　　<br/>　　    往前推几十年的时候，这老人中年时在蛊上面的造诣是非常高的，在黔地中他下蛊的本事都是出神入化的，后来巫门不知道怎么的，就听说了此人，于是前来请他出山去巫门，但对方没答应，并且他养蛊也纯粹是为了寨子里的传承不断，完全无心任何纷争，只想守着寨子过一辈子就可以了。<br/>　　<br/>　　    巫门前后几次都没能请动他，这件事就不了了之了，又过了两年吧，寨子里忽然来了个青年，说是过来游山玩水无意中发现了这个苗寨，并且还挺喜欢这个地方的，于是便住了下来，而且一住就是半个多月。<br/>　　<br/>　　    然后他为人来十分会来事，跟苗寨的人相处的也很好，最关键的是，这青年对蛊很感兴趣，在这方面天赋还特别的高，达到了触类旁通的程度，属于一点就透的那种。<br/>　　<br/>　　    于是这青年在寨子里就安顿了下来，过了两年后，他又娶了这里的一个苗女为妻生活在了一起，这苗女就是这老人的女儿，从此以后他也算是入赘到了这个寨子。<br/>　　<br/>　　    到这还算是个挺美好的故事，但是过了几年后，美好一去不复返了。<br/>　　<br/>　　    那青年把这老人手中所有下蛊和解蛊的手法全都给学了过去，基本上算是把老人的家底都给掏空了，但这还不算，凄惨的故事就这么发生了。<br/>　　<br/>　　    某一天的时候，那青年和苗女还有老人进山采药，谁也没曾想到的是变故突然就发生了，青年将那苗女给推下了山涧，然后又逼着老人将他手中最后一项看家的本事给交出来，就是解这虱蛊的方式，他只是学会了下蛊，至于怎么解老人一直都没告诉他。<br/>　　<br/>　　    到最后，这老人说啥也没干，因为他知道自己交出去之后，对方肯定也不会放过他的，就说什么都没吐露出来，恼羞成怒的青年最后用刀砍断了这老人的四肢也给他推下了山涧，但没想到，他大难不死，被寨子里的人给发现然后救了回来，然后一过就是几十年了。<br/>　　<br/>　　    小草听完之后对这个凄惨的故事潸然泪下，王惊蛰则是心里一顿冷笑，也庆幸自己断言的没错，对方给出的解蛊的法子，是根本不存在的，吴洋和周深压根就不知道。<br/>　　<br/>　　    真要是傻傻的就信了他们，小草可真就彻底成为一株死草了。<br/>　　<br/>　　    这老人随后就和王惊蛰说了下自己的条件，总而言之一句话，很简单。<br/>　　<br/>　　    那就是，有仇报仇，有怨报怨！<br/>　　<br/>　　    老人可以为小草解蛊，但要的条件就是王惊蛰把当初那个青年给手刃了，如果在他能力范围之内的话，最好再把巫门给收拾了一顿那就更好了。<br/>　　<br/>　　    “我这辈子，活成这样了还没有死，全靠着心里的一口气在撑着呢，气没散我就舍不得死啊，还好，还好……”藤床上的老人脸色红润的用两只断手撑起了自己的身子，瞅着小草和王惊蛰点头说道：“还好，老天还算长眼了，让我还能有几年好活的时候，碰上了你，呵呵……”<br/>　　<br/>　　    王惊蛰轻声说道：“其实，我都已经打算好了，如果她身上的蛊解不了的话，我回去之后就要把他们给一锅端了，您对他们是恨得啃其骨，我也不比你差多少，您放心，我说的话不是敷衍，我跟他们之间的仇怨已经结的这么深了，自然没有惯着的道理，于情于理我都会下这个手的”<br/>　　<br/>　　    “那是你的问题，我要的是你答应我的”老人一脸戾气的说道。<br/>　　<br/>　　    王惊蛰顿时笃定的点头说道：“好，我答应你……如果做不到的话，我就受你们苗人中最歹毒的蛊毒而死！”<br/>　　<br/>　　    这天晚上，茅小草和王惊蛰夜宿在了苗寨里，这是一月有余中两人睡的较为踏实和香甜的一觉，因为那老人明天答应为小草解蛊。<br/>　　<br/>　　    奔波千里，深山中走过了几十个昼夜，在第四十八天的时候，一切的努力到底还是没有白费。<br/>　　<br/>　　    隔天，清晨时，寨子里忽然响起了一串悦耳的山歌，两人从睡梦中醒来之后，站在窗前看着外面，寨子中的人开始了一天的劳作，寨子外河道旁就是一片田地，此处的人靠山吃山靠水吃水，过着与世无争的生活。<br/>　　<br/>　　    他们日出而作日落而息，没有外面世界的浮华和喧嚣，生活中最多的就是两个字，平静。<br/>　　<br/>　　    “我也想，等哪一天老了之后，也过上这种不知道什么叫做忧虑的生活就好了，除了吃喝拉撒睡以外，什么都不用考虑……”<br/>　　<br/>　　    王惊蛰摸了摸她的头发，笑道：“是不得算我一个？”<br/>　　<br/>　　    小草抿嘴一笑，说道：“你表现的还算不错，我可以把你带上”<br/>　　<br/>　　    绝大多数的时候，女人对男人的要求真的没有那么多，在吃饱喝足之后，可能要的最多的就是陪伴和长情了，王惊蛰在这方面做的，确实有可圈可点之处，坦白来讲，还真挑不出什么毛病来。<br/>　　<br/>　　    简单吃了点早餐之后，王惊蛰和小草就来到了那栋草屋前，但老人没让王惊蛰进去让他在外面等着，小草是独自一人进去的。<br/>　　<br/>　　    王惊蛰在屋前席地而坐，紧张，有点皱眉的看着茅屋，尽管知道对方已经答应了他们，但没见到活蹦乱跳的小草，他的心还是提着的。<br/>　　<br/>　　    但他是进不去的，解蛊的方式是老人最后一手压箱底的本事，外人是看不了的。<br/>　　<br/>　　    不知过了多久，一股淡淡的腥臭味飘了出来，王惊蛰忽然发觉，他脚下的蚂蚁还有附近飞着的蝴蝶都仓惶的朝着四周逃窜着。<br/>　　<br/>　　    这股味道挺邪恶的，让人闻了之后都一种抗拒的心里。<br/>　　<br/>　　    两个多小时之后，茅屋的门开了，小草拖着跟灌铅了似的两条腿走了出来，王惊蛰连忙上前问道：“完事了？”<br/>　　<br/>　　    “哪里有这么快，这才刚刚开始，至少还得要几天才能解开”<br/>　　<br/>　　    “那今天不是最后一天了么？”王惊蛰皱眉问道。<br/>　　<br/>　　    “唰”小草摊开手掌，她的手里攥着一只黑色的甲虫，就是当初从丧尸身体里飞出来的那玩意。<br/>　　<br/>　　    “妈蛋的，就它让我糟了几十天的罪，还差点香消玉损了……”
