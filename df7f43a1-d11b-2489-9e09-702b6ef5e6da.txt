　　“星百里？”
<br/>　　“是谁？”
<br/>　　“这难道是百里长老的真名？”
<br/>　　听到楚风眠的话，在那老者的背后，一群天枢殿弟子，纷纷议论了起来。
<br/>　　而那被称为星百里的老者，则是脸色猛然一变，死死的盯着楚风眠，吃惊无比的说道。
<br/>　　“小子，你究竟是什么人？”
<br/>　　也不怪他吃惊，星百里可是活了万年以上的人物了，在剑道门存在的那个时代之中，他就存在。
<br/>　　经历万年的时间，一直都是在天枢殿之中潜修，绝对是老古董之中的老古董。
<br/>　　别说是外人，纵然是在天枢殿之中，知晓他真名的人，都是寥寥无几。
<br/>　　人人都知晓他是剑道门的百里长老，但是星百里这个名字，他却是几乎数千年，没有被人提起过了。
<br/>　　楚风眠仅仅看了他一眼，就叫出了他的名字，让他如何不吃惊。
<br/>　　“剩下的人，别躲着了，还要我一个个请你们出来吗？”
<br/>　　楚风眠没有回答那星百里的话，而是目光又看向了一侧，这一侧，依然是一片星空。
<br/>　　一眼望上去，几乎是看不见一点点的灵力波动。
<br/>　　随着楚风眠开口，那一侧的星空，依然是没有一点变化。
<br/>　　“看来荒古门的人，还是跟以前一样，藏头露尾的！给我滚出来！”
<br/>　　楚风眠大手五指猛然一抓，便是向着那一片星空之中，抓了过去，这一片星空，都在楚风眠的五指之下，抓的扭曲起来。
<br/>　　这一片扭曲的星空，逐渐化为了一条古老金色的大道，这一条古老金色的大道，仿佛是连接这古老的荒古时代之中。
<br/>　　这一条古老金色大道，虽然位于星空之中，但是仿佛在这古老金色大道的尽头，就是另外一个时代。
<br/>　　贯穿于时代之中的一条长河。
<br/>　　“这就是九域之中的秘术？穿梭时代，这简直无法破解！”
<br/>　　青梦见到这一条古老金色大道的出现，不由的吓了一大跳，这一招的玄妙，超出她在大罗天星域所知晓的无数秘术。
<br/>　　荒古门，传承数万年，哪怕是九域没落了，荒古门之中留下的无数秘术，也依然传承了下来。
<br/>　　燕皇也是同样的震惊，在他们看来，这九域已经是没落了。
<br/>　　可哪怕是在没落，九域依然是曾经的圣域，瘦死的骆驼比马大，许多古老的秘术，在大罗天星域之中，也堪称强横。
<br/>　　“小子，看来你的确是剑道门的人，除了剑道门的人，天下间没几个知晓我们荒古门的秘密。”
<br/>　　在这古老金色星河之中，一声苍老的声音，缓缓响起。
<br/>　　“藏头露尾的，滚出来说话！”
<br/>　　楚风眠听到这苍老的声音，冷喝一声，手心之中猛然一捏，他的五根手指，突然之间打出了一道道剑气，向着那古老金色长河之中斩杀而去。
<br/>　　嘶拉！
<br/>　　这一条古老金色长河，在这一道道剑气之下，被突然斩裂开来了无数口子，只见众人都感觉到。
<br/>　　这一条古老金色长河，已经是跟荒古时代，切断了联系，随着楚风眠手心猛然一拍。
<br/>　　这一条古老金色长河，就这样在楚风眠的面前被拍成了碎片，同时其中一道道身影，也被打飞了出来。
<br/>　　这一道道身影，身上的衣着都是一样，在他们的胸口之上，一个大大的荒字，代表了他们的身份。
<br/>　　荒古门弟子！
<br/>　　这些荒古门弟子，都被从古老金色长河之中被楚风眠一掌逼迫了出来，不少的身上，都是受了一些伤势，愤怒的看着楚风眠。
<br/>　　而其中为首的一位老者，目光却是不可思议的看着楚风眠，震惊道。
<br/>　　“你，你到底是什么人？这荒古大道，你怎么能够打破？”
<br/>　　这震惊的老者，正是方才在金色古老大道之中开口的那一道声音。
<br/>　　荒古大道，荒古门之中最为强横的秘术，沟通荒古时代，足以使人深处荒古之中。
<br/>　　一切轰击进来的力量，都会被打入到荒古之中来，让其中的人，却是会毫发无伤，如果没有交手过，可以察觉出其中的破绽的话，这一招，几乎是堪称无敌。
<br/>　　可现在楚风眠一出手，就是轻易找到了荒古大道之中的破绽，几道剑气，就将这荒古大道所打破。
<br/>　　这是对荒古门之中，有着极深的了解，才可能做得到的事。
<br/>　　如今这一位老者的惊讶，丝毫不在那星百里之下。
<br/>　　又是一个熟人。
<br/>　　楚风眠看向这老者的目光之中，露出了几分玩味的笑容，冷笑一声道。
<br/>　　“东宫绝？看来你跟这星百里一样的废物，万年岁月，都活到了狗身上了吧，现在才不过是九劫古帝的境界，丢进了东宫世家的脸。”
<br/>　　东宫世家，荒古门之中最大的世家，荒古门历代无数门主，都是出身自这东宫世家。
<br/>　　这东宫绝，也是曾经东宫世家的一位弟子，在万年之前，就踏入了准帝的境界，也算是当时的一位天才了。
<br/>　　可现在，万年之后，这东宫绝才不过是九劫古帝的境界。
<br/>　　在楚风眠看来，这东宫绝，距离半步天人境，还有着很长的距离，这种突破的速度，在武者之中，算是平常。
<br/>　　不过若是在楚风眠的面前，简直是跟废物没有任何的两样，楚风眠重生归来，甚至才不过数年的时间，就已经是拥有了如此的实力。
<br/>　　岁月活在狗身上，从楚风眠的口中说出来，倒是无比具有说服力。
<br/>　　“师弟，认输这二人？”
<br/>　　青梦听到楚风眠的话，看向这两名老者，心中无比震惊。
<br/>　　她当然看得出来，这星百里，东宫绝的年纪，只怕都有着万岁之上了。
<br/>　　可听着楚风眠的语气，似乎是对着二人，都是极为熟识的，让这一幕看起来，是无比诡异。
<br/>　　不止是青梦，那一群荒古门弟子，看着楚风眠也是有些不可置信，他们其中不少人，都听闻过楚风眠的名字。
<br/>　　知晓楚风眠是九域之中，北邙学院之中新崛起的天才弟子。
<br/>　　可没想到楚风眠居然是跟他们老祖宗一样的长老相识。
<br/>　　“小子，你到底是谁？”
<br/>　　东宫绝看到楚风眠说出他的名字，怒声吼道。
