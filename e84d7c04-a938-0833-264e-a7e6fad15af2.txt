　　察觉到四周压迫过来的恐怖气息，红裙女子那诡异的笑容瞬间凝固在脸上，眼中浮现出一抹惊恐。
<br/>　　下一秒，她就感觉颈部传来一阵挤压感，那颇为秀丽的脸蛋，顷刻间变得狰狞无比。
<br/>　　她知道来者不善，却不曾想到后者道行如此恐怖！
<br/>　　望着显露出厉鬼姿态的红裙女子，慕清霄古井无波，再一次开口问道：“你是也叶沉香？”
<br/>　　“男人都该死，休想我告诉你！”
<br/>　　虽然生命不再自己手中，但红裙女鬼丝毫不惧，张嘴就朝慕清霄吹了一口阴风。
<br/>　　见能够令人陷入幻境的阴风不起效果，便将脑袋瞥向一边。
<br/>　　慕清霄剑眉一皱，也不打算继续跟她废话。
<br/>　　星眸中浮现三颗勾玉，四目相对，红裙女子的眼中也倒映出三颗勾玉，狰狞的表情也变得呆泄起来。
<br/>　　“你是叶沉香？”
<br/>　　“是。”
<br/>　　其实，看到叶沉香的第一眼，慕清霄就已经猜到了她的身份，不经意间流露出的气质，可不是普通丫鬟身上所有的。
<br/>　　哪怕她已经死去数十年，但生前的气质依旧长存，不经意间就会流露出来。
<br/>　　更何况，她的穿着与打扮，完全不像是丫鬟，倒像是某富贵人家的千金大小姐。
<br/>　　确认红裙女子的身份后，慕清霄就将储存空间中的一般玉衡取了出来，道：“见过这东西吗？”
<br/>　　“见过，就在主人手中。”
<br/>　　看了一眼玉衡，叶沉香目光呆泄的点了点头。
<br/>　　目光一凝，慕清霄对于叶沉香口中的主人有些好奇，道：“带我去见你的主人。”
<br/>　　“是。”
<br/>　　如同傀儡一般，叶沉香漂浮在前面，慕清霄则紧随其后，慢悠悠的向玄廊的深处的阴暗中走去。
<br/>　　玄廊不知通往院落的最深处，越是深入，阴气就越是浓郁。
<br/>　　由于数十年没有人迹的缘故，玄廊通道两侧的墙壁上都布满尘埃，甚至滋生出一陀陀青紫色霉菌。
<br/>　　通道中的霉味很浓，慕清霄觉得有些不适应，便将嗅觉屏蔽。
<br/>　　几分钟后，面前就出现一扇金属大门，大门上的厚重的铁锁已经掉落在地上，此处似乎是叶家的仓库。
<br/>　　当叶沉香在仓库门口停下，慕清霄知道，所谓的主人恐怕就躲在这仓库中。
<br/>　　因为，他能感觉到，此处不但阴气旺盛，连天地间的灵气也格外的浓郁，甚至不断的从四周汇聚过来。
<br/>　　也只有仙器玉衡，才会出现如此异象。
<br/>　　“轰隆......”
<br/>　　伴随着一阵轰鸣声响，诡异的一幕出现，只见厚重的金属大门慢慢开启，旋即一股浓郁的阴气自其中狂涌而出。
<br/>　　“桀桀桀，真是不知死活的小子，居然敢闯进本王的巢穴，不过，你体内的气息可真是诱人，比山下的普通人美味多了。”
<br/>　　阴气回荡，整个玄廊中暗淡的光线，仿佛都在此刻消失，阴森可怖的笑声响起，一团黑气就出现在玄廊中。
<br/>　　黑雾悬浮在半空，最后凝聚成一个魁梧的身躯，身躯被黑雾笼罩，却释放出极其恐怖的气息。
<br/>　　望着悬浮在空中的黑雾，慕清霄有些不确定的开口道：“梦魔？”
<br/>　　梦魔是古剑奇谭中一种极其强悍的妖魔，喜欢躲在阴气旺盛的地方，比如坟墓，乱葬岗更是它们的最爱。
<br/>　　梦魔性格极其残忍，最大的喜好，便是吸食人体内的阴气与阳气，以此来增强他们的修为。
<br/>　　不管男女老少，只要是被它们看上的，不出半个月便会离奇死亡。
<br/>　　当年，梦魔一族祸害世间，几大门派的弟子也惨遭毒手，顿时引起了修真宗门们的忌惮。
<br/>　　要知道，一般的法术对于梦魔根本无效，也就是说，只要修为低于它，它简直就是无敌的存在。
<br/>　　这也正是宗门忌惮它们，甚至灭之而后快的原因。
<br/>　　梦魔一族覆灭，修真宗门的高层也终于安心，只是没想到，现在竟然还有梦魔存在与世间。
<br/>　　关于梦魔的信息，慕清霄也是当初在天墉城，翻阅古籍的时候，不经意间看到的。
<br/>　　只是没想到，真会有遇到梦魔的一天。
<br/>　　“桀桀......你小子居然能猜出本座的身份，既然如此，此处就是你的葬身之地。”
<br/>　　被慕清霄一语道破身份，梦魔心中一惊，梦魔一族已经消失百年，居然还有人能认出它来。
<br/>　　要是此事传到修真宗门耳中，恐怕它往后都不得安宁，顿时一股极强的杀意涌上心头。
<br/>　　绝不能让此子离开，哪怕尸首也不行，此事结束后，碧山脚下的村民也必须全部灭口，以免消息泄露。
<br/>　　想到此处，一只由黑雾凝聚而成的爪子，直接向慕清霄的心脏掏去。
<br/>　　冷哼一声，慕清霄调动灵气，道：“练虚三重，简直不知死活，正好用你来试试剑法的威力。”
<br/>　　“空明剑！”
<br/>　　话语刚落，恐怖的气息就将玄廊顶部掀开，数十道虚幻剑影自天际纵贯而下。
<br/>　　未曾反应过来，包裹梦魔的黑雾就被剑影刺成马蜂窝，梦魔似乎也遭受重创，不断的发出气急败坏的咆哮声。
<br/>　　“可恶，可恶的人类，本座要将你的生命吸食殆尽，在将你做成尸魂，杀光山下的所有人！”
<br/>　　“哼，等你能活着离开在说吧！”
<br/>　　浑身剑气凝聚于一体，慕清霄食指与中指合拢，只见一道由剑气凝聚而成的巨大剑柱纵贯而下。
<br/>　　“太虚剑！”
<br/>　　在恐怖的剑气下，梦魔极力挣扎，黑雾狂涌，但碍于慕清霄实力太恐怖，丝毫没有反抗的余地。
<br/>　　巨大的剑影仿佛打破时间与空间的限制，瞬间纵观而下，将梦魔与地面齐齐贯穿。
<br/>　　恐怖的剑气肆虐，玄廊两侧的墙壁早就破坏殆尽，四周更是满目疮痍。
<br/>　　在太虚剑将梦魔贯穿后，慕清霄目光一凝，黑雾依旧在蠕动，旋即其中发出一道虚弱的声音。
<br/>　　“小子，本座记住你了。”说罢，梦魔就化作一道黑雾，以一种诡异的速度向自闲山庄外掠去。
<br/>　　慕清霄嘴角牵起一丝冷笑：“我不让你走，你能走的掉吗？”
<br/>　　话语刚落，刚掠过山庄的梦魔悬浮在了空中，声音中满是惊恐，竭斯底里，道：“不可能，以你的实力，怎么可能布下结界！”
