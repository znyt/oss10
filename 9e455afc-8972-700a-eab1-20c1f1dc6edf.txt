　　注意到刘哔目光中的兴奋，紫女眼中精光一闪，笑吟吟的继续开口道：“不过，第五层也只是入门阶段，真正厉害的，是最后的那四层。”
<br/>　　“最后四层每一个境界，都有通天彻地的威能，那种力量，不是凡人可以想象的。”停顿了一下，紫女叹了口气：“只不过，想要到达那种境界，实在是太难了。”
<br/>　　若有所思的点了点头，刘哔没有回话，而是根据紫女传授的方法，自发开始修炼了起来。
<br/>　　他已经将这套功法熟悉的差不多了，不知道是不是错觉，将这套功法翻译成白话文以后，他也没感觉有多么困难。
<br/>　　按照紫女所教的奇异姿势，刘哔盘坐在床上，慢慢感悟着周遭的天地灵气，不知道过了多久，在他的意识中感受到了一缕透明色的气劲。
<br/>　　说来奇怪，这股气劲明明没有颜色，可他依旧能从周遭的空气中分辨出来，刘哔心想，这或许就是紫女之前所说的天地灵气了。
<br/>　　感应到天地灵气，只是修炼的第一步，接下来最重要的，还是要将灵气吸纳进体内，并且环绕身体运行一个大周天，在体内种下灵种，这样才算真正拥有了修炼的资格。
<br/>　　因为心神的恍惚，差点没从修炼状态退了出去，感应到那道灵气的状态有些不稳，刘哔立马收敛心神，缓缓将气劲引入到了身体。
<br/>　　没有任何排斥，引入的过程很顺畅，透明色的气劲顺着体内脉络缓缓流动，颜色也逐渐变紫，最终形成一个深紫色的圆球，沉入到了丹田之中。
<br/>　　瞬间，一股逐渐增大的力量感从身体四处传来，刘哔嘴角勾起一抹笑容，他知道，自己现在应该算是成功了。
<br/>　　看着灵种在自己体内逐渐成型，刘哔并没有停止，而是继续勾起一缕天地元气，按照之前的路线缓缓运转着。
<br/>　　他想趁此机会，将灵种稳固下来，毕竟明天还有一场战斗要打，虽然一晚上的时间也起不到什么至关重要的结果，但也聊胜于无吧。
<br/>　　随着时间的推移，刘哔体内的灵种越来越大，修炼也逐渐开始进入了佳境，每时每刻，他的实力都在稳固提升着。
<br/>　　站在画卷之中，紫女盯着正在认真修炼的刘哔，脸上的笑意，并不比当事人要少，脸上的表情，也是一副满意至极的样子。
<br/>　　对于对方能这么快就可以凝结灵种，她也是稍微有些惊讶，虽然因为这套功法特殊性，修炼速度要比寻常功法快很多，但能做到这种地步，也证明了刘哔的天赋颇为不错。
<br/>　　可惜的是，这么好的天赋，却不是紫府的弟子，就这样当做肥料，也着实有些可惜。
<br/>　　修炼任何功法，都需要灵种作为媒介，只不过寻常功法使用天地灵气作为灵种，而她修炼的这个，却不太一样。
<br/>　　微微感叹了一下，紫女随即抛开了心中的想法，看到刘哔这边没有什么问题了，便自动切断了和外界的关联，床边上的画卷，自发闭合在了一起。
<br/>　　……
<br/>　　第二天清晨，刘哔是被一阵敲门声惊醒的，下意识的看了看表，他才惊讶的发现，从昨天早上修炼到了现在，接近一整天的时间，自己竟然都在修炼。
<br/>　　不过，好在收获不小，也没有浪费这一天的时间罢了。
<br/>　　“师弟，快起床了，吃点东西，我们还要去帮任老爷迁坟呢！”门外，文才依旧很执着的敲着屋门，从声音上来听，稍微有些急切。
<br/>　　随口应了一声，刘哔从床上跳了下来，已经僵硬的骨头发出一阵清脆的‘咔吧’声，但他却感觉，自己现在的状态，比往常好了不知道多少倍。
<br/>　　因为丧尸的特殊体质，他对于睡眠并没有什么需求，虽然想睡的时候也可以睡着，但不睡其实也不存在什么影响。
<br/>　　顶多，也就是稍微有些不习惯而已。
<br/>　　但是他昨天修炼的那套功法，本来就有恢复精神力的作用，自然会让他感觉神清气爽了。
<br/>　　随便找出了一件干净衣服换上，刘哔眼角一瞥，看到了床上不知何时卷起来的画像，不禁略微有些惊奇。
<br/>　　犹豫了一下，或许是出于不放心，他将画像拿起，贴身放好，方才推门走了出去。
<br/>　　“怎么这么慢啊，文才师弟都叫了你半天了，昨天叫你吃饭的时候也是，我们还以为你出了什么事呢。”身穿一身麻衣的秋生，蹲在院子中央，正在检查着要携带的物品，听见刘哔下楼的声音，不禁开口抱怨道。
<br/>　　想到了窗户上突然出现的小洞，刘哔笑了笑，没有言语，昨天他感应到有人来找过他，不过他当时修炼的正爽，并没有搭理那个找他的人。
<br/>　　现在看来，那个人应该是文才了，这家伙，还把自己的窗户纸给捅破了。
<br/>　　“阿哔，听说你昨天受伤了？”九叔身穿一身古板的黄色道袍，从一旁的花圃中走来，关切的看着刘哔问道：“伤势重不重，要不你今天在家里休息？”
<br/>　　恭敬的行了一礼，刘哔回答，道：“谢谢师父关心，是昨天弟子出去时，不小心被一个树枝给划到了，伤口很浅，现在已经痊愈的差不多了。”
<br/>　　点了点头，九叔也没有追问，习惯性的伸手，拍向了刘哔后脑勺，训斥道：“毛毛躁躁的，不过没事就好，赶紧去吃饭吧，等你吃完我们就出发。”
<br/>　　条件反射一般，偏头躲开了九叔的巴掌，他心中隐隐有一股戾气即将翻腾上来。
<br/>　　感觉自己的状态有些不太对劲，刘哔稳了稳心神，低头回道：“不用了师父，你们都吃完了吧，要不我就等回来以后再吃吧。”
<br/>　　丝毫没有察觉到刘哔的异样，九叔掏出一块样式古朴的怀表，查看了一下时间，点了点头。
<br/>　　“也行，时间快来不及了，我们现在就过去吧。”招呼了一声，九叔喃喃道：“要不过了正午，棺材就不太适合见光了。”
<br/>　　低头走在九叔后面，没有人注意到，刘哔此时的双眼血红一片，一缕缕凶煞之气，从他身体内部散发了出来。
