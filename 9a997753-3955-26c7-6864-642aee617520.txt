　　第53章
<br/>　　嘟，获得恐惧值88
<br/>　　系统提示消息令程大雷怔了怔，紧跟着他看到苏樱后退一步，拔下头上金钗抵住自己喉咙。
<br/>　　“你不要过来！”
<br/>　　程大雷怔了怔，忙往后退了退，举起双手。
<br/>　　“你别怕，我不会伤害你的。”
<br/>　　苏樱盯着程大雷，用那双令人着迷的大眼睛。她害怕这个从天而降，突然跳到自己身边的山贼。可是，明察秋毫的属性又告诉她，程大雷对自己并无恶意。
<br/>　　二人距离大概三步左右，尴尬得沉默着。程大雷不知该如何打破着沉默，忽然心中一动，从怀里掏出一枚香囊。
<br/>　　“喏，送给你，喜不喜欢？”
<br/>　　面前这张龇牙咧嘴的大脸，很难让苏樱再有什么提防之意，她看了眼杵在自己面前的香囊，微微颦起眉头。
<br/>　　“女人香，你从女人处来？”
<br/>　　呃……
<br/>　　这香囊的确是程大雷从柳芷那里顺过来的，却是没想到苏樱连这点都看得出来。唉，女人太过聪明，的确会令男人头疼。
<br/>　　“怎么会，是我特意买给你的，只是卖香囊的是个小姑娘。”
<br/>　　“假话。”
<br/>　　苏樱将金钗插回头上，顺手将香囊接过来，放在鼻尖嗅了嗅。
<br/>　　“香嘛，我特意买给你的，很贵的，她们说是最好的。”
<br/>　　“我从不戴香囊。”
<br/>　　“为什么？”
<br/>　　“天生丽质。”苏樱不动声色把香囊收在袖子里，问：“你为什么会来？”
<br/>　　“哈。”程大雷理理衣冠，用满怀深情的声音道：“青牛山一别，屈指不过月余，姑娘倩影却是留在我心头，挥之不去。我是食也不甘，睡也难免，朝朝暮暮难以忘怀。”
<br/>　　苏樱往后退了退：“你这个人……不正经。”
<br/>　　“我这次说的可是真的，特意过来看你的。你最近过得好不好？”程大雷目光落在苏樱脸颊，忽然一怔，下意识道：“你瘦了，难道……在减肥？”
<br/>　　苏樱微微侧头，避开程大雷的目光，心底有些波动。
<br/>　　她最近这些日子，又怎是区区不好两个字可以形容的。
<br/>　　苏樱在去黑石城的半路上被劫走，后来韩家被灭门，虽然她连韩玄之的面都没见过，但在其他人眼里，她已经是韩家的人，韩家被灭门后，她就是——寡妇。
<br/>　　而也有很多人，把韩家灭门的事情归责在苏樱身上，说是她克得韩家遭此厄运。
<br/>　　如此，苏樱就被当做不祥之人，苏家想把她赶紧扫地出门，家人仆役视她为灾星，不想沾染厄运。
<br/>　　这样，在很多人看来，苏樱能嫁给那年近半百的黄员外已经是高攀了，否则就算把她送给家丁，家丁也是不要的。
<br/>　　这些事情在程大雷听来瞠目结舌，匪夷所思，可对于这个时代的人来说，却是理所应当，再正常也没有。
<br/>　　“那你爹呢，他就不管嘛？”程大雷问。
<br/>　　“家父身染重病，一直卧床不起，这些事他怕是不知道的。”苏樱说着，轻叹了口气。
<br/>　　有些事苏樱没有对程大雷讲，这种事便是苏父知道又能如何，怕还是会说些『你不要给苏家丢人』『有气节的早就悬梁自尽了』之类的话。
<br/>　　程大雷无语，良久长叹口气，莫名其妙感慨一句。
<br/>　　“封建礼教害死人呦。”
<br/>　　苏樱正在暗自神伤，她不把伤心给人看，在家里所有人面前都做出一副平静模样。但这，并不代表她心底就不会伤心。
<br/>　　不怕所有人都这么看，其实苏樱心底也是如此认为的，身在这个时代中，她就跳不出这个时代的思维。
<br/>　　“不怕，他们都是混蛋，有我呢。”
<br/>　　苏樱下意识抬起头，看到程大雷拍着胸脯。
<br/>　　“我会保护你的。”
<br/>　　苏樱怔住很久，感觉自己鼻头有些发酸，她揉了揉鼻子，笑着问：“你打算怎么帮我。”
<br/>　　“一切包在我身上，你是我的压寨夫人，我怎么会让人欺负你。”
<br/>　　这个时代少男少女之间的感情都是很含蓄的，很少有人如程大雷这样直白坦率，或者说，像他这样不要脸。所以听到程大雷的话，苏樱脸皮有些发烫。
<br/>　　“你好好待在这里，我会替你解决掉麻烦，等我的消息。”
<br/>　　说罢，程大雷准备跳墙离开，看着他的背影，苏樱忽然道：“等等。”
<br/>　　“怎么了？”
<br/>　　苏樱递给苏樱一笼鸽子，道：“你带着它走。”
<br/>　　“这个……我知道你关心我，我还是很开心的。不过我已经吃过晚饭，而且烤乳鸽什么的，不是特别喜欢。”
<br/>　　苏樱哭笑不得：“让你用这个联系我。”
<br/>　　……
<br/>　　程大雷跳墙离开苏家，一路没被任何人发现。他连夜回到客栈，等待着徐神机的消息。
<br/>　　徐神机是第二天中午回来的，那个时候程大雷正在房间里午睡。
<br/>　　“大当家，大当家……”
<br/>　　程大雷迷迷糊糊醒来，看到徐神机立在自己面前，孟子云也在一边站着，面带困惑。
<br/>　　他瞬间清醒，从床上爬起来，问：“还有谁来了？”
<br/>　　林少羽从门外进来，微微拱手，立在程大雷身边。
<br/>　　紧跟着，秦蛮也进来，魁梧的身躯宛若一座黑铁塔，也同样立在程大雷身边。
<br/>　　好嘛，蛤蟆寨最强两个打手都来了，程大雷心中有了底气，可以说豪气顿生。
<br/>　　孟子云皱起眉头：“你就是程大雷？”
<br/>　　“是呐，怎么你有意见？”
<br/>　　“小子，弄虚作假，竟敢骗我。”
<br/>　　说着，孟子云一拳便向程大雷打来，拳势如虎。可程大雷连眼皮都没抬一下，这个孩子还以为自己是昨天呢，该让他明白，形势已经不同了，真如果揍你，都不用我动手。
<br/>　　程大雷没有动手，动手的是林少羽，他单掌拦住孟子云，身子往前靠，狠狠撞在孟子云身上。
<br/>　　双方硬碰硬，各自往后退了一步，平分秋色。
<br/>　　“小小山贼，却敢向我动手，找死！”孟子云握紧腰间佩刀。
<br/>　　林少羽不动如山，冲他招招手，做出一个『来』的姿势。
<br/>　　双方剑拔弩张，危机一触即发。这个时候程大雷走在二人中间，笑着冲孟子云道：“想动手随时可以，但现在是打架要紧，还是救人要紧？”
<br/>　　孟子云微微沉默，握刀的手松开。
<br/>　　程大雷笑笑，不掩饰自己的轻蔑。
<br/>　　“钱带来了没有？”程大雷问。
<br/>
