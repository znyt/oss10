”<br/>    <br/>    几个小时后，在篝火旁，卡西对一个女佣兵说道。<br/>    <br/>    这个女佣兵穿着游牧民族的服装，包着灰色的头巾，穿着短衫和长裙，外套一件皮质的马夹，脚下是一对做工粗糙的羊皮靴，用牛筋绑着。背上背着一把突击步枪，腰上挂着弯刀，靴子口露出半个刀柄。这是典型的沙民装束，无论行走荒野还是行走沙漠，都是一流的。<br/>    <br/>    她露在外面的皮肤，在火光下呈现出红铜色，也是典型的沙民肤色。相貌算不上好看，但也不难看，眼窝有些轻微凹陷，年龄在三十岁左右，有几分成熟女人的韵味。身材倒是不错，该大的地方大，该小的地方小。能出来当佣兵的女人，至少赘肉不多。<br/>    <br/>    “我好像闻到了马尿的馊味。”阿乔木捂着鼻子说道。<br/>    <br/>    “该死！”卡西翻了下白眼，咒骂道。<br/>    <br/>    “你最好小声音点。”阿乔木说道。<br/>    <br/>    “为什么？”卡西有些懵。<br/>    <br/>    “他来了。”阿乔木说着，朝侧面瞟了一眼。<br/>    <br/>    卡西朝侧面望去，忍不住嘀咕了一句，“该死！”<br/>    <br/>    陈兴走了过来，朝卡西晃了晃手中的扑克牌，“来玩玩？”那扑克牌，正是卡西刚才遗落的。<br/>    <br/>    “玩就玩。”卡西硬着口气说道。<br/>    <br/>    正要去接扑克牌，陈兴毫无征兆地扑向一边。紧接着，一声沉闷的枪响打破了夜晚的寂静。<br/>    <br/>    卡西下意识地转头看去，坐在篝火对面的男性佣兵耷拉着脑袋，鲜血和脑浆喷了一地。<br/>    <br/>    “谢特！”卡西大叫一声，朝暗处扑去。<br/>    <br/>    阿乔木则抓起旁边的水桶，朝篝火上一泼，翻身侧滚，融入黑暗之中。<br/>    <br/>    “袭击，有袭击！”“趴下，快趴下！”“把火都灭了！”听到枪声，营地里起了一阵骚乱。不过大部分人都是商团的老雇工，见过了场面，没有太过于慌乱。<br/>    <br/>    陈兴在黑暗中爬出十几米，靠在一辆卡车的轮子旁，卸下背后的布套，朝地上一摊，三两下组装好“猎象”狙击枪，“咔嚓”的一声，拉栓上弹，然后贴着轮子，朝朝土山的方向望去。<br/>    <br/>    “嘭！”<br/>    <br/>    土山上火光一闪，营地里倒下一个逃窜的工人。<br/>    <br/>    陈兴迅速打开狙击镜的盖子，瞄了过去。<br/>    <br/>    八倍镜的十字准星中，出现一个模糊的人影。陈兴凝神聚气，缠着布条的枪身逐渐点亮。“猎象”狙击枪主要是狩猎用的，没有隐蔽充能的设计，白天还没什么，一到晚上就必须包着厚布，不然简直是夜里点火把。<br/>    <br/>    陈兴暗念着，等去了乌鸦市，一定要换把专业点儿的。<br/>    <br/>    十来秒钟后，充能完毕，陈兴能感觉到枪膛内子弹充盈躁动的灵能。准星挪到人影的上方，猛地扣下了扳机。<br/>    <br/>    “嘭！”<br/>    <br/>    一米多长的蓝焰喷吐而出，一道蓝芒刺破夜空，瞄准镜中的人影应声而倒。<br/>    <br/>    一枪毙命，陈兴迅速转移位置。而这时，对面枪响了。身后火光一闪，热浪袭来。子弹击中了卡车的油箱，发生了爆炸。陈兴瞬间烧成了火团，不过他没有感觉到疼痛，反而皮肤凉凉的。在地上打了几个滚，火就灭了。<br/>    <br/>    对面的卡西看到这一幕，整个人都呆住了，“我的神啊……”<br/>    <br/>    <br/>
