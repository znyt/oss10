　　混乱虚空之中，一处空间裂缝之中，一个黑点若隐若现，散发着幽暗深邃的光芒，在这片灰色空间之中，毫不起眼。
<br/>　　但元始却是发现了它。
<br/>　　元始知道，这就是一个小世界。虽然它外表看着很小，但领悟空间的他知道，黑点之内空间很大。
<br/>　　来到近前，元始看着眼前这个时而发出幽暗光芒的黑点就是龙族发现的那个小世界了。
<br/>　　只是，如今要如何进去？
<br/>　　这是一个很值得深思的问题。
<br/>　　要是龙族，他们肯定有着不为人知的手段进去，要不然，他们又是如何进去小世界埋葬真龙的呢。一般人，就算发现了这个小世界，他也进不去，甚至拿不走，因为此处的虚空有些特殊。
<br/>　　小世界处在一个关键的空间节点之中，一旦将这个小世界拿走，这片虚空都会坍塌破碎，是个人都会葬身于此，更甚至小世界都会被毁灭。
<br/>　　要是之前的元始，他就算发现了此处，也会感到麻烦。当然，也仅仅是麻烦而已。
<br/>　　但，如今，这点小问题，对于他来说，已经不是多大的问题了。
<br/>　　领悟了空间法则的他，对于眼前这个世界，要进去很简单。
<br/>　　尽管他还只是领悟了空间法则的皮毛，但对于破开一个小世界的界壁，还是可以做到的。即便这个小世界有些特殊。
<br/>　　“破——”
<br/>　　淡淡的一道声音响起，黑点裂开一道细缝，元始化为一道金光钻了进去，之后，细缝慢慢弥合起来。
<br/>　　“嗯？没有灵气流动？”
<br/>　　刚从空间通道之中出来，元始还没有来得及察看四周环境，唯一的感受就是这空间之中并没有灵气波动，在这里修炼吸收不到丝毫灵气。
<br/>　　元始放眼看去，四周漆黑一片，黑暗深邃，死气纵横，没有一个生物。
<br/>　　这小世界并不大，只有方圆数百万里大小，元始神识瞬间就覆盖整个小世界。
<br/>　　在他的神识之中，能够看到数千个大大小小的坟墓，其中，最大的有五个，它们都在小世界最中心，四周都是一些较小的坟墓。
<br/>　　在空间之中，元始还能够感受到浓浓的龙气，金色龙气化为龙形，笼罩着每一座坟墓之上。威压天地，龙威镇世。要是一般金仙在此，都会被这些龙威所慑服。
<br/>　　这龙墓并不是什么龙族后裔死后都可以进入，这一定要龙族的嫡系成员，而前还是强大的嫡系成员死后，才能进入龙墓安葬。要不然，什么龙族都可以进来的话，那此处早就堆满了真龙坟包了。
<br/>　　越是龙族嫡系，其体内的龙气越是浓郁，实力也越是强大，所以元始才会选择进入龙墓修炼《玄黄不灭》第四重。
<br/>　　“嗯？”元始顿时发现了不一样的东西。
<br/>　　一步踏出，元始便来到世界最中心，此处，有一个巨大的池子，池子呈圆形，直径十丈大小。池子之内，布满混沌般的金色液体，龙气浓郁成液体，散发着浓郁的是香气。与周边暗色死气，极不和谐。
<br/>　　“化龙池！”
<br/>　　元始慢慢吐出一个名字。神色之中，有些震撼。同时，他心里也有些明白，龙族为何要将死后的真龙葬于此处了。
<br/>　　化龙池，顾名思义，就是为了化龙而准备的，龙族一般成员，若是能够入这化龙池一趟，不仅能够提高血脉纯度，还能增强实力。
<br/>　　就算不是龙族成员，一般人就算进入化龙池之中，也能够实力大增，肉体真身强度大增，堪比龙族肉体。
<br/>　　众所周知，龙族强大，不仅仅是因为他那得天独厚的天赋，还因为他们那强悍无比的肉体，在这洪荒之中，还真没有几个种族的肉身能够堪比龙族的。
<br/>　　现在这里有一个化龙池，那就意味着能够造出一大批真龙。
<br/>　　龙族的目的那就不言而喻了，他们将死去的真龙运来此处安葬，目的就是让这些真龙体内的龙气、龙珠、龙血进入化龙池之中，增强化龙池底蕴，为后世更多龙族子孙造福。
<br/>　　“化龙池，这是好东西，合该为我所用。”
<br/>　　元始大喜，自己真是要什么来什么啊，自己刚要大量龙气，此处就有比龙气更好的化龙池。
<br/>　　在查看四周没有然任何危险后，元始就不再犹豫，闪身进入化龙池之中修炼。
<br/>　　化龙池之中有一种极为浓郁的灵气，不但毫无杂质，而且蕴含着强大龙气，以及浓郁的天地规则。怪不得小世界中没有丝毫灵气，原来是都被龙族布阵引到了这化龙池之中。
<br/>　　在运转《玄黄不灭》之时，元始感受到一种从未有过的畅快。
<br/>　　无穷无尽的纯净规则龙气被元始卷走，而更多的龙灵之气再次疯狂涌来。形成一个巨大的灵气风暴。
<br/>　　坐于化龙池之中，元始感觉到自己的整个肉身都在渐渐淡去，存在的只有他的元神。这种精神上没有肉身的阻碍，修炼的吸收效果更是直线狂飙。
<br/>　　化龙池之中的纯净气息，在不断地冲刷着元始的肉体，他那本来就没有什么杂质的身体更是被冲刷的一尘不染。
<br/>　　修为气息在不断的上涨，再上涨……
<br/>　　汩汩神元犹如奔腾长江在元始的周围环绕，一滴滴金色的真龙之血也在这时被元始取了出来吞下，真龙精血还来不及化成龙形，就被元始吞入腹中。
<br/>　　真龙精血配合浓郁的真龙龙气，不断地冲刷着元始的身体，体内不断的咔咔声传来，元始差点叫出声来，这种修炼除了痛快还是痛快。
<br/>　　时间在流逝，元始连自己的肉身都感受不到，更是感受不到时间的流逝。
<br/>　　也不知道过了多久，元始突兀的感受到自己的骨骼开始重组，骨骼破碎，从脊骨开始，再到肋骨，手骨，脚骨，最后甚至连头骨都尽皆破碎，整个人都瘫软在地。
<br/>　　元始还是沉醉在那种畅快的修炼之中，不可自拔。要是此时有人偷袭于他，元始没有一点防备。
<br/>　　金色龙血以及金色龙气不断地冲刷着他的肉体，更多的是融入到他的骨骼之中。
<br/>　　《玄黄不灭》第四重龙脉融龙骨自行运转起来，开始不断修正，沿着元始收集到的龙族功法，再结合他自己的推演，不断地修复……
<br/>　　骨骼破碎又重组，重组又破碎，不断循环下去……
<br/>(本章完)
