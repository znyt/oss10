        不再有过多的话语，傅夜沉拍完明司寒的肩膀，便只身离开。
<br/>
<br/>        明司寒看着镜子里傅夜沉离去的背影，心里一片茫然。
<br/>
<br/>        是不是，下次再见面时，彼此就真的站在了敌对的两端。
<br/>
<br/>        连星儿回国后，安全抵达家里的第一件事情，就是给明司寒打电话。
<br/>
<br/>        这电话倒是打通了，只是无人接听，让连星儿感到纳闷。
<br/>
<br/>        或许是时差的问题？
<br/>
<br/>        连星儿悻悻地放下手机，先去洗了个澡，然后下楼去吃饭。
<br/>
<br/>        当她走下楼时，童筱颖正拿着手机，在客厅里一边徘徊，一边打电话。
<br/>
<br/>        “那你这次又要在外待多久？”童筱颖发出不容置喙地质问。
<br/>
<br/>        “……”
<br/>
<br/>        “你最近回来的次数越来越少了。”
<br/>
<br/>        “……”
<br/>
<br/>        “我只是担心你。”
<br/>
<br/>        “……”
<br/>
<br/>        “我需要的是你的陪伴！”童筱颖恼火地说完后，挂了电话，直接将手机扔到了一旁的沙发上，似乎是气不打一处来地一手叉腰，一手抚着额头很是头疼的样子。
<br/>
<br/>        连星儿不知道爹地是怎么惹妈咪生气了，见此情形，她悄悄地从楼梯旁的花瓶里摘下一朵粉玫瑰藏在身后，立马迎上前去，嬉皮笑脸地哄童筱颖开心：“妈咪，生气的话，会长皱纹的噢！”
<br/>
<br/>        童筱颖瞥了连星儿一眼，然后往身旁的沙发上坐去，双手心力憔悴地抵着额头。
<br/>
<br/>        “妈咪，我替爹地陪着你。”连星儿随即在童筱颖的身旁坐下，伸手挽住童筱颖的臂弯，将小脑袋凑了过去，“妈咪若是生气了，不开心了，爹地也会很难过。”
<br/>
<br/>        “……”童筱颖没吭声，放下手来，抬头看着连星儿时，心里五味成杂般的难受。
<br/>
<br/>        连星儿嘴角微扬道：“我给妈咪变个魔术，怎么样？”
<br/>
<br/>        “嗯？”童筱颖心不在焉地应了声。
<br/>
<br/>        也正是因为她的心不在焉，连星儿得以在她面前乱挥双手。
<br/>
<br/>        “锵锵锵！变！”连星儿捏住刚刚那朵粉玫瑰，举到了童筱颖的眼前，“粉玫瑰送给妈咪！”
<br/>
<br/>        童筱颖忍俊不禁地接过连星儿手中的粉玫瑰，欣慰地抱了抱这个讨人欢心的小棉袄。
<br/>
<br/>        明明眼前是如此的美好，可下一秒，她恍若产生了一种时空交叉的错觉，一眼看穿了这丫头的未来。
<br/>
<br/>        如果没有这个女儿，她的老公就不会那么累。
<br/>
<br/>        最近，童筱颖隐隐感觉到有危险的气息在萦绕，刘婶却总是在劝她，是她多心了。
<br/>
<br/>        可是傅夜沉不经常回家，就代表不是她多心。
<br/>
<br/>        吃过晚饭后，连星儿又给明司寒打了一通电话，仍旧是无人接听。
<br/>
<br/>        无奈之下，她只能给明司寒留言。留完言后，她捧着手机，翘着二郎腿，躺床上打起了游戏。
<br/>
<br/>        有个时候，手机未拿稳，直接砸到了她的脸上，让她好一阵的生痛。
<br/>
<br/>        连星儿一直在等明司寒的回讯，等着等着，游戏还未打完，她本人就先睡了过去。
<br/>
<br/>        童筱颖推门进来时，拿起连星儿的手机看了看，只见正在进行的游戏里，连星儿的队友纷纷在说，要举报连星儿所操作的英雄有挂机行为。
<br/>
<br/>        替连星儿将手机关机后，童筱颖拉起一旁的薄毯，准备给连星儿盖上时，脑海里突然闪过一个从未有过的念头。
<br/>
<br/>        无论国内还是国外，很多暗中组织，都想得到星儿，拿星儿去做实验研究。不如，她干脆把星儿交给父亲连中凯。
<br/>
<br/>        连中凯其实也想拿星儿做研究，如果不是碍于她这个女儿和傅夜沉的势力，只怕连中凯早就对星儿下手了。
<br/>
<br/>        既然，注定不能平凡地度过一生，不如……
<br/>
<br/>        想到这里，童筱颖的手，隐隐地开始颤抖。
<br/>
<br/>        是不是把星儿交给连中凯，大哥就能不用再那么累？
<br/>
<br/>        童筱颖重新拿起连星儿的手机，找到了父亲连中凯的联系方式。
<br/>
<br/>        就在她准备拨过去的时候，手中的手机，突然被一只手给夺走。
<br/>
<br/>        童筱颖惊怔地转过身去。
<br/>
<br/>        只见一个脸上戴着半边银色恶魔面具的少年，身着一袭特制的黑色异服，手里拿着从她手中夺过的手机，面无表情地出现在了她的面前。
<br/>
<br/>        “你是谁？”童筱颖微微蹙起眉心，毫不畏惧地质问道。
<br/>
<br/>        少年揭下脸上的半边银色恶魔面具，两眼闪过一抹金色的瞳光。
<br/>
<br/>        童筱颖身心一滞，立马想起了什么，接着问道：“你……是……郎安？”
<br/>
<br/>        郎安垂下眼帘，瞳仁恢复了正常的颜色，毕恭毕敬地向童筱颖欠了欠身：“是傅先生，让我过来的。”
<br/>
<br/>        外人要想进私宅并不容易，除非得到了傅夜沉特许。
