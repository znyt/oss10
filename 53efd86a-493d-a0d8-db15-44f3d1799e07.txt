        落以柔吸了吸鼻子，抬眸看了童筱颖一眼，微微点了点头。
<br/>
<br/>        童筱颖不再问话，而是转身离去。
<br/>
<br/>        落以柔看着童筱颖离去的背影，在心底为傅夜沉感到不值。
<br/>
<br/>        瞧瞧她童筱颖这个女人是有多薄情！
<br/>
<br/>        她也想像她一样薄情，但是她做不到。
<br/>
<br/>        童筱颖在临海城市中心找了一家酒店住下，在酒店的单间里，她拿着笔和纸，匍匐在桌前写着什么。
<br/>
<br/>        落以柔说，车子被人动过手脚。
<br/>
<br/>        很显然，有人就是想要大哥的命。
<br/>
<br/>        是八爷，还是那个十爷？
<br/>
<br/>        夜惑的幕后，到底是何人？
<br/>
<br/>        此时此刻，比起悲痛，童筱颖心里想得更多的是报仇。
<br/>
<br/>        她明明知道，那个八爷和十爷跟大哥傅夜沉是死对头，大哥傅夜沉也正在查他们，但是她却为了季雅珊，对大哥隐瞒了他们的事。
<br/>
<br/>        如果，当初她跟大哥坦白，大哥是不是就不会死？
<br/>
<br/>        一天一夜后，傅昼景找了过来。
<br/>
<br/>        而童筱颖正打算离开临海城。
<br/>
<br/>        “明天大哥的灵柩上山，你不想去看看大哥吗？”傅昼景见童筱颖在收拾东西，深沉地问道。
<br/>
<br/>        童筱颖瞥了傅昼景一眼，淡漠地回答道：“他都不是为我而死，我又何必去墓园看他？”
<br/>
<br/>        从她得知傅夜沉的死讯后，到现在傅昼景就没见她童筱颖笑过，他甚至猜不透她的心思。
<br/>
<br/>        “也对！昨天晚上，是大哥的头七。那个女秘书，吞安眠药自杀了。”傅昼景感慨道。
<br/>
<br/>        童筱颖正整理手提包的手滞了一下，默不吭声。
<br/>
<br/>        傅昼景接着说道：“好在我们发现得早，她被她师父及时送去了医院洗胃，现在已经苏醒了。”
<br/>
<br/>        “这跟我有什么关系？”童筱颖拉上手提包上的拉链，抬眸看着傅昼景，冷冷地问。
<br/>
<br/>        傅昼景抿了抿唇，点头道：“对！跟你没关系！那么，我们来谈谈跟你有关系的事。”
<br/>
<br/>        “有话快说。”童筱颖索性在大床边的单人沙发里坐了下来，听他傅昼景把话说完。
<br/>
<br/>        傅昼景有些气恼道：“筱颍，大哥他真的一直在防着你。大哥的意外身亡，傅氏集团除了周转资金以外，全部被封禁在银行里了。”
<br/>
<br/>        “你还惦记着大哥的公司啊？”童筱颖忍不住嗤之以鼻。
<br/>
<br/>        傅昼景在童筱颖对面的大床边坐下，抬起手来比划道：“傅氏集图在大哥的操作下，现在有一套独立的运作系统。现在，傅氏集团旗下各子公司，都有总经理，总部有一个姓裴的掌管，不管是子公司还是总部，都有不同的第三方和第四方财审公司监管。除了周转资金，所有的盈利全都继续往大哥的私人账户里进，除了大哥，谁也无法将那笔资金取出来。筱颍，就连你作为大哥的发妻，作为第一继承人，却连那些遗产的继承权也没有。你不觉得，大哥对你很过分吗？”
<br/>
<br/>        “其实，你是觉得，大哥对你很过分吧？”童筱颖怀抱双臂，不以为然地反驳。
<br/>
<br/>        傅家的人，一定以为，傅夜沉死了以后，他们都有资格继承或者是瓜分傅氏集团。
<br/>
<br/>        可是，谁也没料到，傅夜沉会把自己的集团运作到一个几近完美循环的状态。
<br/>
<br/>        简而言之，那就是，傅氏集团就算没有了傅夜沉，依旧可以继续运作下去，不会出现群龙无首而慌乱的状态。
<br/>
<br/>        傅昼景淡然道：“大哥对我，早就因为三年前的那件事情心生芥蒂。所以，我也早就对傅氏集团不感兴趣了。只是，我的那些堂叔，倒是对傅氏集团虎视眈眈。”
<br/>
<br/>        “昼景。”童筱颖放下手，从单人沙发上起身，凝望着傅昼景，深沉道，“你好好跟季思妍带着小尚轩一起过日子。不要再对我纠缠不清了。就算没有大哥，我和你也已经不可能了。此外，苏漫雪不是一个好女人。你能撇开她，最好撇开她。”
<br/>
<br/>        “你这算是在跟我作告别？”傅昼景凝了凝眉，“你要去哪？”
<br/>
<br/>        “想出去旅游散心。”童筱颖脸不红心不跳地撒谎。
<br/>
<br/>        大哥死了，他对她，确实没有什么可以纠缠在一起的必要了。
<br/>
<br/>        “筱颍，我还爱着你。”傅昼景站起身来，想要伸手去拉童筱颖的手。
<br/>
<br/>        但在半空中，她微微侧身，躲开了他的亲密。
<br/>
<br/>        “我已经不爱你了。”童筱颖不假思索地回答。
<br/>
<br/>        “你今天就要离开吗？以后我们……”
<br/>
<br/>        “以后，你不要再来找我。我也不会主动联系你。就这样。”童筱颖直接打断了傅昼景的话，提起自己的手提包，只身离开了房间。
