　　此刻，已经不止米国、华夏和岛国，在密切关注着叶尘的一举一动，其余各大国同样纷纷调动卫星，对准米国的第九区军事基地。
<br/>　　“原来叶狂仙的目标，是米国的超能局！”
<br/>　　“这下米国政府肯定不会坐视不理了吧！”
<br/>　　“好戏终于要开始了吗？”
<br/>　　……
<br/>　　除了华夏和米国两方，其余各大国的高层，完全都是抱着看热闹的心态。
<br/>　　正所谓，两虎相争，必有一伤，无论哪一方出事，对于他们来说，都不失为一件好事。
<br/>　　当然了，最好的结果，自然是两败俱伤，甚至米国和华夏，也因此开战，那就更好不过了。
<br/>　　“叶狂仙！这一年来，我们超能局十二主神，几乎全部死在你的手上，你还想怎样？”
<br/>　　叶尘杀入军事基地后，忽然下方传来一声暴喝，赫然正是来自超能局局长波尔。
<br/>　　叶尘冷冷一笑，
<br/>　　“既然选择招惹了我，那就应该做好付出代价的准备！”
<br/>　　说完这话，叶尘又随手一剑斩出，前来支援的数十辆装甲车，竟然被叶尘这随手一剑砍翻！
<br/>　　“叶狂仙！我再给你最后一次机会，就此罢手，今后我们互不侵犯，否则后果自负！”
<br/>　　波尔再次发出警告道。
<br/>　　叶尘却理都不理，直接提剑便朝波尔的方向杀了过去。
<br/>　　波尔见状，不由得两眼一凝，也不躲闪倒退，脸上反而露出一抹决然的笑容，
<br/>　　“用我的性命，来换取联邦百年安宁，我也知足了！”
<br/>　　说完这话的同时，波尔猛然按动手中的一个遥空装置。
<br/>　　“嗯？”
<br/>　　叶尘此时离波尔也只有百米左右的距离，他目力惊人，瞬间便察觉到了波尔的举动，而且紧接着感应到一股前所未有的危机感。
<br/>　　“不好！原来是个陷阱！”
<br/>　　叶尘反应过来之后，硬生生止住了脚步，然后朝后方迅速暴退。
<br/>　　而就在这时，
<br/>　　彭！
<br/>　　大地猛然震动，犹如发生了十级地震！
<br/>　　继而，
<br/>　　轰隆隆！！！
<br/>　　一道硕大的蘑菇云，冲天而起，璀璨的强光，瞬间将方圆百里之内，都完全笼罩在了其中！
<br/>　　看到这一幕，除了米国一方事先有了心理准备之外，其余诸国的高层，顿时纷纷从座位上站了起来，两眼死死的盯着屏幕，满脸全是惊骇之色。
<br/>　　叶天歌更是脸色惨白，不由得一连倒退数步，险些晕厥过去。
<br/>　　“核弹！至少也是百万吨级的核弹！”
<br/>　　各国高层纷纷惊呼起来。
<br/>　　要知道，当年米国在岛国投放的那两颗原子弹，也不过才只相当于一万多吨TNT当量，就已经摧毁了岛国两座城池。
<br/>　　而眼下这个百万吨级的核弹，则是当年的上百倍！
<br/>　　这种级别的核弹，足以让方圆百里之内，瞬间寸草不生，化成不毛之地，而辐射污染更是能持续数年乃至数十年。
<br/>　　“米国果然是好魄力！竟然不惜舍弃超能局，也要将叶狂仙置于死地！”
<br/>　　“话不能这么说，叶狂仙的目标本来就是超能局，既然明知道保不住，还不如同归于尽！”
<br/>　　“百万吨级的核弹都已经动用，这下那叶狂仙总该死了吧？”
<br/>　　……
<br/>　　各国高层望着屏幕上，那道久久不息的惊天蘑菇云，纷纷议论开来。
<br/>　　过了许久，蘑菇云总算化成尘埃，开始缓缓消散。
<br/>　　只见，以超能局为中心，出现了一个硕大的深坑，而周围方圆百里之内，早已经成为一片废墟，被一层厚厚的尘埃所覆盖，根本看不到任何生命的气息。
<br/>　　“这下那个家伙总该死了吧？”
<br/>　　“想我联邦超能局，自一战之时成立，至今已经一百多年，为联邦的军事发展，做出了无比卓越的贡献，如今却全部毁于一旦了！”
<br/>　　“不过，牺牲一个超能局，能将那个恶魔除掉，倒也划算！”
<br/>　　“这就是和我们米国作对的下场！”
<br/>　　……
<br/>　　米国国会的众议员，唏嘘感叹了一阵之后，脸上纷纷露出一抹轻松和欣慰之色。
<br/>　　“这恶魔已死，我米国依然是世界第一强国！”
<br/>　　“至于这笔账，我们自然要从华夏身上讨还回来！”
<br/>　　“不过，这一次我军损失惨重，眼下还是不能轻举妄动，必须要休养一段时间才行！”
<br/>　　……
<br/>　　就在米国国会的众议员们，已经开始商议接下来的战略计划之时，忽然有人指着大屏幕，两眼瞪得老大，露出满脸不可思议的神色，
<br/>　　“快看！那……那是什么！”
<br/>　　而与此同时，
<br/>　　就在所有人都以为，叶狂仙已经死在这枚百万吨级核弹的轰炸之下，接下来却看到了无比震惊的一幕。
<br/>　　只见，
<br/>　　那片被核弹轰炸的废墟之中，
<br/>　　“彭！”
<br/>　　一道紫色金芒，忽然冲天而起！
<br/>　　这一刻，仿佛连同整个北美大陆，都狠狠晃动了一下，所有人看的目瞪口呆。
<br/>　　而待那周围的尘埃散去，那紫色金芒之中，赫然是一道瘦弱的少年身影，不是叶尘又是哪个？
<br/>　　“太好了！小叶将军还活着！他没死！”
<br/>　　华夏一方的众将，看到这振奋人心的一幕，几乎所有人都高兴的跳了起来，个个一脸激动之色。
<br/>　　不知道从什么时候起，叶尘早已经成为了他们心中的神灵，之前虽然看到叶尘被核弹淹没，虽然理智告诉他们，没有人可以在百万吨级核弹的轰炸下存活，可是心中或多或少，还是心存着一丝希望。
<br/>　　如今看到叶尘果然还活着，激动的心情可想而知！
<br/>　　而米国国会一方的反应，则是截然相反，
<br/>　　他们牺牲了超能局，乃至整个第九区军事基地，在本土动用了百万吨级的核弹，竟然依旧没能把这头恶魔杀死！
<br/>　　“这……这不可能！”
<br/>　　“他怎么可能会没死！”
<br/>　　“我不相信！”
<br/>　　……
<br/>　　平日高高在上的国会议员们，如今个个惊慌失措，不甘的大吼大叫起来。
<br/>　　至于其他诸国的高层们，看到这惊天一幕，则纷纷陷入了沉默之中，
<br/>　　“看来核弹对此人，的确已经构不成威胁，难道他真是神灵下凡不成？”
<br/>　　“连百万吨级的核弹，都奈何不了此人，这岂不是代表着，他已经具备叫板核武大国的资本？”
<br/>　　“此人今日不死，日后恐怕就要凌驾于诸国之上了！”
