　　第3章
<br/>　　“灵儿，快关寨门……”徐神机神速跑回山寨。
<br/>　　“爹，你怎么一个人回来了，小寨主呢？”
<br/>　　“唉，那个愣头青已经死了，要不是我足智多谋，神机妙算，最重要是跑得快，不然你就见不到爹了。”
<br/>　　徐神机说着忽然一拍脑袋，脸上露出痛苦表情。
<br/>　　“爹，人死不能复生，你也不要太难过。”徐灵儿轻叹口气。
<br/>　　“难过，我难过什么，我难过的是那匹马呦，可惜了，唉……”徐神机来了精神：“山寨不能再待下去，你看有什么东西，都带上，我记得厨房还有点大豆。”
<br/>　　……
<br/>　　纵马狂奔，程大雷满头热汗，身后林少羽一帮人紧追不舍，距离越来越近，若被他们追上，估计自己只有死路一条。
<br/>　　嘟，你获得抽奖机会1，是否现在抽取。
<br/>　　“抽，抽，抽……”程大雷鞭子抽在马屁股上。
<br/>　　嘟，充满干劲的少年喔，你是要抽马还是要抽奖？
<br/>　　“我抽你大爷。”程大雷气得爆炸，这系统是要玩死我吗，都现在了，能不能干脆痛快些：“我要抽奖。”
<br/>　　嘟，开始抽奖。
<br/>　　脑海中出现一扭蛋机状的物事，扭蛋翻滚，程大雷心急如焚，感觉过去的每一秒都无比漫长。
<br/>　　回过头，林少阳追得越来越近，眼睛泛红，只要被他追上，不用怀疑肯定会一枪给自己来个透心凉。
<br/>　　“系统大大，您能不能稍微快点，我时间没多少呃。”
<br/>　　终于，一颗扭蛋从扭蛋机中掉下来，扭蛋缓缓打开，光华一闪之后消失。
<br/>　　嘟，恭喜你获得默默无闻的优秀猎人秦蛮。
<br/>　　咦，这抽奖原来还可以抽到活人。
<br/>　　只是人在哪里，周围光秃秃的，一个人影都没有。难道是从天而降，还是说自己默念一番咒语，然后从天而降……
<br/>　　“去吧，皮卡丘！”
<br/>　　“以小樱的名义召唤你”
<br/>　　“全体守望先锋，全体守望先锋……你们来不来！”
<br/>　　“聚集的希望引向新的地平线，成为光芒闪耀的道路吧！同调召唤，王道战士！”
<br/>　　“巴啦啦能量，变……”
<br/>　　身后已经能听见重重的马蹄声，林少阳咬牙切齿，双目赤红，手中铁枪前举，距离马车只剩下一个马身的距离。
<br/>　　“大当家不必惊慌，秦蛮在此，料也无妨。”
<br/>　　前方山道之上，站着一九尺壮汉，方口阔目，手中拎一根大棍，大有一夫当关万夫莫开之势。
<br/>　　原来是这样来的，程大雷顾不得多想，驱马车从秦蛮身边饶过。
<br/>　　林少阳骑马冲来，眼看已经要撞在秦蛮身上。
<br/>　　“快躲开，我勒不住马！”
<br/>　　“下马！”
<br/>　　秦蛮一声闷吼，手中大棍砸在马头之上，奔马昂起前蹄，秦蛮微微侧身，伸手抓住马缰绳，脸上肌肉拧起，呈现如岩石般的线条。
<br/>　　“俺叫你下马。”
<br/>　　力从地起，腰马合一，轰隆一声，奔马硬生生被秦蛮掀翻在地。
<br/>　　程大雷倒抽一口凉气，力挽奔马，此人好大力气。
<br/>　　黄三元带人赶到，也被这一幕镇住，他大着胆子道：“呔，我们是黑石城城主的人，不想死的，赶紧把人放了。否则大兵发来，将这里碾成平地。”
<br/>　　秦蛮大棍挑起一块石子，嗖地一声，砸在黄三元嘴上，敲下一颗门牙。
<br/>　　“滚！”秦蛮惜字如金。
<br/>　　一群人心惊胆战，终于惧了秦蛮，默默离开。
<br/>　　秦蛮拖着大棍来到程大雷身边，双手一抱拳：“大当家，秦蛮特来投靠。”
<br/>　　程大雷集中注意力在秦蛮身上，脑海中多了一道信息。
<br/>　　姓名：秦蛮（默默无闻的优秀猎人）
<br/>　　年龄：31
<br/>　　技能：梅花枪（三蕊）
<br/>　　隐藏属性：无
<br/>　　生活在黑石城附近的猎人，因被黑石城通缉，来蛤蟆寨投奔你。
<br/>　　程大雷忽然明白，系统不能让奖励凭空出现，总会为它们安排合理的出场方式。
<br/>　　秦蛮本是黑石城附近的猎户，与妻子生活在一起。只因那黑石城主听闻秦妻美貌，趁着秦蛮外出派人来捉，秦妻被逼的跳崖，等秦蛮回来看到妻子尸体，一怒之下杀了几个护卫，成了黑石城的通缉对象，所以才过来投奔蛤蟆寨。
<br/>　　看着铁打般的男儿，脸上露出痛苦的表情，程大雷不忍再追问下去。对于自己，这可能只是人物背景，但却是秦蛮的亲身经历。程大雷轻轻叹口气，想要在这个世道活下去……难喔。
<br/>　　正这个时候，那匹被程大雷抛下的瘦马，竟然颠颠的自己跑过来。
<br/>　　“秦大哥，上马，回去。”
<br/>　　……
<br/>　　“灵儿，把那口锅背上，看还有什么能拿的。”
<br/>　　“爹，我背不动了。”
<br/>　　“你以为爹就背的动，可没这些，咱们下山怎么活。”徐神机呵斥自己女儿。
<br/>　　“要不，我给你们一匹马？”程大雷刚回山寨，就看到这一幕：徐灵儿背着一口铁锅，脖子上挂着蒜头，徐神机更是夸张，连那张太师椅都没有放过。
<br/>　　“大当家……你没死！”徐神机吃了一惊，然后哽咽道：“大当家没死，我就放心了，我知道大当家怪我没用，大当家不用赶，我自己走。”
<br/>　　“走吧，走吧。”程大雷挥挥手：“不过我报名时我说的是徐神机，他们正在山下找这个人呢。”
<br/>　　“你！”徐神机瞬间睁大眼睛，深吸一口气道：“老朽虽老，臭骨头还有几根，他们敢和大当家为难，就把我这老骨头碾碎了。”
<br/>　　程大雷白他一眼，把马车的缰绳丢给他。
<br/>　　“这是秦蛮，你们自己认识一下，灵儿喂马，秦大哥做饭，军师把马车里的人关进我的房间。”程大雷深吸口气：“今天晚上，本大王要洞房。”
<br/>　　各人安排下任务，程大雷自己检查着系统界面，刚才又完成一个任务，跑得太急，奖励还没看呢。
<br/>　　山寨：蛤蟆寨（破败的初级山寨）
<br/>　　人口：4
<br/>　　技能：无
<br/>　　嘟，完成抢夺压寨夫人初级阶段，奖励山寨随机一人觉醒隐藏属性。
<br/>　　是一个属性值的奖励，山寨目前一共有四个人，不知道这奖励会落在谁身上。
<br/>　　“大当家，大事不好，大事不好啦。”
<br/>　　徐神机急匆匆跑过来，程大雷下意识集中注意力在他身上，整个人忽然一楞。
<br/>　　觉醒隐藏属性的竟然是你！
<br/>　　
<br/>(本章完)
