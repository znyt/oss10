　　看着众人心思各异的走出大殿之后，元始便不再去管其他人，而是将心思放在了诸天万我这一件事上。
<br/>　　从伏羲这一件事上，他看到了另一条修炼之路，不过，是否可行，终究还是需要实验一番。至于怎么实验，伏羲不是穿越者吗，那方世界既然有伏羲，那么，想必也会有元始吧。
<br/>　　想要找到那方世界，对于别人来说，可能有苦难，那样无异于大海捞针，但对于元始来说，只要他们之间存在因果联系，他便可以通过因果来定位，直接无视时间与空间，将那方世界找了出来。
<br/>　　元始前身身为因果魔神，执掌大道因果法则，随着修为的提升，因果法则也已经解封了大部分，如今的“超我”之身原始，更是执掌天道因果，想要探查洪荒世界中一个人的因果，还是很简单的。
<br/>　　就算那人是混元强者，也休想阻拦，因为越是强大的修士，就越是重视自身因果。
<br/>　　元始利用原始之身，调动天道因果法则，查看伏羲身上的因果之力，自身更是调动大道因果法则，配合着天道因果法则，两者一同调用，对着目标伏羲。
<br/>　　嗡！
<br/>　　法则波动，瞬间便覆盖洪荒，因果法则之网，交织着扩散到整片空间之中，如天罗地网，让人无处可逃。
<br/>　　元始没有去管其他人，而是直接将注意力放到伏羲身上。他仔细查看，果然在伏羲身上发现很多因果之线，其中，大部分因果之力在人族，占据了其中的九成，当然，也包括他的亲人之类的。这也十分容易理解，毕竟他如今是人道天皇，与人族气运相连，那是再正常不过的了。
<br/>　　之后，便是女娲的那份因果最大，伏羲前身为女娲的哥哥，就算如今融合了，也改变不了结果，再说，女娲在伏羲证道这件事上，也出过不少力，有因果牵连，也没有出乎元始的意料。
<br/>　　接下来，便是属于通天的那份因果了，通天为伏羲的师祖，因果相连也是可以理解。
<br/>　　元始不停，继续往下看去，想要发现伏羲身上另一界的因果。
<br/>　　“嗯？两台伏羲琴？”
<br/>　　这一探查，元始发现伏羲竟有两把伏羲琴，这让他吃惊的同时，也有些理解了，毕竟这个伏羲终究是两个世界伏羲的融合体。
<br/>　　不过，有两把伏羲琴还是让元始吃惊不已，难道另一个世界的伏羲琴也随着他穿越过来了？这怎么可能？一把琴竟然能够承受得住穿越时的时空之力的冲刷？
<br/>　　元始心念一动，顿时就知道了这把琴的不凡之处。他仔细一看，知道这把琴原先并不叫伏羲琴，而是名为“七绝琴”，能够断七情、绝六欲，为一把魔琴。而且这把琴的质量也超乎元始的想象，这绝对是先天至宝级别的灵宝，而且还是一把证道道器，承载着一个绝世高手的道。
<br/>　　这绝对不是伏羲的琴！
<br/>　　这是元始的第一想法，不管是之前的伏羲，还是洪荒伏羲，都没有能力能够创造出这样一把道器魔琴，不是元始小看他们，这是事实。没有混元后期以上的修为，绝对不能创造出这样一把魔琴，而且伏羲也不会七绝之道。
<br/>　　那么，问题来了，它既不是异界伏羲的，也不是洪荒伏羲的，那这台魔琴又会是谁的呢？
<br/>　　微微向深处一想，元始顿时感觉到一股神秘的阴谋气息在里面。
<br/>　　这七绝魔琴的真正主人，绝对超乎想象，他不会是洪荒之人，也不会是伏羲那方世界的人，而是来自一个绝对不弱于洪荒的世界。
<br/>　　元始微微一想，便能知道，这琴的主人实力绝不会弱于自己，因为，这把琴竟能够隔绝自己的因果探查。
<br/>　　元始是谁？他的前身可是因果之主啊，最后更是接触到了天道境的因果大道法则，虽说如今他还没有完全恢复自己的全部实力，但如今也有混元九重天的因果大道之力了，再加上自己如今快混元八重天的元始大道，竟然不能够查看这琴真正主人的因果，这就有些恐怖了。
<br/>　　对此，有三种情况，要么就是其主人已经死了，要么就是这七绝琴的主人实力比元始高深，要么就是他有什么隔绝因果的宝物。对此，元始更相信最后一种判断。
<br/>　　那么，这琴又是如何到伏羲手中的？元始更远相信这不是偶然，而是有所预谋。
<br/>　　不再去多想其他，元始继续探查伏羲的因果，这琴的主人有什么预谋管他什么事，只要不招惹到他，他才懒得去管。若真是招惹到他头上，他才管他是什么人，他绝对会让他后悔。
<br/>　　另一台琴，也并没有出乎元始的意料，他的确是洪荒伏羲的伴生灵宝伏羲琴。元始并没有过多关注，而是将更多的注意放在伏羲本人身上。他是异界穿越而来之人，身上或多或少的会留下点滴与异界的因果之力，这种因果虽然极为隐晦，但却也极难消除。
<br/>　　“没有？怎么会没有？”
<br/>　　通过一番探查，元始并没有找到那段因果，这让他很是诧异，按理来说，这段因果不会消除，可这是怎么回事。
<br/>　　元始不信，继续搜寻，比耐心，元始最不缺。经过仔细搜寻，还真让元始发现了一些端倪，这段本身就因果极为隐晦，而且还被伏羲用大神通给遮掩了，若不是元始本身就精通因果法则，他还真让伏羲给骗过去了。

<br/>　　“哼！区区掩天手段，圣人也可能被你骗过，但本座乃是因果之主，若是仅凭这点手段，就想骗过本座，休想！”
<br/>　　元始一声冷哼，表示不屑，他手捏印诀，口中低喝一声：
<br/>　　“天地无踪，因果无痕，
<br/>　　天网恢恢，疏而不漏，
<br/>　　诸果之因，万界追踪！”
