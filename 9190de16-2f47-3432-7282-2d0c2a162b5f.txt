        童筱颖连忙举起自己的左手，示意夏凌焰看自己左手中指上戴的这枚粉钻戒指，解释道：“这枚戒指，我摘不下来！所以，你得冒充我的二婚老公，跟我配合一出戏，赶走我的前夫！”
<br/>
<br/>        “门外来访人是你前夫？我去！他是怎么找上门来的？”夏凌焰一脸惊悚。
<br/>
<br/>        “这事，我怎么知道？”童筱颖也很纳闷。
<br/>
<br/>        夏凌焰随即指了指卧室，接着说道：“你去卧室里躲躲，我去应付你前夫。”
<br/>
<br/>        “你能行吗？”童筱颖狐疑的看着夏凌焰。
<br/>
<br/>        夏凌焰随即向童筱颖眨了眨右眼，咧嘴道：“别小看我！”
<br/>
<br/>        童筱颖这才自顾自地往卧室里走去，顺便还关上了卧室门。
<br/>
<br/>        夏凌焰理了理自己身上的着装，抬起手来又撩了撩刘海，才从容不迫地走过去打开房门。
<br/>
<br/>        “先生，请问你找谁？”夏凌焰露出了一个非常标准而友好的微笑。
<br/>
<br/>        而出现在他的视野里的这个男人，虽是西装革履，却未打领带，衣领口的两粒纽扣未扣，衣领松垮显得有些随意，又或者说是他无心去打理。
<br/>
<br/>        男人脸上十分憔悴，眼窝很深，下巴上还有些新生的胡渣，整个人看上去很没精神，但不减他英俊的姿色。
<br/>
<br/>        这就是筱颖的前夫吗？
<br/>
<br/>        夏凌焰上上下下地将这男人打量了一番后，忍不住在心里调侃。
<br/>
<br/>        长得也就一般般啦！
<br/>
<br/>        “我找童筱颖。”傅昼景看到夏凌焰的同时，也将夏凌焰打量了一番，而且瞬间对夏凌焰产生了一种敌意。
<br/>
<br/>        “你是谁啊？找我老婆做什么？”夏凌焰一脸不耐烦地说。
<br/>
<br/>        傅昼景二话不说，直接推开夏凌焰，强行闯入了房间内。
<br/>
<br/>        “筱颍！筱颍！筱颍，你出来！我们俩谈谈！”傅昼景扯着嗓子大喊道。
<br/>
<br/>        隔着一张门板，童筱颖背靠着门，柳眉微蹙，咬着下唇不吭声。
<br/>
<br/>        “筱颍，我找了你一天一夜！你不要再折磨我了，好不好？”傅昼景四下看了看后，目光落在了那张紧闭的卧室门上。
<br/>
<br/>        当他想要走过去压下手柄打开房门时，夏凌焰连忙踱步上前，拦住了傅昼景的去路。
<br/>
<br/>        “喂！我跟我老婆在度蜜月，你这个外人闯进来大喊大叫，是几个意思啊？再不滚出去，我可要喊酒店保安了！”夏凌焰扬起下巴，吹胡子瞪眼地说。
<br/>
<br/>        傅昼景完全无视夏凌焰的存在，对着卧室门，大喊道：“筱颍，我知道你在里面！你出来，好不好？再给我两年的时间，我就能处理好季思妍的事情，然后，我去澳洲找你，我们重新开始！筱颍，你相信我！”
<br/>
<br/>        “喂喂喂！我忘记告诉你了！我跟我老婆，已经领证了！你呢？哪来的，就滚哪里去，OK？”夏凌焰不耐烦地对傅昼景推了推手。
<br/>
<br/>        傅昼景双手握拳，急红了眼，猝不及防地往夏凌焰的脸上给了一拳。
<br/>
<br/>        “啊——”夏凌焰突然叫了一声。
<br/>
<br/>        卧室里，童筱颖有些担心夏凌焰一个人扛不住，立即打开了房门。
<br/>
<br/>        傅昼景看到童筱颖，不禁喜上眉梢，然而，却只见她拉住夏凌焰的手，只关心夏凌焰有没有伤到哪里，这让他心里又恼火了起来。
<br/>
<br/>        童筱颖走出来的时候，夏凌焰是捂着自己的左脸颊，当她拉着他的手，看看他的伤势时，却有些不厚道地“噗嗤”一笑。
<br/>
<br/>        “亲爱的，你还笑！”夏凌焰憋着嘴说。
<br/>
<br/>        童筱颖看着夏凌焰左眼被打成了“熊猫眼”，一脸抱歉加愧疚。
<br/>
<br/>        傅昼景跨步上前，夏凌焰吓得立即往童筱颖身后躲去。
<br/>
<br/>        “……”这是童筱颖第一次见夏凌焰如此“怂”，与他那留着络腮胡渣，深眼窝，立体的脸庞透着设计师的艺术气息完全不搭。
<br/>
<br/>        “筱颍……”傅昼景深情款款地唤了声，伸出手想要去牵童筱颖的手时，童筱颖却微微侧身，躲开了他的手。
<br/>
<br/>        童筱颖不想直视傅昼景的眼睛，只得将目光落向一旁，淡淡地扯了扯嘴角，说道：“请回吧！”
<br/>
<br/>        “澳洲那边宠物家园给我打电话了，他们说你把‘小爱’寄养在他们那儿后，就一直未去把‘小爱’接走，就连你的手机号码都打不通。筱颍，这段日子，你一直都没回家吗？你连我们的‘小爱’都不要了吗？”
<br/>
<br/>        童筱颖心痛地皱了下眉头，眸子微冷：“是你不要我的！”
<br/>
<br/>        傅昼景微微垂下眼帘，下一刻，他再抬眸看着她时，声音有些妥协：“筱颍，我有苦衷，我没有不要你，我希望你给我一些时间。”
