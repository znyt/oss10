　　寒月漓跟洛汐，可都是八劫古帝，寒月漓更是因为毁灭劫雷，得到了巨大的好处，现在一身实力，达到了八劫古帝巅峰。
<br/>　　比起施展血祭之前的陈少主，都丝毫不弱。
<br/>　　他们二人联手轰击之下，陈少主身上的力量顿时被镇压大半。
<br/>　　楚风眠一个人对付这陈少主，也许还要耗费一些手段，但是在加上寒月漓跟洛汐。
<br/>　　对楚风眠来说，杀了这陈少主，轻而易举。
<br/>　　“双剑齐出！”
<br/>　　“九玄剑术第一式！贪狼！”
<br/>　　“第二式！巨门！”
<br/>　　双剑之上，星辰之力凝聚在了剑锋，随着楚风眠一剑斩杀而出，顿时向着陈少主轰击而去。
<br/>　　两道剑锋，凝聚星辰之力，斩出的一刻，便是化为了一道星河，将这陈少主打入到了星河之中。
<br/>　　以漫天星辰之力，源源不断的轰击这他。
<br/>　　那九层宝塔，也随即落下，砸在了这陈少主的身躯之上。
<br/>　　一股股力量，镇压的这陈少主都动弹不得。
<br/>　　“孽畜！该死！”
<br/>　　陈少主疯狂的咆哮着，纵然是他的力量再强，也无法挣脱楚风眠三人联手的镇压。
<br/>　　更是在这之中，陈少主的力量在节节摔落。
<br/>　　血祭之术得到的力量，本就不能长久，更何况陈少主还是在灯油枯尽的情况之下，强行施展血祭之术。
<br/>　　拼的就是一口气。
<br/>　　如果让陈少主进攻，可能还会约战越勇，可现在一旦是被镇压，陷入被动，血祭之术的力量，也不能长久。
<br/>　　“不用挣扎了！死吧！”
<br/>　　楚风眠看着陈少主的力量逐渐衰败，就在这时，剑锋凝聚，一剑斩出，化为了万丈剑芒，斩落在陈少主的身躯之上。
<br/>　　他的肉身，再次破碎，化为了一团血雾！
<br/>　　“吞天！”
<br/>　　这一次楚风眠已经是早早的做好了准备，施展出吞天秘术来，这一团精血，尽数被吞噬到了吞天秘术之中。
<br/>　　随着楚风眠运转吞天兽心脏，顿时将这一团精血，彻底的炼化。
<br/>　　在楚风眠的背后，一尊巨大无比的身影，突然显现出来，这一尊身影，无比模糊，无法看清其真容。
<br/>　　唯有看见的，只有这一道身影，在星河之中，一张口，无数星辰，无数大陆，都被他一口吞噬。
<br/>　　这才是真正的吞噬诸天之力！
<br/>　　不用任何的猜想，都足有看出这身影的身份，吞天兽。
<br/>　　真正的吞天之主！
<br/>　　这陈少主的血脉之中，可也是拥有这吞天兽的血脉，落入到吞天秘术之中。
<br/>　　顿时被楚风眠直接吞噬，楚风眠感觉到，那吞天兽心脏，在楚风眠吞噬这陈少主精血的时候，都在颤动了一下。
<br/>　　这让楚风眠一下子吓了一跳，莫非是这心脏，还可以复活过来？
<br/>　　这吞天兽心脏，可是一个死物，与其说是心脏，倒不如说是化为了一件类似于灵器一般的存在。
<br/>　　可随着楚风眠吞噬血脉，这变化，也是让楚风眠一愣，他仔细观察，才发现这吞天兽心脏的力量，似乎变强了几分。
<br/>　　至于之前的颤动，却是消失了，还是如同一个死物一样。
<br/>　　楚风眠又观察了几下，没有察觉到其他异常后，才再度炼化起来。
<br/>　　很快这陈少主身上的精血，都被楚风眠全部吞噬殆尽。
<br/>　　这陈少主，不止是有着吞天兽血脉，更是一名八劫巅峰古帝，这一身精血所蕴含的力量，比起寻常三五名八劫古帝的还要多。
<br/>　　被楚风眠所吞噬，楚风眠感觉到他的力量，又是增强了不少，他现在的实力，已经是无限与接近九劫古帝了。
<br/>　　不如真正的九劫古帝，但是在八劫古帝的级别，已经是堪称无敌。
<br/>　　九劫古帝之下，能够跟他一战的，只有像是小叶帝，小绝帝这样得到过力量传承的人物了。
<br/>　　楚风眠苏醒过来，凌空一抓，十几个空戒都是落入到了楚风眠的手心之中，这就是陈家诸位古帝的空戒。
<br/>　　楚风眠稍微扫了一眼，就发现其中的无数圣丹，诸天奇物，一个个都是富得流油。
<br/>　　甚至是许多已经消失的古圣丹，古圣药，都赫然在列。
<br/>　　显然是这群陈家古帝，只怕是已经从九域之中得到了极大的好处了。
<br/>　　这些东西，楚风眠看了一眼都全部收了起来。
<br/>　　现在寻常的古圣丹，古圣药对楚风眠实力的提升都没有多少了。
<br/>　　而寒月漓跟洛汐，也是刚刚突破境界，现在正是需要巩固境界的时候，也不需要在提升力量。
<br/>　　楚风眠现在需要的，就是做突破帝尊境的准备。
<br/>　　踏入帝尊境，楚风眠才是真正的一飞冲天，等到他踏入帝尊境，只怕是实力会达到一种不可思议的地步。
<br/>　　他的积累堪称天下无双，简直是整个大罗天星域的历史之中，都从未有出现过楚风眠这样的天才。
<br/>　　一旦是踏入帝尊境，便是真正的所向睥睨，无人能敌。
<br/>　　本来楚风眠距离帝尊境还相差不少，但是杀了陈少主，在吞噬了陈少主的精血之后，楚风眠的力量再度攀升。
<br/>　　现在距离帝尊境，已经是只差一步之遥。
<br/>　　对楚风眠来说，他已经是足以去寻找一个机会，尝试踏入帝尊境了。
<br/>　　“楚风眠，多谢你为北邙学院铲除了外敌，这一次的功劳，我们不会忘记的，从此之后，你就是北邙学院的院长，我们愿意以你为尊。”
<br/>　　就在楚风眠还在查看他的境界的时候，那北邙学院的院长，突然走了出来，一脸正色的说道。
<br/>　　“这一次北邙学院受到外敌入侵，还要多谢你，拯救了学院。”
<br/>　　“不错，这陈家小人，祸害学院，罪大恶极，我们没有能力反抗，只能够是虚与委蛇，今日你回来了，北邙学院算是有救了。”
<br/>　　又一名北邙学院的老祖，上前恭喜道。
<br/>　　之前的那副嚣张模样，已经是全然不见，如今都是一脸讨好。
<br/>　　就连跟楚风眠有些仇怨的阳皇等人，都是急忙走了过来，一脸恭敬。
<br/>　　“祸害学院的小人，可不止是陈家的那些人吧。”
<br/>　　楚风眠目光闪烁，看向北邙学院的一群人。
<br/>　　被楚风眠目光扫过，北邙学院的一群人，都是感觉到一股刺骨寒冷，身躯忍不住的发抖。
