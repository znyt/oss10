　　楚风眠凌空负手而立，一动不动，五分钟后，他的脸色才微微变化了一下。
<br/>　　这五分钟的时间，楚风眠便是就在探察那炎阳门的记忆，尤其是在归墟大陆之中所发生的种种。
<br/>　　这三名炎阳门弟子，在一开始进入到归墟天的时候，便是直接来到了归墟大陆中，三人联合起来，布置陷阱。
<br/>　　一边搜寻这归墟天中的种种宝藏，一边便是想要猎杀武者，只是他们有些不走运，第一个碰到的人，便是楚风眠。
<br/>　　以至于现在，他们三人已经是全部的死在了楚风眠的手心之中。
<br/>　　不过在这炎阳门弟子的记忆之中，倒是遇着一些有用的事，其中之一，便是代表着归墟天的最近的变化。
<br/>　　就在这一次归墟天开启的一刻，归墟大陆上，也是突然出现了无数种种异变，其中最大的一个，便是一处远古遗迹，突然出世了。
<br/>　　“云羽天宫的传承，居然出现了？”
<br/>　　楚风眠都是不由的惊呼了一声。
<br/>　　云羽天宫，曾经就在十万年前，在远古时代的九域之中足以称得上是称王称霸的一方势力。
<br/>　　比起现在的七大宗门来说，甚至都是要强大几分。
<br/>　　楚风眠在古籍之中，才曾经知晓过云羽天宫这一方势力的存在，没想到它的遗址，便是就在这归墟天之中。
<br/>　　如今居然是已经要即将出世了。
<br/>　　根据传闻之中，这种远古遗迹的出世，很有可能便是代表有着远古时代的传承，一起出世。
<br/>　　在这归墟天之中曾经的不少远古宗门，都是在当初九域之中足以称王称霸的存在，虽然已经是消失万年，但是他们的传承，在许多时候依然还会保留下来。
<br/>　　所以在这归墟天之中，经常便是会有一些远古遗址出世，要将那些远古宗门的传承，再度传递下去。
<br/>　　这就是归墟天中，最大的机缘。
<br/>　　“云羽天宫的传承，我必须要得到，曾经足以在九域称王称霸的宗门，他们的传承，非同小可。”
<br/>　　楚风眠心中暗自想到，也自然清楚，这云羽天宫的传承，他是必须要得到。
<br/>　　云羽天宫的传承，比起任何一方强者的传承，都要珍贵，重要的多。
<br/>　　就算是楚风眠曾经得到过的颜古龙帝的传承，真正的在这云羽天宫的传承面前，都根本算不得什么。
<br/>　　毕竟颜古龙帝再强，也只是一位强者，而云羽天宫代表着，可是九域的一个时代。
<br/>　　起码在他所在的巅峰时代，云羽天宫是绝对无敌的存在。
<br/>　　谁得到了这云羽天宫的传承，只怕都会九域之中第一的天才，这种机缘，千百年都碰不到过一次。
<br/>　　楚风眠曾经在古籍上，了解过这云羽天宫，自然也清楚这云羽天宫到底会有着多么可怕。
<br/>　　他的传承，楚风眠必须要得到，不惜一切代价的得到。
<br/>　　“云羽天宫的传承，如今还没有彻底开启，不过已经是随时有可能开启，没有在耽误下去的时间了，而且在这云羽天宫的遗迹之中，更是还有着无上剑气的存在！”
<br/>　　楚风眠心神之中，隐约已经是感觉得到，在云羽天宫的遗迹之中，正是有着无上剑气的存在。
<br/>　　不管是这云羽天宫的传承，还是其中的无上剑气，楚风眠都是必须要得到。
<br/>　　“走！”
<br/>　　楚风眠当机立断，毫不犹豫，直接便是化为了一道遁光，向着云羽天宫遗迹所在的区域飞遁而去。
<br/>　　一路上楚风眠碰到的无数宝藏，他都毫不理会，跟云羽天宫的传承，还有无上剑气相比，这种小东西，根本算不了什么。
<br/>　　一两个时辰过去，楚风眠的遁光，已经是在归墟大陆之中，横穿了无数区域。
<br/>　　这一路楚风眠所路过的区域，许多都是一些绝地，一眼望去，都令人生畏，不过每一处绝地之中，虽然危险，但是其中的宝藏，却也是一点也不少。
<br/>　　不过这些宝藏，楚风眠都是毫不理会，他的遁光，一直便是向着一个方向飞去。
<br/>　　足足是三个时辰的时间，在他的面前，突然之间，便是显现出了一座巨大无比的城池。
<br/>　　这城池，就位于归墟大陆之中，庞大无比，却也显得极为古老，已经是破旧了不少。
<br/>　　上面曾经雕刻过的一些禁制的痕迹，现在也已经全部破碎，显然是没落依旧。
<br/>　　只是牌匾上刻画有着的“天羽城”三个字，却是清晰可见。
<br/>　　天羽城。
<br/>　　这里，便是曾经云羽天宫的外围都城，就像是在北域之中的北邙城一样，代表着是云羽天宫的世俗力量。
<br/>　　在远古时代，这天羽城只怕是足以称得上是当时九域之中数一数二的庞大都城。
<br/>　　如今因为这云羽天宫的衰败，这云羽城也一样衰败，到现在，终于是成为了一方废墟。
<br/>　　不过虽然只是一处废墟，但是楚风眠的心中，却是丝毫不敢放松警惕，因为楚风眠已经感觉的出，这天羽城中，已经是有着不少的武者。
<br/>　　显然是这云羽天宫即将出世的消息传来，在归墟天中的不少武者，都是前来到了这里，等待着云羽天宫的开启。
<br/>　　一个个，都已经是在天羽城中，寻找好了最好的位置，一旦是云羽天宫开启，便是开始争抢。
<br/>　　楚风眠观察了一眼，便是脚下一动，一下子踏入到了这云羽城之中。
<br/>　　就在楚风眠踏入到天羽城中的一刻，无数灵识，便是毫不遮掩的，直接向着楚风眠扫视过来，似乎是在打量着楚风眠。
<br/>　　其中的几道灵识，更是毫不顾忌，甚至是要看穿楚风眠的身形，观察出楚风眠的底牌来。
<br/>　　“滚！”
<br/>　　楚风眠眼神冰冷，口中淡然吐出一个字。
<br/>　　猛然之间，一声龙吟，便是突然响彻而起。
<br/>　　这些观察着楚风眠的灵识，被纷纷打的破碎开来，就在这云羽城之中，都是有着几声惨叫，突然响起。
<br/>　　这些惨叫声，就是刚刚想要打量楚风眠身形底牌的灵识的主人。
<br/>　　如今在楚风眠这一声龙吟面前，一个个都是被直接震得重伤。
<br/>　　“哼！”
<br/>　　楚风眠冷哼一声。
<br/>　　这些武者，一个个上来就想要给他一个下马威，在这归墟天，人弱便是只能被人欺辱，只有强者，才有资格在其中争夺。
