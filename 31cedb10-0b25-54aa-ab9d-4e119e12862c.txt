　　“女娲师妹，你我论道一番，如何？”
<br/>　　元始那淡淡声音突然在娲皇宫中响起，让女娲那想要出手的念头为之一滞。
<br/>　　“好，既然如此，那师妹便来领教一下师兄的高招！”
<br/>　　女娲眼中闪过一丝怒色，不过，很快她便调整过来了。她心中知道，其他圣人绝对不会放任她插手三族大战，因为这场大战就有几位圣人的算计在其中，她有想过肯定会有人来阻止她，但却没想过这人竟会是元始。
<br/>　　元始作为六圣之首，实力也最为强大，平时不出手，没人会知道他的实力究竟到了哪个地步，她如今正好试探一番。
<br/>　　女娲圣人知道元始的厉害，所以一出手便毫不留情，女娲单手一翻，先天灵宝红绣球便出现在她手中，轻轻一抛，红绣球迅速变大，绽放出耀眼的红芒，翻滚之中，带着毁灭之光，女娲造化法则加持其身，向着元始飞速撞击而来。
<br/>　　混沌翻滚不休，迅速被红绣球打破，地水火风涌动翻滚，被造化法则赋予各种形态，向着元始冲击而来。
<br/>　　“定！”
<br/>　　看着那携着开天辟地般浩大神威向他击来的红绣球，元始不为所动，面色淡然，看不出丝毫的变化，他缓缓伸出一指，点向虚空。这一指看似很慢，实则很快，没有玄光，没有法则波动，亦没有神秘力量，有的只是简简单单，朴实无华的一指。
<br/>　　顿时，虚空停顿，时间仿佛变得缓慢起来，周围的一切仿佛都消失了，只剩下一个简单的指头。
<br/>　　“去！”
<br/>　　元始那指头轻轻一弹，顿时，那携带着无上伟力的红绣球便被元始的指头弹了倒飞回去，同样带着无双的力量，撞击回去。
<br/>　　“轰！”
<br/>　　女娲接住倒退的红绣球，顿时娲皇宫中产生剧烈的动荡，女娲圣人倒退数十步，方才停了下来。
<br/>　　“再来！”
<br/>　　女娲不服输，红绣球在她手中挥舞起来，从她的一只手滚落到另一只玉手之上，她手中法则玄奥，虚空生波，造化万物，之后带着巨大的力量，飞了起来，女娲翻身一脚踢在红绣球之上。
<br/>　　顿时，那红绣球带着一股更加强烈的力量撞击而去，虚空直接被撕裂开来，要知道此处可是圣人宫殿，空间可比洪荒之中坚固数倍，但依旧被直接毁灭，可见其力度之大，超乎想象。
<br/>　　红绣球没入虚空之中，带着无匹威力，仿若一尊巨大古神，从混沌之中觉醒，一拳轰击而来。
<br/>　　女娲不停，双手不停击落在虚空之中，不过片刻时间，她便接连在虚空之中开辟出数十个小世界，世界沉浮，汇聚成一方浩大的世界，被女娲一掌劈出，带着毁天灭地的毁灭之力，撞击元始而去。
<br/>　　“有点意思，圣人二重天巅峰，还是有些弱。”
<br/>　　元始饶有兴趣的看着这一切，面色淡然，突然，他向着旁边的虚空之中轰的一拳轰出。
<br/>　　“轰隆隆！”
<br/>　　虚空炸裂，露出里面的红绣球，接着他一掌拍出，又将红绣球送会女娲手中。元始不停，看着在自己面前几乎快要爆炸开的浩大世界，他迅速打出几道法诀，将世界封印，之后他单手一抓，将那方世界抓到手中，向着混沌深处扔去。
<br/>　　“轰隆隆！”
<br/>　　混沌深处传来一声声剧烈的爆炸声，毁天灭地，混沌风暴不断凝聚出来，暴风不断，混沌气流肆虐，一股强大的浪涛波及而来，就连娲皇宫都被震得摇晃起来。
<br/>　　元始全身气势一压，那股摇晃瞬间便被压迫了下来，娲皇宫再次恢复了平静。
<br/>　　昆仑山，太清与通天双眼一睁，看向混沌方向，若有所思，接着便不再关注，而是将目光放在三族大战之中。
<br/>　　西方灵山，接引与准提圣人对视一眼，神秘一笑。
<br/>　　“师兄，看来是元始与女娲师妹交手了。”准提道。
<br/>　　“也好，元始去得罪女娲师妹，终究是对我们有利。不过，话又说回来了，玉清元始的实力可真是深不可测，每次看到他，我都有种看不透的情况，就仿佛是遇到了老师。按理来说，我们都是圣人，这不应该啊。”接引暗暗道。
<br/>　　“是啊，元始，三清，没一个是简单之辈。我西方何时能够大兴啊？”准提叹息一声，心中悲苦不已。
<br/>　　“师弟莫忧，我西方迟早会大兴。”
<br/>　　“……”
<br/>　　混沌天外天，娲皇宫。
<br/>　　“如何，女娲师妹，还要继续吗？”
<br/>　　元始淡然立于虚空，好比日月星辰般璀璨，诸般法相，尽在他身上体现出来，他就仿佛是道的化身。
<br/>　　“师兄道行高深，师妹不如也！”
<br/>　　女娲眼中闪过一丝不甘，但更多的是无奈，元始的实力太强了，此番交手，虽然仅仅三招，但她已经用尽了自身全部实力，不求探出元始的全部实力，但也应该窥探出元始的一部分虚实了。但，事实却是让她心惊不已，因为她如今还是丝毫不知道元始的法力道行极限是在哪里，就算是元始的冰山一角，也让她深感无力。
<br/>　　元始微微一笑，对女娲服输丝毫不觉意外，女娲关心的是外面正在大战之中的伏羲妖圣，没必要在这里与自己耗费太多精力。
<br/>　　“对于人、妖、巫三族大战，师妹如何看待？”
<br/>　　沉默一会儿后，元始问出了这样一个问题，想知道女娲心中的想法。
<br/>　　女娲看了一眼元始，好似知道他心中所想，她淡淡道：
<br/>　　“师兄心中所想，我已知晓，放心，师妹是不会去插手三族大战的。”
<br/>　　“那再好不过，打打杀杀，有什么好的，哪有逍遥自在来得快活？对于那些想要打打杀杀的势力，就没必要存在了。”
<br/>　　元始露出一丝警告，同时，也一言决定了巫、妖两族的未来命运。
<br/>　　“你——”
<br/>　　女娲有些生气，俏脸变得怒红，双眼圆瞪，怒视了元始一眼，她知道元始心中对妖族没有好感，想要借助此战，削去妖族的霸主地位，虽然答应过元始，不插手三族大战，但她还是有些不高兴的。事实的确如此，任谁被人威胁了，都会感到不高兴。
<br/>　　“哼！师妹我要闭关了，还请师兄回去。”
<br/>　　女娲冷哼一声，对元始没有好脸色，淡淡道，直接开始赶人了。
<br/>　　“哈！我觉得师妹这里环境不错，多待一会儿也无妨。”元始面对女娲的驱赶，毫不尴尬。
<br/>　　“哼！”
<br/>　　女娲冷哼一声，在旁边的亭子里坐了下来，也没有多管，因为她知道，对方那是不信任她。实际上，她自己话虽如此，但若真是妖族遭遇灭族之危，她一定会出手的，因为她不能眼睁睁看着妖族被灭了。
<br/>　　元始大手一挥，面前的池塘之中出现一副画面，正是洪荒之中三族大战的场面。
<br/>　　两人就这样静静地观看着三族的惨烈大战，当然，最让他们关注的是，人族武祖玄天、巫族祖巫帝江、妖族帝俊与东皇太一之间的战斗。
