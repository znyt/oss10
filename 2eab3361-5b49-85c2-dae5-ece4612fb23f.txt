夏夕绾脸上一白，她没有想到海城一别，才短短三个月，奶奶的身体已经病到这种程度了。
<br/>
<br/>        当时她没有跟奶奶道别，一来怕奶奶伤心，二来也觉得只要陆寒霆好了，奶奶自然就好了，会长命百岁的，谁知道事情竟成了这样。
<br/>
<br/>        奶奶身体已经大了，一旦开始不禁食，剩下的日子就不多了。
<br/>
<br/>        “绾绾，我觉得老夫人这是心病，陆寒霆虽然是她的孙子，但是老夫人也一直把你当亲生的来疼爱的，你为了救陆寒霆而消失，老夫人应该是悲伤过度，忧思成疾，所以才一病不起的。”叶翎道。
<br/>
<br/>        夏夕绾白皙的眼眶红红的，“翎翎，我现在就要去见奶奶，我要想办法让奶奶好起来！可是，现在奶奶在陆寒霆的西苑里，我用什么理由进去看奶奶？”
<br/>
<br/>        “我听说陆寒霆对外一直在招营养师专业的厨娘，绾绾，你可以去面试美厨娘，这样你就有理由正大光明的照顾老夫人了。”
<br/>
<br/>        夏夕绾双眼一亮，“好。”
<br/>
<br/>        ……
<br/>
<br/>        西苑。
<br/>
<br/>        吴妈伺候陆老夫人几十年了，现在老夫人身体不好，吴妈当然是寸步不离的跟了过来。
<br/>
<br/>        这时“叮铃”一声，别墅的门铃响起了。
<br/>
<br/>        谁？
<br/>
<br/>        吴妈从厨房里跑出来，迅速打开了别墅大门，外面是…厉嫣然。
<br/>
<br/>        “厉小姐，你怎么来了？”吴妈诧异道。
<br/>
<br/>        厉嫣然露出甜美的微笑，她扬了扬手里的袋子，“吴妈，我知道陆奶奶最近胃口不好，我是营养师嘛，我最近已经研究出了一套针对陆奶奶的食谱，刚才又让人买了我想要的食材，正好过来看陆奶奶。”
<br/>
<br/>        吴妈让出了道，“厉小姐，你有心了，快进来吧。”
<br/>
<br/>        厉嫣然走了进去，“陆奶奶呢？”
<br/>
<br/>        “在楼上房间里，刚打了营养液，现在睡下了。”
<br/>
<br/>        “那我们先准备一顿营养餐吧，这样等陆奶奶醒了让她吃吃看。”
<br/>
<br/>        “好。”
<br/>
<br/>        两个人走进了厨房，厉嫣然将食谱和食材都递给了吴妈，她是十指不沾阳春水的千金大小姐，自然是不会下厨的，所以她讲解给吴妈听。
<br/>
<br/>        吴妈按照她的话开始准备营养晚餐，现在只要是对老夫人有益的事情，吴妈都愿意去做的。
<br/>
<br/>        一个小时后，耳畔传来了开门声，吴妈迅速道，“应该是少爷回来了。”
<br/>
<br/>        “我去看看。”厉嫣然欢快的跑了出去，很快她就在门边看到了一道高大英挺的身躯，真的是陆寒霆回来了。
<br/>
<br/>        今天的陆寒霆一身手工版的黑色西装，昂贵的布料被熨帖的没有丝毫褶皱，他挺括的肩上染着外间的清寒，整个人优雅矜贵，又散发着一层生人勿进的高冷疏离。
<br/>
<br/>        “寒霆哥哥，你回来了？”厉嫣然开心的叫道。
<br/>
<br/>        陆寒霆拔开长腿走进来，他深邃的狭眸落在厉嫣然漂亮的小脸上，“嫣然，你怎么来了？”
<br/>
<br/>        厉嫣然迅速跑进厨房将吴妈刚做好的营养餐端了出来，献宝似的给陆寒霆看，“寒霆哥哥，这是我给陆奶奶准备的营养餐，私人订制，怎么样，我厉害吧？”
